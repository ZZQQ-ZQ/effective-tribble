# 🔴 每次出提示词前强制工作流（v4.0 · 强制翻库）

**此文件高于一切。每次出提示词前必须从门0走到门4，跳一步=违规。**

---

## 门0：沟通铁律（打字之前）

```
[ ] 导演上一句话里有「发」「出」「来吧」「写吧」吗？
    → 没有 → STOP。只回答问题，不输出提示词/分镜/方案。
    → 有 → 进门1。
```

---

## 门1：写前确认 + 强制翻库（动笔之前）

```
[ ] 导演要求什么时长？（15秒/10秒/其他）→ 确认
[ ] 场景类型？→ 文戏/打斗/空镜
[ ] 有哪些角色？→ 名字/数量
[ ] 角色有没有@图参考？→ 有→不写外貌，只写站位+动作。不知道→问。
[ ] 🔴 视觉基调？→ 查 skills/VIDEO/视觉光效基调/SKILL.md 匹配场景，定块1三标签

### 🔴 强制翻库（前四项每条必须拉出 2-3 个不重复项，不能凭记忆）

[ ] 🔴 光效拉库 → 查 skills/VIDEO/视觉光效基调/SKILL.md 单灯/氛围光/电影光/特殊光
    → 按场景类型匹配，每段拉 2-3 种不重复光效
    → ❌ 禁止默认走"侧光+轮廓光+三点布光"三板斧
    
[ ] 🔴 运镜拉库 → 查 skills/VIDEO/运镜速查表/SKILL.md 场景→运镜速查 → 运动镜头24种/情绪镜头20种/动作镜头16种
    → 每镜从库里拉不同运镜，整段不重复超过1次
    → ❌ 禁止默认走"固定+跟拍+环绕"三板斧

[ ] 🔴 构图拉库 → 查 skills/VIDEO/构图技能/SKILL.md 23种构图法则 + 场景→构图速配表
    → 每镜从库里拉不同构图，整段不重复超过1次
    → ❌ 禁止默认走"三分法+中心构图"二板斧

[ ] 🔴 表情拉库 → 查 skills/VIDEO/人物表演管理技能/SKILL.md
    → 对话/互动场景：查情绪→微表情速查表，拉对应微表情
    → 动作/战斗场景：写身体不写脸谱（铁律1-5）
    → ❌ 禁止写"面无表情"或通用脸谱描述

[ ] 打斗类？→ 查项目 specs（技能锚定/怪物设定/击杀机制/装备掉落）
    → 查 skills/VIDEO/打斗设计技能/SKILL.md
    → 查 skills/VIDEO/动态运镜提升技能/SKILL.md
    
    → 全确认 → 进门2。
```

## 门2：写完自检

```
[ ] 4块结构完整？
    块1：不要字幕，不要背景音乐。[年代地域+光线风格+摄影质感]，4K
    画面基调：（一句话定全段15秒视觉底色）
    色彩与影调：（主调+补色+反差+高光+暗部处理）
    主体：分行列角色/场景/道具
    分镜：每镜X-Y秒 / 景别 / 构图 / 运镜手法 / 画面内动

[ ] 每镜四字段完整：景别 + 构图 + 运镜手法 + 画面内动
[ ] 文戏每镜3-5秒，打斗每镜1-2.5秒
[ ] 台词用反引号包 \`台词\`，写在画面内动中
[ ] 有@图引用 → 不写外貌
[ ] 打斗镜头是运动镜头（非固定）
[ ] 全文无违禁词
[ ] 段头=动作进行中，段尾=最后一击即结束（不站定不pose）

### 🔴 技能多样性检查

[ ] 光效多样性 → 本段用了哪些光效？（列出名称）
    → 是否 ≥ 2 种？是否都是三板斧（侧光/轮廓光/三点布光）？
    → 三板斧占比超过50% → 回头重拉库

[ ] 运镜多样性 → 本段每镜运镜手法？（逐镜列出）
    → 是否有重复？重复超过1次 → 回头重拉库

[ ] 构图多样性 → 本段每镜构图方式？（逐镜列出）
    → 是否有重复？重复超过1次 → 回头重拉库

[ ] 表情审查 → 有对话场景吗？
    → 有 → 入画角色都写了微表情吗？微表情是否查了速查表而非凭记忆写？
    → 无 → 是否遵守了"写身体不写脸谱"五铁律？

[ ] 场景一致性 → 查 skills/VIDEO/场景一致性保持技能/SKILL.md
    → 空间逻辑/连续性/光线锁死/持物手延续 全部确认
    
    → 全过 → 进门3。
```

## 门3：发前最终

```
[ ] 全文扫描：页码/时长/图号/段头段尾 全部正确？
[ ] 行尾无多余字段（无旧格式残留的光效：/角度：/镜头：等字段名）
    → 全过 → 出门发。
```

---

## 🔴 铁律

1. **跳一步=废稿。**
2. **门1「强制翻库」四项缺一不可——每项必须真的打开对应技能文件，不准凭记忆。**
3. **多样性检查不过 → 不准出门。回头重拉库。**
4. **三板斧（侧光+轮廓光+三点/固定+跟拍+环绕/三分法+中心）占比超50% → 立刻重拉。**
