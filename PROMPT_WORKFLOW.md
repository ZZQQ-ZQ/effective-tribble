# 🔴 每次出提示词前强制工作流（v3.0 · 导演指令式）

**此文件高于一切。每次出提示词前必须从门0走到门3，跳一步=违规。**

---

## 门0：沟通铁律（打字之前）

```
[ ] 导演上一句话里有「发」「出」「来吧」「写吧」吗？
    → 没有 → STOP。只回答问题，不输出提示词/分镜/方案。
    → 有 → 进门1。
```

---

## 门1：写前确认（动笔之前）

```
[ ] 导演要求什么时长？（15秒/10秒/其他）→ 确认
[ ] 场景类型？→ 文戏/打斗/空镜
[ ] 有哪些角色？→ 名字/数量
[ ] 角色有没有@图参考？→ 有→不写外貌，只写站位+动作。不知道→问。
[ ] 视觉基调？→ 查 skills/VIDEO/视觉基调/SKILL.md 匹配场景，定块1三标签（年代地域+光线风格+摄影质感）
[ ] 画面基调？→ 每段不同，一句话定全段15秒视觉底色
[ ] 色彩与影调？→ 主调+补色+反差+高光处理+暗部处理（每段不同）
[ ] 打斗类？→ 查 project specs（技能锚定/怪物设定/击杀机制/装备掉落）
    → 全确认 → 进门2。
```

---

## 门2：写完自检

```
[ ] 4块结构完整？
    块1：不要字幕，不要背景音乐。[年代地域+光线风格+摄影质感]，4K，超写实，极致逼真，UE5，Octane渲染
    画面基调：（一句话定全段15秒视觉底色）
    色彩与影调：（主调+补色+反差+高光+暗部处理）
    主体：分行列角色/场景/道具
    分镜：每镜X-Y秒 / 景别 / 构图 / 运镜手法 / 画面内动

[ ] 每镜四字段完整：景别 + 构图 + 运镜手法 + 画面内动
[ ] 文戏每镜3-5秒，打斗每镜1-2.5秒
[ ] 台词用反引号包 \`台词\`，写在画面内动中
[ ] 有@图引用 → 不写外貌
[ ] 打斗镜头是运动镜头（非固定）
[ ] 全文无违禁词
[ ] 段头=动作进行中，段尾=最后一击即结束（不站定不pose）
    → 全过 → 进门3。
```

---

## 门3：发前最终

```
[ ] 全文扫描：页码/时长/图号/段头段尾 全部正确？
[ ] 行尾无多余字段（无旧格式残留的光效：/角度：/镜头：等字段名）
    → 全过 → 可以发。
```
