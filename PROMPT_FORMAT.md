## 规则

| # | 规则 |
|---|------|
| 0 | 🔴 4块+段尾负面提示词（2026-07-24 导演指定） |
| 1 | 4块结构：头部→画面基调→色彩与影调→主体→分镜。分镜全部结束后另起一行，段尾负面提示词。 |
| 2 | 块1：不要字幕+不要背景音乐+年代地域+光线风格+摄影质感+4K+超写实+极致逼真+UE5+Octane渲染 |
| 3 | 画面基调：摄影机行（机身+焦段光圈+胶片），一行收住 |
| 4 | 色彩与影调：技术参数（主调+补色+反差+高光+阴影区）+ 句号 + 氛围描述（只提炼分镜里已有的视觉元素，不脑补新场景） |
| 5 | 主体：只写角色名+手持武器，**禁止写画左/画右/画中/站位**（位置放每镜构图里去写） |
| 5a | 🔴 主体段是演员表不是站位表。空间关系逐镜独立绑定，不与全局交叉 |
| 6 | 分镜格式：每个镜头拆为 `景别 / 构图 / 运镜手法 / 画面内动` 四字段。每镜构图自己写该镜空间位置+机位方位，不依赖其他镜头 |
| 7 | 文戏每镜3-5秒，打斗每镜1-2.5秒 |
| 8 | 台词用反引号包：\`台词内容\`，写在画面内动中 |
| 9 | 有@图引用的角色不写外貌，只写站位+动作状态 |
| 10 | 🔴 段尾负面提示词：所有分镜结束后另起一行，固定文本锁死：`避免塑料感，避免蜡像皮肤，避免木偶脸，避免眼神呆滞，避免双脚滑步，避免手指变形，避免面部特征漂移，避免形态变形，避免动作僵硬不自然。` 一字不改。|
| 11 | 战斗类先查项目设定（技能锚定/怪物设定/击杀机制/装备掉落） |
