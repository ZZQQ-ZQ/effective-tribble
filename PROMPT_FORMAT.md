## 格式结构（4块）

```
不要字幕，不要背景音乐。[年代地域] + [光线风格] + [摄影质感]，4K

画面基调：（一段15秒的整体画面视觉底色，一句话收住，每段不同）
色彩与影调：（主调+补色+反差+高光处理+暗部处理）

主体：
角色1
角色2
场景：地点+时间+光效
道具：
道具1
道具2

X-Y秒 景别，运镜。动作描述。光效。台词。
```

## 完整示例

### 文戏
```
不要字幕，不要背景音乐。真人写实，暖金日光，固定丝滑，4K

画面基调：午后室内，暖金色阳光从窗格斜射，逆光人物剪影，空气中有浮尘粒子缓慢飘动。
色彩与影调：暖金主调，琥珀补色，中调低反差，高光柔和漫射，暗部保留纹理。

主体：
秦轩（画面中央，正对镜头，双手垂立）
场景：室内窗边，午后，暖金日光

0-3秒  中景，固定镜头。秦轩站在窗边，逆光剪影，风吹窗帘缓缓飘动。
3-7秒  近景，镜头缓推。秦轩转身面向镜头，嘴角微扬，眼神从空洞恢复聚焦。
7-12秒 过肩镜头，低角度仰拍。秦轩迈步走向门口，手握门把停顿，肩膀起伏深吸一口气。
12-15秒 特写手部。门把缓缓转动，暖光从门缝涌入照亮手指。
```

### 打斗
```
不要字幕，不要背景音乐。写实武侠，冷月夜色，手持晃动，4K

画面基调：月光下演武场，冷蓝月光穿透扬尘成束，尘埃弥漫空间，刀光与月影冷暖冲撞。
色彩与影调：冷蓝主调，暗金点缀，高反差高饱和，高光冷白光刃，暗部压纯黑。

主体：
秦轩（画右，右手持刀，弓步蓄力）对峙林莫（画左，双手握棍，重心下沉）
场景：夜间演武场，月光，扬尘弥漫

0-1.5秒 全景，低角度跟拍。月光穿过扬尘形成斜光束，秦轩拖刀疾冲入镜，刀尖刮地拉火星。
1.5-3秒 近景，手持晃动。秦轩蹬地腾空转体，刀身反射冷蓝月光划弧，转体瞬间刀光爆闪，画面一瞬过曝后恢复。
3-4秒 特写，震屏。刀刃砸棍身交击点，冷暖光对冲——暗金刀光与冷蓝棍身反光交汇，火星环形扩散，冲击波震尘土。
4-5.5秒 仰拍，缓镜慢放。林莫双膝跪地，棍身断裂飞散，秦轩收刀入鞘，刀身暗金纹路最后一次闪亮后暗去。
5.5-7秒 全景，固定镜头。演武场尘埃缓缓沉降，两人剪影一立一跪，月光重新穿透扬尘形成光束。
```

### 空镜（无人）
```
不要字幕，不要背景音乐。荒废写实，暖金暮色，固定横移，4K

画面基调：废弃演武场被暮色包裹，暖金光线掠过石质纹理，浮尘在光中缓慢飘散，寂静萧瑟。
色彩与影调：暖金主调，暗蓝补色，中调高反差，高光暖金漫射，暗部偏冷深。

0-4秒  全景，固定镜头。废弃演武场，夕阳穿过枯树投下长影，落叶在石缝间被风推着缓慢移动。
4-8秒  中景，缓慢横移。石质地面纹路被暖金色光打亮，浮尘在光束中缓慢飘动。
8-12秒 近景，缓推。干涸的剑痕刻在石柱上，光影缓慢滑过凹槽纹理。
12-15秒 远景，固定镜头。暮色渐深，演武场隐入暗蓝暮霭，远处一盏孤灯亮起。
```

## 规则

| # | 规则 |
|---|------|
| 1 | 4块结构：头部→画面基调→色彩与影调→主体→分镜 |
| 2 | 块1写：不要字幕+不要背景音乐+年代地域+光线风格+摄影质感+4K（限3-4词，不写色彩） |
| 3 | 画面基调：一句话定全段15秒视觉底色，每段不同 |
| 4 | 色彩与影调：主调+补色+反差+高光处理+暗部处理，块1不重复写色彩 |
| 5 | 主体：分行列出角色/场景/道具 |
| 6 | 分镜格式：`X-Y秒 景别，运镜。动作。光效。台词。` — 一行一个镜头 |
| 7 | 文戏时间段3-5秒/镜，打斗时间段1-2.5秒/镜 |
| 8 | 台词用反引号包：\`台词内容\` |
| 9 | 有@图引用的角色不写外貌，只写站位+动作状态 |
| 10 | 战斗类先查项目设定（技能锚定/怪物设定/击杀机制/装备掉落） |
