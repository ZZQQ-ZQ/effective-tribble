## 完整示例

### 文戏
```
不要字幕，不要背景音乐。真人写实，暖金日光，固定丝滑，4K，超写实，极致逼真，UE5，Octane渲染

画面基调：摄影机：ARRI Alexa 65，50mm f/2.8，Kodak Vision3 500T
色彩与影调：暖金主调，琥珀补色，中调低反差，高光柔和漫射，暗部保留纹理。午后室内，暖金色阳光从窗格斜射，逆光人物剪影，空气中有浮尘粒子缓慢飘动。

主体：
秦轩（画面中央，正对镜头，双手垂立）
场景：室内窗边，午后，暖金日光

0-3秒
景别：中景
构图：秦轩画中站立，窗格在背景投下斑驳光影
运镜手法：固定镜头
画面内动：秦轩站在窗边，逆光剪影，风吹窗帘缓缓飘动。

3-7秒
景别：近景
构图：秦轩面部画中偏左，右后方窗口柔光涌入
运镜手法：缓推（dolly in），焦点从窗格拉到面部
画面内动：秦轩转身面向镜头，嘴角微扬，眼神从空洞恢复聚焦。

7-12秒
景别：中景
构图：秦轩画右走向门口，门框在画左形成纵深
运镜手法：低角度仰拍，过肩跟踪
画面内动：秦轩迈步走向门口，手握门把停顿，肩膀起伏深吸一口气。

12-15秒
景别：特写
构图：手部在画中，门把占右下角
运镜手法：固定镜头
画面内动：门把缓缓转动，暖光从门缝涌入照亮手指。
```

### 打斗
```
不要字幕，不要背景音乐。写实武侠，冷月夜色，手持晃动，4K，超写实，极致逼真，UE5，Octane渲染

画面基调：摄影机：RED V-Raptor，24mm f/4.0，Kodak Vision3 200T
色彩与影调：冷蓝主调，暗金点缀，高反差高饱和，高光冷白光刃，暗部压纯黑。月光下演武场，冷蓝月光穿透扬尘成束，尘埃弥漫空间，刀光与月影冷暖冲撞。

主体：
秦轩（画右，右手持刀，弓步蓄力）对峙林莫（画左，双手握棍，重心下沉）
场景：夜间演武场，月光，扬尘弥漫

0-1.5秒
景别：全景
构图：秦轩画右拖刀入镜，林莫画左弓步戒备，月光斜束穿过画面中轴
运镜手法：低角度跟拍，手持晃动
画面内动：月光穿过扬尘形成斜光束，秦轩拖刀疾冲入镜，刀尖刮地拉火星。

1.5-3秒
景别：近景
构图：秦轩占据画面中心，刀身上挑至画面上方
运镜手法：手持晃动，仰拍
画面内动：秦轩蹬地腾空转体，刀身反射冷蓝月光划弧，转体瞬间刀光爆闪，画面一瞬过曝后恢复。

3-4秒
景别：特写
构图：刀刃棍身交击点占画面中央，火星辐射至画框边缘
运镜手法：震屏，固定镜头慢镜
画面内动：刀刃砸棍身交击点，冷暖光对冲——暗金刀光与冷蓝棍身反光交汇，火星环形扩散，冲击波震尘土。

4-5.5秒
景别：仰拍全景
构图：秦轩画左收刀站立，林莫画右跪地
运镜手法：缓镜慢放
画面内动：林莫双膝跪地，棍身断裂飞散，秦轩收刀入鞘，刀身暗金纹路最后一次闪亮后暗去。

5.5-7秒
景别：全景
构图：两人剪影一立一跪在画框下三分之一，月光重新穿透画面上方扬尘
运镜手法：固定镜头
画面内动：演武场尘埃缓缓沉降，月光重新穿透扬尘形成光束。
```

### 空镜（无人）
```
不要字幕，不要背景音乐。荒废写实，暖金暮色，固定横移，4K，超写实，极致逼真，UE5，Octane渲染

画面基调：摄影机：ARRI Alexa 65，24mm f/8.0，Kodak Vision3 250D
色彩与影调：暖金主调，暗蓝补色，中调高反差，高光暖金漫射，暗部偏冷深。废弃演武场被暮色包裹，暖金光线掠过石质纹理，浮尘在光中缓慢飘散，寂静萧瑟。

0-4秒
景别：全景
构图：枯树在画左，落日低角度从画右射入，石质地面延伸至画面深处
运镜手法：固定镜头
画面内动：夕阳穿过枯树投下长影，落叶在石缝间被风推着缓慢移动。

4-8秒
景别：中景
构图：石质地面纹理占满画面
运镜手法：缓慢横移（truck left）
画面内动：石质地面纹路被暖金色光打亮，浮尘在光束中缓慢飘动。

8-12秒
景别：近景
构图：石柱刻痕在画中，光影斜掠过凹槽
运镜手法：缓推（dolly in）
画面内动：干涸的剑痕刻在石柱上，光影缓慢滑过凹槽纹理。

12-15秒
景别：远景
构图：演武场退至画面下方，天空占据上方三分之二，孤灯在画右远角
运镜手法：固定镜头
画面内动：暮色渐深，演武场隐入暗蓝暮霭，远处一盏孤灯亮起。
```

## 规则

| # | 规则 |
|---|------|
| 1 | 4块结构：头部→画面基调→色彩与影调→主体→分镜 |
| 2 | 块1：不要字幕+不要背景音乐+年代地域+光线风格+摄影质感+4K+超写实+极致逼真+UE5+Octane渲染 |
| 3 | 画面基调：摄影机行（机身+焦段光圈+胶片），一行收住 |
| 4 | 色彩与影调：技术参数（主调+补色+反差+高光+暗部）+ 句号 + 氛围描述（只提炼分镜里已有的视觉元素，不脑补新场景） |
| 5 | 主体：分行列出角色/场景/道具 |
| 6 | 分镜格式：每个镜头拆为 `景别 / 构图 / 运镜手法 / 画面内动` 四字段 |
| 7 | 文戏每镜3-5秒，打斗每镜1-2.5秒 |
| 8 | 台词用反引号包：\`台词内容\`，写在画面内动中 |
| 9 | 有@图引用的角色不写外貌，只写站位+动作状态 |
| 10 | 战斗类先查项目设定（技能锚定/怪物设定/击杀机制/装备掉落） |
