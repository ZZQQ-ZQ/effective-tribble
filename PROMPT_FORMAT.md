# 提示词格式（导演指令式 · v3.0）

> 🔴 出任何提示词前强制调用此底板。
> **最后更新**：2026-07-05 — 全网调研后大改：5层字段→3块结构，分时段描述，光效嵌入画面不另起行。
> **旧版归档**：`PROMPT_FORMAT_V1.md`（5层递进）`PROMPT_FORMAT_V2.md`（7字段）— 已废弃，仅存档。

---

## 格式结构（3块）

### 块1：头部锚定（一行）
```
不要字幕，不要背景音乐。[摄影规格/风格/光调/氛围/色调] [分辨率]
```

### 块2：引用+站位（有素材/有人物时写，一行）
```
@图N的角色名（站位+朝向+手持物+状态），@图N的场景名
```

### 块3：时间线分镜（逐行）
```
X-Y秒  景别，运镜。动作描述。光效事件。
```

## 完整示例

### 文戏
```
不要字幕，不要背景音乐。真人写实，暖金色自然光从窗格斜射，阴影柔和，胶片颗粒，浅景深，4K

@图7的秦轩（画面中央，正对镜头，双手垂立）

0-3秒  中景，固定镜头。秦轩站在窗边，逆光剪影，风吹窗帘缓缓飘动。
3-7秒  近景，镜头缓推。秦轩转身面向镜头，嘴角微扬，眼神从空洞恢复聚焦。
7-12秒 过肩镜头，低角度仰拍。秦轩迈步走向门口，手握门把停顿，肩膀起伏深吸一口气。
12-15秒 特写手部。门把缓缓转动，暖光从门缝涌入照亮手指。
```

### 打斗
```
不要字幕，不要背景音乐。手持纪实风格，冷蓝月光×暗金刀光冷暖对冲，阴影压纯黑，画面颗粒感，ARRI Alexa质感，4K

@图7的秦轩（画右，右手持刀，弓步蓄力）对峙@图6的林莫（画左，双手握棍，重心下沉），@图2的演武场（夜间）

0-1.5秒 全景，低角度跟拍。月光穿过扬尘形成斜光束，秦轩拖刀疾冲入镜，刀尖刮地拉火星。
1.5-3秒 近景，手持晃动。秦轩蹬地腾空转体，刀身反射冷蓝月光划弧，转体瞬间刀光爆闪，画面一瞬过曝后恢复。
3-4秒 特写，震屏。刀刃砸棍身交击点，冷暖光对冲——暗金刀光与冷蓝棍身反光交汇，火星环形扩散，冲击波震尘土。
4-5.5秒 仰拍，缓镜慢放。林莫双膝跪地，棍身断裂飞散，秦轩收刀入鞘，刀身暗金纹路最后一次闪亮后暗去。
5.5-7秒 全景，固定镜头。演武场尘埃缓缓沉降，两人剪影一立一跪，月光重新穿透扬尘形成光束。
```

### 空镜（无人）
```
不要字幕，不要背景音乐。电影级柔光，暖金色黄昏色调，浅景深，4K

0-4秒  全景，固定镜头。废弃演武场，夕阳穿过枯树投下长影，落叶在石缝间被风推着缓慢移动。
4-8秒  中景，缓慢横移。石质地面纹路被暖金色光打亮，浮尘在光束中缓慢飘动。
8-12秒 近景，缓推。干涸的剑痕刻在石柱上，光影缓慢滑过凹槽纹理。
12-15秒 远景，固定镜头。暮色渐深，演武场隐入暗蓝暮霭，远处一盏孤灯亮起。
```

## 规则

| # | 规则 |
|---|------|
| 1 | 3块结构：头部锚定（一行）→ 引用+站位（有素材时一行）→ 时间线分镜（逐行） |
| 2 | 块1写：不要字幕+不要背景音乐+摄影规格+风格+光调+氛围+色调+分辨率 |
| 3 | 块2写：@图N角色（站位+朝向+手持物+状态），场景。无素材时空镜跳过此块 |
| 4 | 分镜格式：`X-Y秒 景别，运镜。动作。光效事件。` — 一行一个镜头 |
| 5 | 光效不另起行、不写"光影："——写成画面事件嵌入动作描述 |
| 6 | 文戏时间段3-5秒/镜，打斗时间段1-2.5秒/镜 |
| 7 | 台词用反引号包：\`台词内容\` |
| 8 | 有@图引用的角色不写外貌，只写站位+动作状态 |
| 9 | 战斗类先查项目设定（技能锚定/怪物设定/击杀机制/装备掉落） |

## 景别速查

| 缩写 | 中文 | 可见范围 |
|------|------|---------|
| EWS | 大远景 | 环境全貌，人物极小 |
| WS | 远景 | 完整环境+人物全身 |
| FS | 全景 | 人物全身+部分环境 |
| MS | 中景 | 腰部以上 |
| MCU | 中近景 | 胸部以上 |
| CU | 近景 | 肩部以上 |
| ECU | 特写 | 面部/手部/道具细节 |

## 运镜速查

| 类型 | 运镜 | 适用场景 |
|------|------|---------|
| 静态 | 固定镜头 | 文戏/对话/情绪/对峙 |
| 缓动 | 缓推/缓拉 | 情绪递进/揭示/收尾 |
| 跟随 | 跟拍/手持晃动 | 走动/追逐/打斗 |
| 环绕 | 环绕/摇镜 | 对峙/蓄力/展示空间 |
| 冲击 | POV/甩镜/震屏/crash zoom | 重击/惊吓/爆发节点 |

## 与旧格式（v2.0 5层）对比

| 项目 | 旧格式 | 新格式 |
|------|--------|--------|
| 字段数 | 5大段 + 每镜4字段 | 3块，分镜一行一个 |
| Token量 | ~1500-2000/段 | ~500-800/段 |
| 光效 | 独立行 `光效：xxx` | 嵌入动作描述，画面事件 |
| 技术参数 | 焦段/光圈/ISO/ND（模型不认） | 已删除 |
| 开头 | `1.【要求】` 两行固定 | 头部一行全包 |
| 分镜 | 景别/角度/镜头/内容 拆分 | 景别，运镜。内容。 揉一行 |
| 打斗格式 | 同文戏，规则靠检查清单 | 时间段更碎，运镜动态，自然区分 |
