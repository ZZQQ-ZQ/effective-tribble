### 一、三层审核机制（来源：Vicsee / X@YTkexue / Morphic）

| 层级 | 触发时机 | 特点 |
|------|---------|------|
| **① 提示词过滤** | 提交即判 | 基于关键词模式匹配，**非语义理解**。英文比中文严格得多 |
| **② 图片扫描** | 上传图片时 | 独立于提示词的CV人脸检测，真人照上传即拒 |
| **③ 输出过滤** | 生成后 | 偶发，视频生成完才判违规（最坑，白耗积分） |

### 二、硬性拦截（永远过不了，勿浪费）

- ❌ **真人人脸参考图**（deepfake责任）/ 真人照片 → 替换为AI生成头像或插画
- ❌ **知名IP角色** — Iron Man / Spider-Man / Disney角色，**连视觉化描述都拦截**（"glowing chest piece + energy beams + flying"被识别为Iron Man）
- ❌ **真实品牌名/logo**
- ❌ **直白的极端暴力**（stabbing/shooting/graphic injury）
- ❌ **NSFW内容**
- ❌ **"真人"二字** — 替换为"人物"或"写实"

### 三、英文关键词替换表（来源：Vicsee 实测数百条提示词）

**核心原则：描写画面效果，不写行为本质**
- `attack / fight / punch / kick` → `impact / force / momentum / collision`
- `shoot / fire / weapon` → `muzzle flash / tactical equipment`
- `blood / wound / injury` → `crimson liquid / surface damage`
- `explode / destroy` → `rapid expansion / structural failure`
- `kill / death / die` → `final moment / stillness / consequence`
- `young / child / kid / boy / girl` → `small figure` / 用图片承载身份，文本只写镜头所见

### 四、中文已知触发词（来源：知乎/抖音/X实测）

- **"真人"** → 替换为"人物"或"写实"
- **"无数"** → 来自知乎吐槽帖，被列入违禁词
- **"少女"/"儿童"/"小孩"** → 触发未成年人保护加强审核
- **"tiktok"** → 100%拦截（@YTkexue 团队大量测试确认）
- **"死"/"杀"/"血"** → 直白写法必封
- **"裸"/"露"** → NSFW硬拦截
- **"枪"/"武器"** → 英文soft block，中文同理

### 五、软性拦截（有时过有时不过）

- **动作戏** → 手打脚踢比武器射击容易过；动画/风格化比真人写实容易过
- **武器** → "holding a weapon" 直接拦截 / "metallic equipment in tactical gear" 可能过
- **黑暗/恐怖** → 怪物/生物类基本都能过，真人受害者场景不行
- **知名IP形象** → 轻微改写的描述也可能被模式识别

### 六、核心规避策略（全网共识）

**1. 用电影术语当护盾**
提示词里加 2-3 个拍摄术语（镜头类型/运镜/布光/胶片格式），过滤器会把它判为"制作说明"而非暴力描述。同一个场景：
- ❌ `a soldier shoots someone in the street`
- ✅ `wide shot, 1940s street, soldier fires toward off-screen, smoke, overcast light, 35mm grain, documentary handheld`

**2. 补全场景语境，不只写动作**
过滤器需要判断"这是片场还是在描述真实事件"。四点必答：地点在哪 / 画面什么样 / 镜头在做什么 / 整体氛围是什么

**3. 中文提示词通过率 > 英文**
字节的安全词库主要基于英文词汇，中文提示词落在不同的过滤阈值上。场景用中文写，对话保留英文

**4. 图片扛意图，文本打掩护**
有参考图时提示词刻意模糊——图片负责角色外貌，文本只写镜头所见

**5. 风格化 > 写实**
- 真人写实 → 最严
- 3D渲染 → 中等
- 2D动画 → 较松
- 抽象/风格化 → 最松

**6. 重试策略**
同一提示词有时第2-3次就过了，过滤器有概率成分。边缘提示词先重试再改词
