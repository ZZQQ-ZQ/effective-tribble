一、 核心方法论：导演的“三个问题”
在规划镜头前，导演会先思考三个核心问题：
1. 场景的主体是什么？（人物/场景/物品）
2. 想让观众感受到什么情绪？
3. 什么角度最能表达这种感觉？
二、 基础框架：人物出场
文档提出了一个通用的提示词框架：视角+人物+动作+镜头运动+氛围。
1. 英雄出场 (Epic Hero Entrance)
• 核心技巧： 低机位仰拍(Low Angle Shot) + 镜头缓慢推进(Slowly Pushing In)。
• 效果： 塑造人物的强大、威严和史诗感。
• 案例： 沙漠牛仔英雄登场。从马蹄特写开始，低机位向上倾斜推进，掠过马腿、风衣，最后定格在脸部特写，利用逆光勾勒轮廓。
2. 神秘人物出场 (Mysterious Character Entrance)
• 核心技巧： 背面角度 + 遮挡式构图 + 缓慢推进。
• 效果： 制造悬念、神秘感和压迫感。
• 案例： 沙漠牛仔进入酒馆。镜头从背影开始，随人物向内走动缓慢推进，保持面部被帽檐阴影遮挡，最后在进入昏暗室内时，面部被阴影覆盖，增强神秘感。
3. 故事感出场 (Story-driven Entrance)
• 核心技巧： 平视角度 + 中景 + 手持跟拍 + 环绕运镜(Orbit)。
• 效果： 展现人物与环境的关系，增强生活气息和代入感。
• 案例： 牛仔牵马走进小镇。镜头采用手持轻微晃动，并通过环绕运镜，从不同角度捕捉人物行走过程中的街景细节（行人、商铺等），营造真实的生活氛围。
三、 核心场景：人物对话
1. 双人中景 (Two-Shot)
• 作用： 清晰交代人物关系和空间位置，强调互动和氛围。
• 案例： 杰洛特和希里在废墟对话。通过高位或低位环绕运镜，在保持两人位置不变得同时，展示周围燃烧的城镇等环境信息，增强紧张感和保护感。
• 注意： 固定不动的双人中景会显得画面平淡，代入感弱。
2. 过肩反打 (Over-the-Shoulder Shot)
• 作用： 切换对话视角，让观众仿佛置身于对话中，代入感极强。是避免对话场景平淡的常用技巧。
• 案例： 夜晚森林篝火旁的对话。镜头在希里肩膀后方（前景虚化）和杰洛特肩膀后方之间切换，观众可以分别感受两人的表情和反应，氛围安静而富有代入感。
四、 人物情绪表达
1. 俯视角度 (High Angle / Bird's-eye)
• 情绪： 渺小、无助、被压制、脆弱。
• 案例： 受伤男人被枪指着。镜头从平视缓慢提升至俯视，随着视角升高，男人的渺小与无助感逐渐增强。
2. 背面角度 (Over-the-Shoulder / Back Shot)
• 情绪： 孤独、疏离、未知、身临其境。
• 效果： 让观众“成为”角色，与角色一起面对画面中的事物。
• 案例： 暴风雪中的女子。镜头在屋内从背面拍摄她走入风雪，观众只能看到她逐渐消失在漫天风雪中的背影，孤独与未知感强烈。
3. 斜角镜头 (Dutch Angle)
• 情绪： 不稳定、恐惧、心理失衡、紧张感。
• 案例： 夜晚小巷中的女孩。倾斜的镜头配合闪烁的路灯，营造出强烈的心理紧张和惊悚氛围。
4. 侧面特写 (Side Profile Close-up)
• 情绪： 沉思、冷静、压抑、内心活动。
• 效果： 强调轮廓和光影对比，弱化直接表达，引导观众关注内在情绪。

5. 极近特写 (Extreme Close-up)
• 情绪： 强化任何一种情绪，放大细节，极具冲击力。

1. 鸟瞰/航拍 (Bird's-eye / Aerial Shot)
• 效果： 展示宏大与渺小的对比，提供“上帝视角”，清晰展现空间布局。通过由近及远或由远及近的动态运镜，制造强烈视觉冲击。
• 案例： 太空中的宇航员（宏大宇宙与渺小个体）、谈判桌（冷静对峙与棋盘格局）、地震废墟中的小女孩（从个体特写拉升至全局废墟）。
2. POV/FPV 视角 (Point of View / First Person View)
• 效果： 强烈的代入感和沉浸感。可以模拟任何主体的视角（人、动物、甚至无人机），创造独特的视觉体验。
• 案例： 跟随猫咪在巷子中穿梭、模拟骑着猫咪的小人视角、FPV无人机穿越机场车队、第一视角从高空俯冲进入森林和湖面。
3. 虫眼视角 (Worm's-eye View)

七、 进阶技巧
1. V.I.D.E.O. 框架 (提示词优化法) 这是一个结构化的提示词编写方法，将复杂指令拆分为五个模块：
• V (Visuals) 视觉背景与基调： 场景环境、时间、天气、光影。
• I (Intent) 叙事意图与镜头指令： 核心部分，描述镜头角度和运动，应放在提示词最前面。
• D (Details) 画面细节补充： 微观纹理、辅助元素（如汗珠、灰尘）。
• E (Emotion) 情绪渲染与氛围： 核心心理暗示或情感基调。
• O (Output) 技术格式输出： 分辨率、帧率、胶片质感等。
策略：
• 前置核心权重： 将镜头角度和景别（I部分）放在句首。
• 描述效果而非命令： 写“主体填满画框”而非“拉近镜头”。
• 引用大师风格： 如“希区柯克式变焦(Dolly Zoom)”可触发高质量视觉模式。
• 避免否定句： 用“稳定静态镜头(Steady static shot)”替代“镜头不晃动”。
2. 180度法则 (The 180-Degree Rule)
• 定义： 在拍摄两人对话时，在他们之间画一条假想的轴线，摄影机应始终保持在轴线的一侧。
• 作用： 维持角色在画面中的左右位置和视线方向一致，避免观众对空间关系产生困惑。
• 破坏规则（越轴）： 只有通过连续的、一镜到底的摄影机运动（长镜头）跨越轴线，才是被接受的技法。
3. 30度法则 (The 30-Degree Rule)
• 定义： 当剪辑连接对同一主体的两个连续镜头时，摄影机角度变化至少需要30度。
• 作用： 确保切换流畅，避免产生不自然的“跳切(Jump Cut)”。
• 建议操作： 最好在改变角度的同时，也改变景别（如从中景切到特写）。
