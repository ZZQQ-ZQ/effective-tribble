## 2026-07-21 11:47 — 语速/语气控制订正

### 核心订正
**之前错误理解**：以为通过 Prompt 可以控制 AI 角色说话的语速（写"语速极快"/"慢条斯理"/按字数÷语速公式分配时间）。

**正确理解**：
- AI视频模型（Seedance/Kling/Runway）中，语音和视频是分离的
- 语速/语调/停顿 = 由上传的音频文件决定
- 视频 Prompt 只能做：视觉画面描述 + 口型同步
- 正确工作流：TTS/录音 → 调整语速 → 生成音频 → 上传@audio → 口型同步
- 视频层只能通过动作/表情/说法动词暗示语气，不能控制实际音频的速度
