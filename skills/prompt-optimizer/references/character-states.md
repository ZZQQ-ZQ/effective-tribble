# AI人物状态提示词手册

> 2026-06-25 入库。解决AI角色"蜡像感"——通过身体姿态描写让角色拥有真实的情绪和演技。

## 黄金法则

**先写身体姿态，再写面部表情。** 身体先有状态，脸上才有反应。不反过来。

---

## 八种核心人物状态

### 01. 疲惫
- 姿态：眼皮半垂，肩膀塌下，背部微弯，手臂自然下垂，步伐拖沓
- 核心：身体沉重感和无力感
- 提示词写法：`眼皮半垂，肩膀塌下，脚步沉重拖沓，手臂无力垂在身侧`

### 02. 放松
- 姿态：肩颈自然下沉，手臂轻松垂放，呼吸平稳，脸部肌肉完全放松
- 核心：肌肉松弛+呼吸平稳
- 提示词写法：`肩颈自然下沉，手臂轻松垂放，呼吸平稳，脸部肌肉松弛`

### 03. 警觉
- 姿态：身体微微前倾，肩膀收紧，眼神快速扫向四周，随时准备反应
- 核心：紧张感+对环境高度关注
- 提示词写法：`身体微前倾，肩膀收紧，眼神快速扫视四周，手指微屈随时准备拔剑`

### 04. 专注
- 姿态：目光固定在前方一点，眉头轻收，嘴唇闭合，身体稳定少晃动
- 核心：注意力集中+身体静止
- 提示词写法：`目光锁定前方一点，眉头轻收，嘴唇闭合，身体稳如磐石`

### 05. 自信
- 姿态：背部挺直，肩膀打开，下巴轻抬，目光直视前方，站姿稳定
- 核心：开放的肢体语言+坚定的眼神
- 提示词写法：`背部挺直，肩膀打开，下巴微抬，目光直视前方不闪躲`

### 06. 虚弱
- 姿态：脸色发白，身体轻微摇晃，一只手扶住身侧，呼吸变浅，动作无力
- 核心：摇晃+无力动作=病态
- 提示词写法：`脸色发白，身体微晃，单手扶住身侧支撑，呼吸浅而急促，动作绵软无力`

### 07. 慌张
- 姿态：眼神来回闪动，嘴唇微张，双手抓紧衣角，身体不稳定
- 核心：小动作（抓衣角）+眼神游移=内心慌乱
- 提示词写法：`眼神来回闪动不定，嘴唇微张，双手抓紧衣角指节泛白，身体重心不稳`

### 08. 从容
- 姿态：站姿稳，肩膀放松，手部动作缓慢自然，目光平静，节奏不急不慢
- 核心：动作缓慢+心态平和
- 提示词写法：`站姿稳健，肩膀放松下垂，手部动作缓慢自然，目光平静如水`

## 在5层格式中的用法

人物状态不单独写字段，融入【资产】→人物描述和【分镜序列】→内容中：

**【资产】→人物**：
```
秦（画右，身体微前倾，肩膀收紧，右手按在剑柄上，警觉扫视画左暗处）
```

**【分镜】→内容**：
```
秦眼皮半垂，肩膀塌下，沉重的吸气让胸甲微微起伏，右手剑尖斜点地面——打完一场的疲惫不写在脸上，写在身体的重量里
```

## 与表情规则的关系

- **表情压着写**（SOUL.md 规则）：表情用"微扬/淡淡/一闪而过"缓冲
- **身体先于表情**（本手册规则）：先写身体姿态变化，再写脸上微反应
- **二者不矛盾**：身体是大动作（可以写实），表情是微反应（压着写）
