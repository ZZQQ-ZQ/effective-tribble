# 真实感增强技能

> **创建**：2026-07-15 | 全网调研后整理
> **核心理念**：真实感 = 不完美 + 物理正确 + 微观细节。越是「完美」的画面越假，越是「有瑕疵」的画面越真。

---

## 一、八维真实感框架（8-Category Framework）

每个提示词至少覆盖其中4维，追求极致真实感时覆盖全部8维。

| # | 维度 | 核心关键词（Seedance可直接用） | 作用 |
|---|------|---------------------------|------|
| 1 | **真实感触发器** | `photorealistic` `真人写实风格` `真实世界摄影` `lifelike details` | 告诉模型走写实路线，不走风格化 |
| 2 | **摄影机语言** | `35mm镜头` `50mm标准镜头` `85mm人像镜` `浅景深` `自然散景` | 模拟真实光学行为 |
| 3 | **光影物理** | `自然光` `柔光窗光` `黄金时刻` `侧光` `软箱光` `轮廓光` | 光的行为必须符合物理 |
| 4 | **材质纹理** | `可见毛孔` `皮肤微纹理` `面料纹理` `金属拉丝` `皮革褶皱` `粗糙度` | 去除塑料光滑感 |
| 5 | **不完美注入** | `轻微不对称` `自然瑕疵` `微尘` `细小划痕` `真实世界磨损` | 真东西都有瑕疵 |
| 6 | **色彩控制** | `自然饱和度` `胶片色彩` `肤色自然` `无过度HDR` | 避免饱和度爆炸 |
| 7 | **颗粒/质感** | `35mm胶片颗粒` `轻微噪点` `有机质感` | 模拟实拍感 |
| 8 | **排除项** | `避免塑料感` `避免CGI渲染感` `避免过度光滑` `避免娃娃脸` | 排除AI常见翻车特征 |

## 二、人物真实感（三层递进）

### 2.1 第一层：皮肤质感（去塑料感）

**问题根源**：AI默认输出皮肤过于光滑，像塑料/蜡像/游戏角色。

**解决关键词（直接追加到提示词人物描述中）：**

```
皮肤微纹理可见，毛孔细节保留，自然肤质不磨皮，面部有轻微细纹和自然瑕疵，真实人类肌肤质感，无塑料感，无蜡像感，无CGI渲染光滑表面
```

**不同场景下的皮肤质感写法：**

| 场景 | 追加关键词 |
|------|-----------|
| 近景/特写（面部） | `可见毛孔，皮肤微纹理，自然细纹，轻微油光，真实肤质不磨皮` |
| 中景（上半身） | `皮肤质感自然，光影在肌肤上过渡柔和，无明显塑料反光` |
| 全景/远景 | `人物与场景光影融合自然，无悬浮感，不做额外皮肤处理` |
| 打斗/运动 | `汗水在皮肤表面，面部泛红，血管微凸，肌肉绷紧的皮下纹理` |
| 老年人 | `皱纹深刻，皮肤松弛，老年斑，岁月痕迹，真实衰老质感` |
| 受伤/战损 | `伤口边缘红肿，血迹干涸纹理，尘土附着皮肤，不完美愈合疤痕` |

### 2.2 第二层：微表情（去木偶感）

**问题根源**：AI默认表情制式化、眼神空洞、面部肌肉完全静止。

**万能基础微表情（所有有人物的提示词必加）：**

```
自然微表情，眼神灵动有神，缓慢自然眨眼，眼球轻微转动，嘴角若有若无微扬，眉部嘴角有微弱肌肉起伏，面部肌肉自然放松不僵硬，杜绝木偶脸
```

### 2.3 第三层：自然动作（去机器人感）

**问题根源**：AI动作僵硬、双脚滑步、身体悬浮、手指变形。

**动作真实感核心规则：**

| 规则 | 说明 | 提示词写法 |
|------|------|-----------|
| **重量转移** | 每步写清重心落在哪只脚 | `重心从后脚转移到前脚掌，身体随步伐微微起伏` |
| **接触反馈** | 写清身体与物体的互动 | `手掌按在石柱上，指节微微发白，掌心感受到冰冷粗糙` |
| **次级运动** | 主动作之外的跟随运动 | `转头时发丝迟半秒跟随，衣摆随步伐惯性摆动` |
| **呼吸可见** | 情绪重时呼吸带动身体 | `胸膛起伏喘气，肩头随呼吸上下耸动` |
| **关节链** | 动作从核心到末端的顺序 | `腰胯先转→肩膀跟随→手臂甩出→手腕发力→五指收拢` |
| **缓冲结束** | 所有爆发后有缓冲 | `落地时膝盖弯曲缓冲，不是僵直砸地` |

**5类高频动作提示词模板：**

```
# 走路（防滑步）
一步一步走，脚后跟先着地再过渡到前掌，重心自然转移，身体有节奏地微微起伏

# 坐下（防悬浮）
缓缓坐下，身体重量沉入椅子，坐垫微微下陷，靠背承接后背

# 转头（防崩脸）
缓慢转头，从左到右，发丝跟随头部运动延迟半秒，颈侧肌肉轻微拉伸

# 握物（防手指变形）
五指自然弯曲握住杯身，指腹贴紧瓷面，拇指搭在杯沿，力度适中不僵硬

# 奔跑（防浮空）
全力冲刺，每一步脚掌蹬地后膝盖高抬，手臂以肩为轴前后快速摆动，汗水从额角甩出
```

## 三、场景真实感

### 3.1 PBR材质关键词（取代"电影感"）

**核心理念**：2026年AI视频领域——"电影感"已经烂大街。真正拉开差距的是材质细节：
水珠、气泡、冷雾、冰晶、液体高光、金属反射、玻璃折射。

**六大材质真实感词库：**

| 材质 | Seedance关键词 |
|------|---------------|
| **金属** | `金属拉丝纹理，表面有细微划痕和磨损，真实反射环境光，非镜面高光，氧化斑点` |
| **石材** | `石材表面粗糙多孔，风化痕迹，苔藓斑驳，边缘磨损圆角，自然纹理不规则` |
| **木材** | `木纹清晰且不规则，表面有细微裂纹，漆面半哑光，边缘有岁月使用痕迹` |
| **布料** | `面料纹理清晰可辨，褶皱自然垂坠，缝线可见，不同材质光泽差异明显（丝绸高光vs棉麻哑光）` |
| **皮革** | `皮革表面细微毛孔，折痕自然，磨损处颜色变浅，缝线针脚可见` |
| **液体** | `水面微波动，液体表面张力弯月面，气泡自然上升，水珠在表面滚动留有水痕` |

### 3.2 环境细节注入（去空旷感）

```
# 空气质感（必加）
空气中漂浮微尘，光束穿过扬尘形成丁达尔效应，远处有薄雾降低对比度

# 地面（防悬浮的最强锚点）
地面有散落的碎石碎叶，积水反射天空，脚印/血迹/拖痕延续前后帧一致

# 墙面/建筑
墙面有裂缝和污渍，油漆剥落露出下层，墙角有蜘蛛网，窗框生锈

# 道具/摆件
桌面有使用痕迹，纸张微卷，墨水渍，蜡烛烧了一半蜡泪凝固
```

### 3.3 光影物理正确（防画面扁平）

```
# 光源必须具体
光从画右上方窗户斜射入 → 对应阴影落在人物左侧地面
逆光 → 人物边缘轮廓光，面部为柔和辅光填充
顶光 → 眼窝鼻下有明显向下投影

# 多光源必须写清主次
主光源：吊灯暖黄光（主），辅光源：窗外冷蓝月光（次），
人物面部有明显冷暖双色光过渡

# 反射/折射
水面倒影与实物上下对称但略有扭曲，玻璃窗有微弱反光看到室内
```

## 四、不完美法则（Imperfection Doctrine）

**核心理念：真实世界没有完美的东西。AI生成的最大破绽就是"太完美"。**

### 4.1 六维不完美注入

| 维度 | 注入方式 | Seedance写法 |
|------|---------|-------------|
| **对称性** | 人脸轻微不对称 | `面部轻微自然不对称，左右眉眼并非完全镜像` |
| **表面** | 所有表面有使用痕迹 | `表面有自然磨损痕迹，非出厂全新状态` |
| **光影** | 光线有衰减和散射 | `光线在空间中有自然衰减，远处偏暗偏冷` |
| **色彩** | 颜色不完全均匀 | `色彩有轻微色差和褪色，非均匀着色` |
| **运动** | 动作有犹豫和抖动 | `动作有轻微的起始犹豫和惯性过冲，非机械精准` |
| **环境** | 环境有随机元素 | `远处有不相关的随机人物/物品，增加现场随机感` |

### 4.2 不完美速查卡（追加到提示词末尾）

```
# 人物不完美
面部轻微不对称，皮肤有真实纹理和细小瑕疵，非CGI完美光滑表面

# 场景不完美
环境有日常使用的磨损痕迹，地面有随机散落的杂物，墙面有岁月斑驳

# 光影不完美
光线自然衰减，阴影边缘柔和非锐利切割，高光有自然扩散

# 动作不完美  
动作有自然节奏而非机械匀速，力量有爆发和衰减，肌肉有真实紧张和放松交替
```

## 五、动作真实感（Motion Realism）

> 参考来源：Kling 3.0 60次运动测试 + Seedance实战

### 5.1 AI动作三大死穴及解法

| 死穴 | 表现 | 根源 | 提示词解法 |
|------|------|------|-----------|
| **滑步** | 脚不沾地飘着走 | 模型不知道脚怎么落地 | `脚后跟先着地，重心从脚跟过渡到前掌，踏地有轻微扬尘` |
| **悬浮** | 身体不与物体接触 | 模型不知道重量和接触 | `身体重量沉入椅子，坐垫下陷，手掌按在桌面上指腹受力微白` |
| **手指变形** | 手指融合/错位/数量错误 | 模型对手指拓扑理解差 | `五指自然分开，关节清晰可辨，指尖做抓握动作，手腕带动手指` |

### 5.2 次级运动（Secondary Motion）注入

**主动作之外必须写跟随运动，否则就是机器人：**

```
# 头发
转头 → 头发延迟半秒跟随，发梢甩出弧线后回落

# 衣物
奔跑 → 衣摆和袖口随风向后飘扬，落地时衣物因惯性向前荡再回摆
突然停下 → 衣物继续向前荡再落回，不是瞬间静止

# 配饰
跳跃 → 项链/耳坠随身体腾空上飘，落地后因重力下坠再小幅回弹
```

### 5.3 运动物理锚点

```
# 起跳
膝盖弯曲蓄力 → 脚掌蹬地爆发 → 身体直线上升 → 衣物因加速度向下拉扯 → 到达顶点悬浮一瞬 → 开始下落

# 落地
脚尖先触地 → 膝盖弯曲缓冲 → 身体继续下沉一截 → 衣物跟随下落再回弹 → 站稳

# 打击
武器挥出 → 命中瞬间顿帧 → 冲击波从命中点扩散 → 被击方身体变形凹陷 → 向后飞出 → 撞击表面 → 反弹停住
```

## 六、负面提示词库（Negative Prompts）

> 追加到每段提示词末尾，排除AI常见翻车特征。

### 6.1 人物负面词

```
避免塑料感，避免蜡像皮肤，避免CGI渲染光滑表面，避免面部特征漂移，
避免形态变形，避免动作僵硬不自然，避免木偶脸，避免眼神呆滞空洞，
避免假笑，避免表情僵硬，避免娃娃感，避免3D模型感，避免游戏角色感，
避免卡通感，避免过度完美对称，避免悬浮感，避免双脚滑步，避免手指变形融合
```

### 6.2 场景负面词

```
避免场景空旷无细节，避免建筑像刚建成的完美状态，
避免地面干净无杂物，避免光影扁平无层次，
避免物体像刚出厂无使用痕迹，避免材质光滑无纹理，
避免水面像镜子无波动，避免天空纯色无云层变化
```

## 七、分段生成策略

> 来源：全网共识——长视频崩脸概率随秒数指数上升

| 策略 | 说明 |
|------|------|
| **3-5秒一段** | 降低崩脸概率，每段只做1-2个动作 |
| **统一参考图** | 全段用同一张定妆照作为角色参考，不换图 |
| **约束词每段重复** | `全程人物一致、五官不变形、服饰细节不缺失、体态统一` |
| **光影统一** | 同一场景内光线方向、色温、明暗程度不跳变 |
| **先慢后快** | 新手从缓慢简单动作入手，熟练后再加大幅度动作 |

## 🔴 铁律

1. **真实感来自细节，不来自形容词** — `8K超写实`写100遍不如`毛孔可见`写1遍
2. **不完美 > 完美** — 越完美越假，越有瑕疵越真
3. **物理锚点必写** — 脚怎么落地、身体怎么接触物体、重量怎么转移
4. **次级运动不能省** — 头发/衣物的跟随运动是活的vs机器人的分界线
5. **每段追加负面词** — 不依赖模型自己避开，主动排除
6. **皮肤质感词只在近景/特写写** — 全景写毛孔浪费token还干扰模型
7. **光影光源必须可追溯** — 画面里每一束光都要有物理来源
