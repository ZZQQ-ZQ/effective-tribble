# 提示词规范技能

> 🔴 出提示词前唯一必过的内容决策门。本技能锁定：该写什么 / 不该写什么 / 违禁词审查。

---

## 一、必须写的六要素（缺一不可）

| 优先级 | 要素 | 写什么 | 写法 |
|--------|------|--------|------|
| 🔴 P0 | 主体 | 谁、画左/右、朝向、手持物 | 「秦（画左，侧身面对镜头，右手垂握大剑）」 |
| 🔴 P0 | 动作 | 干什么、终点状态 | 写终点不写过程。「飞出去撞在石柱上」≠「被击飞翻滚摔落滑行」 |
| 🔴 P0 | 场景 | 在哪、空间锚点 | 「大殿内，石柱林立，穹顶高耸」 |
| 🟡 P1 | 光线 | 光源方向+光质+色温 | 「侧逆光，柔光罩，色温3200K」→不用形容词堆「温暖琥珀色夕阳」 |
| 🟡 P1 | 运镜+景别 | 每镜必写 | 「近景，固定镜头」/「中景，手持跟拍」 |
| 🟡 P1 | 约束词 | 明确不要什么 | 「不要字幕，不要背景音乐」 |

## 二、十大禁止写的内容

### 🔴 硬性禁令（写出=废稿）

| # | 禁止 | 原因 | 正确做法 |
|---|------|------|---------|
| 1 | 有参考图的角色写外貌/服装/材质 | 图片已有信息→重复写入=冲突+浪费 | 只写位置+朝向+动作状态 |
| 2 | 数值参数（f/2.8, ISO 800, 快门角度180°） | Seedance不认 | 删除，用「浅景深」「高速快门感」替代 |
| 3 | 情绪解读词 | 画面看不到。「内心充满悲伤」→废话 | 写身体状态：「眼眶泛红，泪光闪烁」 |
| 4 | 抽象概念（轴线/越轴/180度规则） | 模型不认 | 写物理坐标：「画左」「画右」「视线向右」 |
| 5 | 暗部类字眼（暗部/压黑/阴影压死/幽暗/黑暗/暗黑/昏暗） | 死规则2·永久禁令 | 用「冷灰」「冷蓝灰」「阴影区」「未受光区域」 |
| 6 | 模糊@引用 | 「参考@视频1」→模型不知道参考什么 | 「参考@视频1的运镜」「参考@图片2的场景」 |
| 7 | 矛盾指令 | 同一镜写「固定镜头」+「环绕镜头」=模型崩溃 | 一镜一种运镜 |
| 8 | 中间过程 | 模型自己补过渡，写终点就够 | 「落在地上」≠「跃起→空中翻滚→坠落→着地」 |

### 🟡 软性禁令（写出=降低质量）

| # | 禁止 | 原因 | 来源 |
|---|------|------|------|
| 9 | 内容过载（4-5秒塞3个场景） | 物理不可行，模型随机跳过 | Dexhunter官方 |
| 10 | 形容词堆砌光影 | "温暖柔和的琥珀色夕阳"=模型不懂 | 光源三问：方向+光质+色温 |

## 三、内容平衡公式

### 3.1 甜点区

```
太少 → under-prompting → 模型乱猜 → 物体残缺/形态错误
太多 → over-prompting → 模型忽略指令 → 随机跳过 → 结果失控
甜点 → 主体+动作+场景+光线+运镜+约束 六要素
```

| 来源 | 核心发现 |
|------|---------|
| JSFILMZ | 平衡公式：不是越详细越好，也不是越简单越好 |
| Reddit r/seedance | 提示词越长越差，~80词是甜点区 |
| Reddit r/PromptEngineering | 停止写段落！模型不奖励冗长 |
| Emily2040技能 | 一镜一意图：一个镜头只做一件事 |

### 3.2 铁律

| 规则 | 说明 |
|------|------|
| 一镜一意图 | 不塞「攻击+反击+再被打」多层因果 |
| 一镜一运镜 | 不堆叠「推+摇+环绕」 |
| 写终点不写过程 | 「落在地上」够用，不写10步分解 |
| 图片说了就不写 | 参考图含的信息→提示词0重复 |

## 五、出稿前强制检查清单

逐条勾选，全过才输出：

### 内容决策检查
- [ ] 六要素齐全（主体/动作/场景/光线/运镜/约束）
- [ ] 有参考图的角色未写外貌服装材质
- [ ] 无数值参数（f值/ISO/快门角度）
- [ ] 无情绪解读词（全换成身体状态描述）
- [ ] 无抽象概念（全换成物理坐标）
- [ ] 全篇无暗部类字眼
- [ ] 每镜一种运镜（不堆叠）
- [ ] 每镜只做一件事（不塞多层因果）
- [ ] 时长相匹配（不4秒塞3场景）
- [ ] @引用全部标注用途
- [ ] 光线写成物理光源（方向+光质+色温，不用形容词堆）

### 违禁词检查
- [ ] P0四红线全篇清零
- [ ] 暴力词已替换为中性描述
- [ ] 灵异/迷信词已科幻化处理
- [ ] 封建/修仙词已现代化处理
- [ ] 黑暗/负面词已调整为中性
- [ ] 无政治/历史敏感词
- [ ] 具体人名已替换（项目专用名除外）
- [ ] 歧义词已替换
- [ ] 空词已替换为具体描述
- [ ] 裸动词已锚定力度词+速度词
- [ ] 「真人」「写实」出现≤1次
- [ ] 无「少女」「儿童」「小孩」
- [ ] 无「死」「杀」「血」单独出现
- [ ] 无「无数」「枪」「武器」
- [ ] 英文词已逐词过§13词库
- [ ] 结尾已加 `避免面部特征漂移，避免形态变形，避免动作僵硬不自然`
