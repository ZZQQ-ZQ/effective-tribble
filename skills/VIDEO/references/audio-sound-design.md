# AI漫剧音频音效设计标准化方案

> **来源**：2026-07-14 导演提供 · AI漫剧工业化制作参考
> **核心理念**：结构化公式解决AI生成音频层次混乱问题，分层控制实现人声清晰突出、音频层次分明

---

## 一、通用公式

```
[环境底噪（音量参数）] + [关键特效音（精准出现时间）] + [背景配乐（曲风+音量参数）]
```

**公式解析：**

| 要素 | 说明 | 参数格式 |
|------|------|---------|
| 环境底噪 | 全程持续的底噪氛围音 | 音效类型 + 音量dB |
| 关键特效音 | 精准时间节点的音效事件 | 音效类型 + 出现秒数 + 音量dB |
| 背景配乐 | 情绪调性配乐 | 曲风类型 + 音量dB |

**音量层级规范（保证人声突出）：**

| 层级 | 典型音量 | 说明 |
|------|---------|------|
| 人声（Voice） | 0dB（基准） | 最突出，不可被遮盖 |
| 关键特效音（SFX） | -6dB ~ -12dB | 精准插入，短促有力 |
| 背景配乐（BGM） | -18dB ~ -24dB | 情绪铺垫，不压人声 |
| 环境底噪（Ambience） | -12dB ~ -20dB | 全程持续，营造空间感 |

## 二、常用素材库

### 2.1 环境底噪（Ambience）

| 类型 | 音效关键词 | 适用场景 |
|------|-----------|---------|
| 自然天气 | 淅沥雨声、自然风声、雷鸣远处、雪落寂声 | 户外/天气切换 |
| 都市环境 | 街道人流嘈杂声、车辆驶过、地铁站广播 | 都市/现代场景 |
| 室内环境 | 室内静谧底噪、空调低鸣、钟表滴答、炉火噼啪 | 室内对话/独处 |
| 自然环境 | 鸟鸣虫叫、溪水潺潺、海浪拍岸、树叶沙沙 | 自然/仙侠外景 |
| 战场氛围 | 远处炮火低响、兵器碰撞铁鸣、旌旗猎猎 | 战争/打斗背景 |
| 超自然 | 低频嗡鸣、电流声、空间扭曲声波 | 科幻/奇幻/施法 |

### 2.2 关键特效音（Key SFX）

| 类型 | 音效关键词 | 触发时机 |
|------|-----------|---------|
| 开关动作 | 开门声响、关门沉闷、抽屉拉开、柜子闭合 | 动作发生瞬间 |
| 脚步移动 | 轻重脚步声、草地沙沙声、石板清脆、木地板咯吱 | 人物移动过程 |
| 物品交互 | 信封撕裂声、纸张翻动、杯子落桌、金属碰撞 | 物品接触瞬间 |
| 自然音效 | 轰鸣雷声、暴雨骤降、火焰燃起、碎石滚落 | 环境事件触发点 |
| 物体接触 | 物体触碰声、拳头打击、剑刃格挡、玻璃碎裂 | 物理碰撞瞬间 |
| 能量/魔法 | 能量爆发嗡鸣、光柱贯穿、法力消散、粒子崩散 | 技能释放/特效 |
| 情感音效 | 心跳加速、呼吸急促、耳鸣高频 | 情绪高潮节点 |
| 过渡音效 | 画面翻转、时间跳跃、闪回切入 | 剪接转场点 |

### 2.3 背景配乐（BGM）

| 曲风类型 | 情绪调性 | 适用场景 |
|---------|---------|---------|
| 治愈钢琴独奏 | 温暖/感伤/安静 | 情感对话、回忆闪回 |
| 氛围感弦乐 | 宏大/悲壮/史诗 | 大场景、出征、牺牲 |
| 潮流电子合成乐 | 酷炫/节奏/动感 | 战斗高潮、追逐、都市 |
| 低沉大提琴 | 沉重/压抑/悬疑 | 对峙、阴谋、黑暗场景 |
| 国风笛箫古筝 | 典雅/悠远/空灵 | 仙侠、古风、修行 |
| 打击乐鼓点 | 紧张/激昂/战争 | 行军、战前、冲锋 |
| 无配乐留白 | 纯粹/真实/凝重 | 重要台词、静默时刻 |
| 环境音即配乐 | 自然/沉浸 | 纯自然场景过渡 |

## 三、场景套用模板

### 模板1：室内情感对话

```
环境底噪：室内静谧底噪 + 窗外淅沥雨声（-14dB）
关键特效音：
  · 0s 开门声（-8dB）
  · 3s 脚步声由远及近（-10dB）
  · 8s 杯子轻放桌面（-8dB）
背景配乐：低沉钢琴独奏（-20dB）
```

### 模板2：室外打斗场景

```
环境底噪：自然风声（-16dB） + 远处旌旗猎猎（-18dB）
关键特效音：
  · 1s 拔剑出鞘金属锋鸣（-6dB）
  · 3s 剑刃碰撞铁鸣（-6dB）
  · 5s 拳头打击闷响+骨骼碎裂（-4dB）
  · 7s 坠地震起尘土（-8dB）
  · 10s 能量爆发嗡鸣（-6dB）
  · 13s 白光粒子崩散（-8dB）
背景配乐：打击乐鼓点+氛围弦乐（-18dB）
```

### 模板3：仙侠施法

```
环境底噪：山间风声（-18dB） + 远处瀑布水声（-20dB）
关键特效音：
  · 1s 符阵激活低频嗡鸣（-8dB）
  · 3s 灵光逸散高频闪烁（-10dB）
  · 6s 法印落下能量冲击（-6dB）
  · 9s 法力消散回响（-12dB）
```

### 模板4：悬疑对峙

```
环境底噪：室内静谧底噪（-20dB） + 钟表滴答声（-14dB）
关键特效音：
  · 2s 门缝透光地板咯吱（-10dB）
  · 5s 重重脚步声逼近（-8dB）
  · 8s 金属碰撞桌面（-6dB）
  · 11s 心跳加速鼓点（-4dB）

```

### 模板5：都市夜行

```

  · 0s 霓虹灯管微电流嗡声（-20dB）

```

## 四、实战示范

### 示范：雨夜拆信

```
全程持续淅沥雨声（-12dB）
2秒节点插入信封撕开特效音（-6dB）
背景搭配低沉舒缓的钢琴独奏（-18dB）

效果：人声清晰突出，雨声营造空间感，撕信音精准触发情绪转折，钢琴情绪铺垫不抢人声
```

## 五、设计原则

1. **人声为王**：所有其他音轨不得遮盖人声，人声始终在0dB基准线上
2. **精准触发**：关键特效音必须与画面动作同步，误差<0.5秒
3. **层次递进**：环境底噪 → 背景配乐 → 关键特效音 → 人声，从背景到前景逐层叠加
4. **留白有力量**：关键台词或静默时刻关掉所有音轨（仅保留极低环境底噪），让视觉说话
5. **情绪匹配**：配乐曲风必须与画面情绪一致，不可以用欢快配乐配悲伤画面
6. **音量渐变**：BGM进出做淡入淡出（fade in/out），不做生硬切入切出
