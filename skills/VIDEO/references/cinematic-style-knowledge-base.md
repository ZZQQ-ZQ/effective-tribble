## C1：德国表现主义（German Expressionism）

**年代/起源：** 1920年代德国，《卡里加里博士的小屋》《大都会》

**核心视觉特征：**
- 扭曲的透视和非常规角度——建筑倾斜、空间变形
- 强烈手绘感的光影——阴影被"画"在墙面上，不是物理打光
- 人物肢体夸张变形，面部表情如面具般僵硬或扭曲
- 高反差黑白或极低饱和色调——比Noir更噩梦化

**色彩体系：**
- 主调：深黑 + 病态苍白
- 补色：选择性高饱和原色点缀（血红色、病态绿）
- 反差：极高反差，阴影被拉伸成图形

**光线方案：**
- "画上去"的几何形阴影投影在墙面
- 极低角度底光——将人物面孔扭曲
- 锐利硬光切割空间

**适配题材：** 心理恐怖、噩梦序列、疯癫角色、罪案闪回、超现实桥段

**Seedance提示词片段：**
```
德国表现主义电影风格，扭曲透视空间，几何形阴影投射墙面，高反差黑白影调，人物面孔如面具般僵硬
```

**嵌入PROMPT_FORMAT：**
- 画面基调：`德国表现主义风格，扭曲空间透视，几何形墙面阴影，高反差黑白，如噩梦般不安`
- 色彩与影调：`深黑主调，苍白灰补色，极高反差，阴影拉伸为图形，底光扭曲人脸`

## C2：意大利新现实主义（Italian Neorealism）

**年代/起源：** 1940-50年代，《偷自行车的人》《罗马不设防》

**核心视觉特征：**
- 真实街头场景，非摄影棚——战后废墟、灰扑扑的街巷
- 自然光为主，不做人工布光修饰
- 中景和全景为主，跟人物保持观察距离
- 平民面孔——非职业演员的普通长相，不做美化
- 画面有生活气息但不刻意浪漫——晾晒衣服、玩耍孩子、旧自行车

**色彩体系：**
- 主调：灰白 + 褪色土褐
- 补色：真实环境色，不调色
- 反差：中低反差，没有人工阴影

**光线方案：**
- 完全依托现场自然光（日光、街灯）
- 不用反光板、轮廓光、三点式布光
- 阴天散射光优先——不强调光影对比

**适配题材：** 社会现实短剧、底层人物叙事、战后/灾后重建、写实街巷

**Seedance提示词片段：**
```
意大利新现实主义风格，真实街头场景，战后灰扑扑的街巷，自然散射光，平民面孔不做美化，中景观察距离
```

**嵌入PROMPT_FORMAT：**
- 画面基调：`意大利新现实主义风格，真实战后街头，阴天散射自然光，平民面孔，中景观察距离`
- 色彩与影调：`灰白主调，褪色土褐补色，低反差，自然光无修饰，不做风格化调色`

## C3：特艺七彩史诗（Technicolor Epic）

**年代/起源：** 1930-50年代好莱坞，《乱世佳人》《绿野仙踪》

**核心视觉特征：**
- 极度饱和但自然的色彩——三带彩色工艺产生独特的色彩分离感
- 红色偏品、绿色偏翠、蓝色偏青——每种原色都过饱和但不刺眼
- 油画般温润厚重——不是数字HDR，是染料转印的质感
- 史诗感构图：大远景+剪影+戏剧性天空

**色彩体系：**
- 主调：品红 + 翠绿 + 青蓝（三带彩色染料层）
- 补色：暖金（高光）+ 深棕（暗部）
- 反差：中高反差，暗部保留丰富层次

**光线方案：**
- 好莱坞经典三点式布光（主光+补光+轮廓光）
- 强烈的逆光剪影+金色轮廓线
- 烟雾/灰尘增加空气透视

**适配题材：** 史诗奇幻、历史剧、古装大片、宏大场景建立

**Seedance提示词片段：**
```
特艺七彩电影风格，极度饱和三带彩色，油画般温润厚重，金色逆光剪影，史诗感大远景
```

**嵌入PROMPT_FORMAT：**
- 画面基调：`特艺七彩电影风格，三带彩色高饱和，油画温润，金色逆光剪影，史诗构图`
- 色彩与影调：`品红翠绿青蓝三带主调，暖金补色，中高反差，暗部保留染料层次`

## C4：法国新浪潮（French New Wave）

**年代/起源：** 1958-1960年代，《精疲力尽》《四百击》

**核心视觉特征：**
- 跳切（jump cut）——同一镜头内时间跳跃，故意破坏连续性
- 手持摄影+自然光+真实巴黎街头
- 人物直接对镜头说话（打破第四面墙）
- 随性构图——人物被切到画框边缘、背面跟拍
- 咖啡馆、街道、公寓——日常空间不做美化

**色彩体系：**
- 主调：巴黎灰（黑白时期）/ 60年代柯达胶片暖调（彩色时期）
- 反差：中反差，不做暗房调整

**光线方案：**
- 可用的自然光+现场实用光源
- 极少人工布光
- 窗外日光为主要光源

**适配题材：** 都市青春、文艺对话、咖啡馆/街景戏、叛逆角色独白

**Seedance提示词片段：**
```
法国新浪潮风格，手持摄影，巴黎街头自然光，跳切节奏，随性构图，人物对镜头独白，60年代青春反叛气息
```

**嵌入PROMPT_FORMAT：**
- 画面基调：`法国新浪潮风格，手持摄影，街头自然光，跳切节奏，随性构图，60年代青春反叛`

## C5：意大利西部片（Spaghetti Western）

**年代/起源：** 1960年代，《荒野大镖客》《黄金三镖客》

**核心视觉特征：**
- 极端面部特写——眼睛占满整个宽银幕画框
- 宽银幕荒原远景——人物在画面中如尘埃般渺小
- 长时间静态对峙——镜头在面部大特写和手枪大特写之间切换
- 风沙、汗水、皮革、旧木头的粗粝质感

**色彩体系：**
- 主调：土黄 + 沙褐 + 褪色蓝（广阔天空）
- 补色：暗红（血迹/围巾）+ 深棕（皮革）
- 反差：极高反差，烈日下的浓黑阴影

**光线方案：**
- 烈日直射——极硬的光影切割，面部一半强光一半浓黑
- 低角度黄金时刻阳光
- 室内：百叶窗+木栅栏条纹投影

**适配题材：** 西部片、荒野对峙、孤胆英雄、正邪决斗、荒漠追逐

**Seedance提示词片段：**
```
意大利西部片风格，烈日下的荒原，宽银幕构图，极端面部大特写，风沙粗粝质感，人物如尘埃般渺小立于天地间
```

**嵌入PROMPT_FORMAT：**
- 画面基调：`意大利西部片风格，烈日荒原，宽银幕构图，极端面部特写，风沙粗粝，对峙张力`

## C6：新黑色电影（Neo-Noir）

**年代/起源：** 1960年代至今，《唐人街》《银翼杀手》《七宗罪》

**核心视觉特征：**
- 彩色但故意冷处理——蓝+橙对照或绿+琥珀对照
- 现代城市夜景：湿路面反射光、霓虹+钠灯混合
- 百叶窗投影/窗帘条纹（继承经典Noir的彩色版）
- 都市孤独感——人物在高楼/车流中渺小孤立
- 镜面反射、玻璃幕墙、车窗反光

**色彩体系：**
- 主调：深蓝 + 琥珀橙（Teal & Orange经典组合）
- 补色：暗绿或冷品
- 反差：中高反差，暗部保留细节不压死黑

**光线方案：**
- 混合光源——霓虹蓝/品+钠灯橙/黄+LED白
- 城市环境反射光填充暗部
- 百叶窗/窗帘/栅栏条纹投影（彩色版）
- 人物面部半明半暗（单侧光源）

**适配题材：** 都市犯罪、侦探故事、悬疑惊悚、道德灰色地带叙事

**Seedance提示词片段：**
```
新黑色电影风格，深蓝琥珀撞色，现代雨夜都市，湿路面反射霓虹与钠灯光，百叶窗条纹投影，都市孤独感
```

**嵌入PROMPT_FORMAT：**

## C8：VHS低保真（VHS Lo-Fi）

**年代/起源：** VHS家用录像带美学在2010年代复兴为艺术风格

**核心视觉特征：**
- 模拟视频特有的色彩漂移（品红/绿边缘色散）
- CRT扫描线+磁带头噪点颗粒
- 画面偶尔抖动/跳帧/信号丢失闪烁
- 色彩偏暖偏品，饱和度高但不稳定
- 边缘柔化、细节丢失（模拟介质的低分辨率感）

**色彩体系：**
- 主调：暖品 + 褪色黄绿（模拟视频色域偏移）
- 补色：扫描线黑 + 磁噪白
- 反差：中反差，高光容易过曝溢出

**光线方案：**
- 不考虑光线美学——这是介质效果，叠加在任何实拍内容上
- 内容适合：低照度、家用荧光灯、手持DV机顶灯

**适配题材：** 模拟恐怖、怀旧向内容、伪纪录片恐怖、Y2K美学

**Seedance提示词片段：**
```

```

**嵌入PROMPT_FORMAT：**
- 画面基调：`VHS模拟录像带美学，磁噪颗粒，扫描线，品红边缘色散，画面抖动，低分辨率柔化`
- 色彩与影调：`暖品主调，褪色黄绿补色，中反差，高光VHS过曝溢出，模拟色彩漂移`

## C9：Liminal Space（阈限空间）

**年代/起源：** 2019年网络美学，基于空间心理学的不安感

**核心视觉特征：**
- 空旷的过渡空间——走廊、停车场、空置商场、酒店大堂
- 空间看起来"正常"但轻微不对——灯光偏黄/偏绿、空间过宽或过窄
- 没有人物（或人物极远极模糊）
- 人工光源过剩或不足——荧光灯管嗡嗡亮/角落极暗
- 画面唤起"似乎来过"的既视感和深层不安

**色彩体系：**
- 主调：过饱和暖黄（老旧荧光灯色温偏移）
- 补色：褪色地毯棕+墙壁灰白+出口标志红
- 反差：中反差，暗部角落深但不过黑

**光线方案：**
- 老旧荧光灯管——偏绿/偏黄的刺眼白光
- 空间自身的人工照明系统
- 强烈灯光衰减——中心过亮→四角陷入昏暗
- 潮湿地面微弱反光

**适配题材：** 心理恐怖、存在主义焦虑、梦境序列、超现实转场、空间恐怖

**Seedance提示词片段：**
```
阈限空间风格，空旷的过渡空间，老旧荧光灯偏黄偏绿，空间灯光衰减向四角陷入昏暗，无人却不安的既视感
```

**嵌入PROMPT_FORMAT：**
- 画面基调：`阈限空间风格，空旷过渡空间，老旧荧光灯偏色，灯光向四角衰减，无人既视感不安`

## C10：意大利铅黄电影（Giallo）

**年代/起源：** 1960-70年代意大利，《阴风阵阵》《深夜止步》

**核心视觉特征：**
- 极度大胆的原色应用——纯红、翠绿、宝蓝、品红大面积铺满
- 黑皮手套特写（杀手视角）——Giallo标志性符号
- 高饱和+高反差——画面像被彩色凝胶灯照亮的舞台
- 巴洛克式构图——繁复装饰、镜子、螺旋楼梯
- 血液是"歌剧红"（不是写实暗红，是明亮的舞台血）

**色彩体系：**
- 主调：纯红 + 翠绿 + 宝蓝（三原色直接堆砌）
- 补色：品红 + 黑色（皮手套/阴影）
- 反差：极高反差，色彩饱和度拉满

**光线方案：**
