# Coze (扣子) 适配指南

> **说明**: 本 Skill 是纯文本 prompt 分发方案，Coze 插件要求提供 API 端点（OpenAPI 3.0），无法直接以标准 SKILL.md 格式发布到 Coze 插件商店。以下是在 Coze 中手动搭建多 Agent 工作流的步骤。

## 方案：Coze 多 Agent 工作流搭建

在 Coze 平台中，创建一个 Bot，添加 5 个子 Agent 节点，手动配置每个角色的提示词。

### 步骤 1：创建主 Bot

1. 打开 [Coze 平台](https://www.coze.cn)
2. 创建新 Bot，命名为 `[项目名] 创意制作工坊`
3. 主 Bot 的人设提示词：

```
你是一位创意项目总导演，负责管理和调度5个专业AI子角色：
1. 导演 — 脚本分解、创意构思、分镜
2. 美术指导 — 场景设计、影调方案
3. 摄影指导 — 运镜设计、构图布光
4. 品牌总监 — 品牌策略、合规审核
5. AI艺术家 — 技术可行性、提示词优化

工作流程：
阶段1（创意构思）→ 调用导演
阶段2（视觉设计）→ 导演调用美术指导 + 摄影指导
阶段3（审核确认）→ 调用品牌总监 + AI艺术家
阶段4（终审执行）→ 你做出最终决策
```

### 步骤 2：配置子 Agent 节点

在 Coze 工作流编辑器中，添加 5 个 Agent 节点，使用以下提示词：

#### 导演节点
从 `references/director-agent.md` 复制完整提示词，将 `[占位符]` 替换为实际值。

#### 美术指导节点
从 `references/art-director-agent.md` 复制完整提示词。

#### 摄影指导节点
从 `references/dop-agent.md` 复制完整提示词。

#### 品牌总监节点
从 `references/brand-director-agent.md` 复制完整提示词。

#### AI艺术家节点
从 `references/ai-artist-agent.md` 复制完整提示词。

### 步骤 3：配置工作流

工作流节点连接图：

```
[开始] → [总导演（主Bot）]
              │
              ▼
         [导演节点]
              │
         ┌────┴────┐
         ▼         ▼
   [美术指导]   [摄影指导]  ← 并行
         │         │
         └────┬────┘
              ▼
         [品牌总监]  [AI艺术家]  ← 并行
              │         │
              └────┬────┘
                   ▼
              [总导演节点] → [结束]
```

### 步骤 4：验证与发布

1. 测试工作流：输入一个创意项目需求，观察各节点是否按顺序激活
2. 调整提示词：根据测试结果优化每个节点的提示词
3. 发布 Bot：Coze 商店 / 飞书 / 个人使用

## 注意事项

- Coze 的插件架构与 agentskills.io 的 SKILL.md 格式不兼容
- 本方案使用 Coze 的 Bot 功能（非插件），手动搭建工作流
- 后续如果 Coze 支持 SKILL.md 格式，可以直接上传
