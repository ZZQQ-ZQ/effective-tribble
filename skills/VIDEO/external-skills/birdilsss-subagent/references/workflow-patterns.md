# 工作流模式参考 (Workflow Patterns)

## 模式 A：影视制作模式（标准 4 阶段）

适用于发布会开场视频、品牌TVC、企业宣传片等。

```
阶段1: 创意构思（导演主导）
  总导演 → 导演子agent
    输出：分镜脚本、创意概念
    ↓
  导演 → 美术指导（视觉场景设计）
  导演 → 摄影指导（运镜方案设计）
    ↓
    输出：视觉方案 + 运镜方案

阶段2: 视觉设计（并行）
  美术指导 → 场景视觉描述、影调方案
  摄影指导 → 运镜设计、构图、布光
    ↓
  向导演汇报 → 导演审核确认

阶段3: 审核确认（并行）
  品牌总监 → 品牌策略审核 + 合规检查
  AI艺术家 → 技术可行性评估 + 提示词初稿
    ↓
  向总导演汇报

阶段4: 终审执行
  总导演综合判断 → 终审定稿
    ↓
  AI艺术家执行 → 实际生图/生视频
```

**依赖关系图：**
```
总导演 ──→ 导演 ──→ 美术指导
  │                └──→ 摄影指导
  │
  ├──→ 品牌总监（阶段3加入）
  └──→ AI艺术家（阶段3加入 → 阶段4执行）
```

## 模式 B：广告创意模式（Brief → Creative → Brand → Production）

适用于社交媒体广告、信息流广告等。

```
阶段1: 简报理解（导演主导）
  总导演 → 导演子agent（解读创意简报）
    输出：创意方向提案

阶段2: 创意发散（并行）
  导演 → 美术指导（视觉风格草稿）
  导演 → 摄影指导（拍摄方案初稿）
    ↓
  输出：3个创意方向方案

阶段3: 品牌审核
  品牌总监 → 品牌一致性审核
    ↓
  总导演 → 选定方向

阶段4: 生产准备
  AI艺术家 → 技术可行性 + 提示词优化
    ↓
  总导演终审 → 执行方案
```

---

## 模式 C：品牌内容模式（Strategy → Visual → Copy → Compliance）

适用于品牌手册、产品详情页、品牌视频等。

```
阶段1: 策略定义（导演+品牌）
  总导演 → 导演子agent + 品牌总监
    输出：内容策略文档

阶段2: 视觉设计
  → 美术指导 → 视觉风格指南
  → 摄影指导 → 视觉参考板

阶段3: 内容制作
  → 导演子agent → 文案脚本

阶段4: 合规审核
  → 品牌总监 → 最终合规审核
  → AI艺术家 → 质量控制
    ↓
  总导演终审
```

---

## 模式 D：数字媒体模式（Concept → Visual → Technical → Publish）

适用于社交媒体、短视频、动态海报等。

```
阶段1: 概念构思
  总导演 → 导演子agent
    输出：创意概念简述

阶段2: 视觉快速原型
  导演 → 美术指导（快速视觉方案）
  导演 → 摄影指导（快速运镜方案）

阶段3: 技术实现
  AI艺术家 → 提示词生成 + 测试
  品牌总监 → 快速合规检查

阶段4: 发布执行
  总导演终审 → 批量生成
```

## 工作流选择决策树

```
项目启动
  │
  ├→ 是否有时长 > 60秒的视频？
  │   ├→ 是 → 影视制作模式（A）
  │   └→ 否 ↓
  │
  ├→ 是否需要品牌深度审核？
  │   ├→ 是 → 品牌内容模式（C）
  │   └→ 否 ↓
  │
  ├→ 是否需要快速出稿？
  │   ├→ 是 → 数字媒体模式（D）
  │   └→ 否 ↓
  │
  └→ 广告创意模式（B）
```

## 角色调用路径汇总

| 模式 | 调用顺序 | 串并行 |
|------|---------|--------|
| A: 影视制作 | 总导演→导演→(美术∥摄影)→(品牌∥AI)→总导演终审 | 先串后并 |
| B: 广告创意 | 总导演→导演→(美术∥摄影)→品牌→AI→总导演终审 | 先串后并 |
| C: 品牌内容 | 总导演→(导演+品牌)→美术→摄影→导演→品牌→AI→总导演 | 串行为主 |
| D: 数字媒体 | 总导演→导演→(美术∥摄影)→AI→品牌→总导演 | 快速通道 |

注：∥ 表示并行，+ 表示协作
