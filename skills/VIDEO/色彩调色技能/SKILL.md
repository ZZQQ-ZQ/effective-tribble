# 色彩调色技能

> 不是理论手册——是实操工具。目标：任何场景→查表→写出对应的画面基调+色彩与影调+分镜光色。

---

## 一、色彩三要素（速读·5分钟入门）

任何颜色都由三个维度定义：

| 维度 | 英文 | 是什么 | 调色含义 |
|------|------|--------|---------|
| **色相** | Hue (H) | 颜色名称：红/蓝/绿/黄/紫 | 画面的主色调——偏暖还是偏冷 |
| **饱和度** | Saturation (S) | 颜色纯度：高=鲜艳浓烈，低=灰淡 | 画面情绪强度——高饱和=冲击力/低饱和=压抑/素雅 |
| **明度** | Value / Brightness (V/L) | 颜色亮暗：高=发白，低=发黑 | 画面整体亮度——高调/低调/中间调 |

**HSV圆柱体模型：**
```
        色相 (H) ← 圆柱顶面，0-360°，红/橙/黄/绿/青/蓝/紫/品
          ↓
    饱和度 (S) ← 圆柱半径，中心=灰白→边缘=纯色
          ↓
    明度 (V) ← 圆柱高度，顶=白 → 底=黑
```

### 🔴 Seedance 不能写数值

Seedance 不支持 `HSL(30,80,60)` 这类数字参数。必须用自然语言描述：
- ❌ `色相120°` `饱和度80%` `明度50%`
- ✅ `翠绿色调` `高饱和浓郁红` `低明度深灰蓝`

## 二、色相环与色彩关系

### 伊登12色相环

```
              红
      红橙    │    紫红
   橙        │         紫
黄橙         │          蓝紫
黄 ──────────+────────── 蓝
黄绿         │          蓝绿
   绿        │         青
      绿青   │    青蓝
              青
```

### 四种色彩关系

| 关系 | 角度 | 效果 | 情绪 |
|------|------|------|------|
| **互补色** | 180°（红/青、橙/蓝、黄/紫） | 最大对比，视觉冲击 | 冲突/张力/对立 |
| **近似色** | 相邻30°内 | 和谐柔和，无冲突感 | 宁静/优雅/舒适 |
| **分裂互补** | 主色+互补色两邻色 | 有对比但不刺眼 | 丰富/有层次/不打架 |
| **三原色** | 120°三等分 | 均衡鲜明，超现实 | 活泼/多彩/童话感 |

## 三、六大电影配色方案

### 3.1 单色方案（Monochromatic）
单一色相，仅变饱和度和明度。
- **效果**：统一/聚焦/情绪沉浸
- **适用**：回忆闪回 / 梦境 / 压抑场景 / 极简风格
- **提示词**：`银灰单色调，仅有明度层次区分空间深度，`
- **代表**：《黑客帝国》绿色单色、《辛德勒名单》灰调+红色

### 3.2 互补色方案（Complementary）⭐最常用
色环对面两色搭配，一主一辅。
- **效果**：视觉冲击力最大
- **适用**：动作片/冲突场景/广告大片
- **经典组合**：
  - **橙+青（Teal & Orange）** ← 好莱坞大片标配
  - **红+青** ← 科幻/电子感
  - **黄+紫** ← 奇幻/贵族
  - **蓝+黄** ← 轻松/阳光/旅行
- **提示词**：`橙青互补色调，暖色主体在冷色背景前突出，`

### 3.3 近似色方案（Analogous）
色环相邻2-3色。
- **效果**：柔和、自然、氛围沉浸
- **适用**：抒情文戏/自然风光/情感段落
- **经典组合**：红-橙-黄 / 蓝-青-绿 / 紫-蓝紫-蓝
- **提示词**：`暖金到琥珀的渐近色调，色彩过渡柔和无冲突`

### 3.4 分裂互补方案（Split-Complementary）
主色+互补色左右两邻色。
- **效果**：有对比但不冲，色彩丰富
- **适用**：叙事段落/群像戏/对话场景
- **提示词**：`金橙主调搭配孔雀蓝与深紫辅助色，`

### 3.5 三色方案（Triadic）
色环120°三等分。
- **效果**：均衡鲜明、活泼
- **适用**：喜剧/动画/超现实/欢快场景
- **提示词**：`红黄蓝均衡三色，画面活泼明快，`

### 3.6 冷暖对比方案（Warm-Cool）⭐动作片第二常用
画面分为暖色区和冷色区。
- **效果**：空间层次、焦点引导
- **适用**：打斗/火焰场景/魔法特效
- **经典**：火焰(暖金) vs 魔法(冷蓝) / 人物(暖肤) vs 环境(冷灰)
- **提示词**：`暖金火光区域与冷蓝暗面形成色温对比，`

## 五、流派→情绪→调色决策链 🔴 第一步就用这个

### 5.1 决策流程

```
1. 确定流派 → 2. 锁定基底方向 → 3. 按情绪微调 → 4. 落笔方案
```

### 5.2 流派→基底调色（先锁这套）

| 流派 | 基底方向 | 色调 | 饱和度 | 明度 | 代表影片 |
|------|---------|------|--------|------|---------|
| 恐怖/惊悚 | 冷调+去饱和 | 深蓝灰/墨绿/青灰 | 低→极低 | 暗压 | 《招魂》《小丑回魂》 |
| 浪漫/喜剧 | 暖调+中高饱和 | 暖金/琥珀/桃色 | 中高 | 中明偏亮 | 《爱乐之城》 |
| 科幻/赛博 | 冷调+高对比 | 冷蓝/钢灰/霓虹点缀 | 选择性高饱和 | 暗+高亮点缀 | 《银翼杀手》 |
| 动作/大片 | 橙青互补 | 暖橙主体+冷青环境 | 中高 | 中调 | 《007:天幕杀机》 |
| 战争/严肃 | 漂白旁路+冷灰 | 去饱和钢灰/泥土棕 | 低 | 暗压 | 《钢锯岭》 |
| 史诗/奇幻 | 互补+拉满 | 金橙主体+冷蓝/紫环境 | 高 | 中调偏亮 | 《指环王》 |
| 文艺/情感 | 柔和近似或单色 | 暖琥珀/冷蓝紫单色 | 中→低 | 中调 | 《爱在黎明破晓前》 |
| 悬疑/黑色 | 高反差单色+冷调 | 蓝灰/银灰+红色点睛 | 中低 | 暗压 | 《七宗罪》《记忆碎片》 |
| 武侠/古装 | 依戏定 | 暖金（正气）/冷灰（阴谋）/褪色（回忆） | 中 | 中调 | 《英雄》五色叙事 |

### 5.3 情绪→微调（基底定完再调）

| 基底 | 需要表达… | 调什么 | 怎么做 |
|------|-----------|--------|--------|
| 任何基底 | 更沉重/压抑 | 饱和度↘ 明度↘ | 降饱和+压暗面+冷偏移 |
| 任何基底 | 更轻快/希望 | 饱和度↗ 明度↗ | 提亮中调+暖偏移+金色高光 |
| 任何基底 | 更紧张 | 反差↗ | 互补色强度拉大+切割光 |
| 任何基底 | 更柔和 | 反差↘ | 补色饱和度降低+柔光漫射 |
| 任何基底 | 更怀旧 | 饱和度↘ +暖偏移 | 褪色暖黄/微品偏移+胶片颗粒感 |
| 任何基底 | 更诡异 | 饱和度↘ +冷偏移 | 低饱和紫灰+单色补冷绿 |

### 5.4 完整决策示例

```
问：武戏打斗 → 流派=武侠/动作
     ↓
基底：橙青互补（动作大片标配）
     ↓
情绪微调：这次是绝境反杀 → 需要沉重感
     ↓ 饱和度↘ 明度↘
最终：低饱和冷灰蓝主调，暖金仅保留在铠甲反光边缘，暗面泛冷紫灰
```

## 六、场景→调色速查表 🔴 查完流派再来这里细化

### 6.1 打斗/战斗

| 阶段 | 方案 | 主调 | 写法 |
|------|------|------|------|
| 进场 | 互补 | 战甲暖金 vs 环境冷灰 | `暖金与冷灰互补，铠甲暖光在灰暗环境中突出` |
| 对抗 | 互补 | 橙火 vs 青蓝剑气 | `橙红火光与青蓝剑气形成强互补碰撞` |
| 冲击 | 冷暖对比 | 碰撞点暖爆 vs 边冷收 | `碰撞中心暖金能量爆发，边缘冷蓝蔓延` |
| 收束 | 低饱和 | 降饱和度→灰 | `饱和度随能量消散递减，画面趋于银灰` |

### 6.2 文戏/对话

| 类型 | 方案 | 主调 | 写法 |
|------|------|------|------|
| 温馨 | 近似 | 暖金→琥珀 | `暖金至琥珀温和渐近，色调柔和统一` |
| 紧张对峙 | 互补（弱化） | 暖肤 vs 冷环境 | `暖调肤色在冷灰环境中拉开空间，互补但不冲` |
| 回忆闪回 | 单色 | 褪色暖黄/银灰 | `褪色暖黄单色调，饱和度压低营造记忆感` |
| 悲伤/沉重 | 单色（冷） | 低饱和蓝灰 | `低饱和蓝灰单色调，画面明度整体压低` |

### 6.3 魔法/奇幻

| 类型 | 方案 | 写法 |
|------|------|------|
| 光明魔法 | 金色+白 高饱和 | `暖金色系高饱和，白光溢散降低环境饱和度` |
| 暗黑魔法 | 深紫+银青 单色 | `冷紫到银青的渐近冷色调，暗面泛紫光` |
| 元素魔法 | 按属性配色 | 火=橙红 / 冰=青白 / 雷=蓝紫 / 风=淡青 |
| 召唤/降临 | 分裂互补 | `中心金白高亮，周围紫蓝暗压，分裂互补层次` |

### 6.4 自然/外景

| 场景 | 色温建议 | 写法 |
|------|---------|------|
| 正午旷野 | 暖金+青空互补 | `暖金阳光与净青天空互补，高饱和硬光` |
| 日落黄昏 | 暖橙单色 | `暖橙色系渐近，从琥珀到深红层层递进` |
| 月下夜景 | 蓝灰单色 | `银蓝灰单色调，月色冷白为唯一高亮` |
| 竹林/森林 | 近似绿系 | `翠绿到墨绿的近似层次，透过叶缝的暖黄光斑` |
| 雨天/阴天 | 低饱和灰调 | `冷灰低饱和，画面褪色感，湿气弥漫` |

### 6.5 情绪氛围一键式

| 想要的感觉 | 方案 | 速写 |
|-----------|------|------|
| 压迫感 | 低明+低饱和+冷单色 | `低明度深蓝灰单色调，空间压缩感` |
| 史诗感 | 互补+高饱和 | `橙暖主体与青冷环境强互补，饱和度拉满` |

## 七、提示词中的色彩写法

### 6.1 画面基调行

一句话定全段色彩底色。写法格式：

```
[主调]+[色彩关系]+[明度说明]+[色彩氛围]

示例：
"冷灰蓝辅以金橙互补，高反差，压迫下的对峙感"
"暖金琥珀渐近单色，中低明度，深沉而不压抑"
"冰蓝与烈火橙强互补，高饱和，能量碰撞的瞬间"
"褪色银灰低饱和单色调，记忆中模糊的往事"
```

### 6.2 色彩与影调行

在现有格式内嵌入色彩方案。写法：

```
色彩：[主调]+[补色]+[饱和度]+[明度对比]

完整行示例：
"主调暖金色，补色冷灰，高饱和低明度反差，硬光锐利切割。大殿内弥漫战前的压迫感。"
"主调青蓝单色，饱和度逐渐从高向低递减，明度从中向暗过渡。能量释放后画面趋于沉寂。"
```

### 6.3 分镜中嵌入光色

不要在分镜里单独写色彩行——把颜色嵌进光效描述中：

```
✅ "暖金侧逆光打进画面，铠甲边缘泛起琥珀色反光"
✅ "他右手凝聚的青蓝光球将周围染上冷色调，"
✅ "火焰从橙红渐变为深红，周围空气因高温扭曲"

❌ "[色彩：暖金+冷蓝互补]"
❌ "此时画面切换为冷色调"
```

## 八、Seedance 色彩输入铁律

### ✅ 能写的方式

| 方式 | 示例 |
|------|------|
| 色名+色温 | `暖金色` `冷灰蓝` `琥珀色` `孔雀蓝` `深绯红` |
| 色彩关系 | `橙青互补` `暖金渐近` `红蓝强对比` |
| 饱和度层次 | `高饱和浓郁` `低饱和褪色` `饱和度从高向低递减` |
| 明度变化 | `明度压低` `高亮突出` `从亮向暗过渡` |
| 色彩氛围 | `浓烈` `柔和` `压抑` `通透` `雾感` |

### ❌ 不能写的方式

| 禁止 | 原因 |
|------|------|
| `HSL(30,80,60)` `#FF6600` | 模型不认数值 |
| `色相偏移30°` | 模型不理解角度 |
| `RGB(255,128,0)` | 同上 |
| `饱和度+50%` `明度-20%` | 无法映射到视觉 |
| "然后画面变成蓝色" | 太生硬，写「冷蓝光漫入画面」 |

## 十、常见色彩错误

| 错误 | 后果 | 纠正 |
|------|------|------|
| 画面基调写"冷暖结合"太笼统 | 模型随机选 | 明确：主调暖金+补色冷灰 |
| 每镜换一套调色方案 | 色彩跳跃 | 全段锁定一套方案，镜间微调 |
| 色彩与影调行和画面基调行矛盾 | 模型混乱 | 两行色彩方向一致，不打架 |
| "电影级调色"不写具体方案 | 无效指令 | 方案→色系→主调补色→饱和度 |
| 饱和度写"正常" | 无意义 | 高饱和/低饱和/中饱和 |
