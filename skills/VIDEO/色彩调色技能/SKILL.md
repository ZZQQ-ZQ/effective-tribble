# 色彩调色技能

> 不是理论手册——是实操工具。目标：任何场景→查表→写出对应的画面基调+色彩与影调+分镜光色。

---

## 一、色彩三要素（速读·5分钟入门）

任何颜色都由三个维度定义：

| 维度 | 英文 | 是什么 | 调色含义 |
|------|------|--------|---------|
| **色相** | Hue (H) | 颜色名称：红/蓝/绿/黄/紫 | 画面的主色调——偏暖还是偏冷 |
| **饱和度** | Saturation (S) | 颜色纯度：高=鲜艳浓烈，低=灰淡 | 画面情绪强度——高饱和=冲击力/低饱和=压抑/素雅 |
| **明度** | Value / Brightness (V/L) | 颜色亮暗：高=发白，低=发黑 | 画面整体亮度——高调/低调/中间调 |

**HSV圆柱体模型：**
```
        色相 (H) ← 圆柱顶面，0-360°，红/橙/黄/绿/青/蓝/紫/品
          ↓
    饱和度 (S) ← 圆柱半径，中心=灰白→边缘=纯色
          ↓
    明度 (V) ← 圆柱高度，顶=白 → 底=黑
```

### 🔴 Seedance 不能写数值

Seedance 不支持 `HSL(30,80,60)` 这类数字参数。必须用自然语言描述：
- ❌ `色相120°` `饱和度80%` `明度50%`
- ✅ `翠绿色调` `高饱和浓郁红` `低明度深灰蓝`

## 二、色相环与色彩关系

### 伊登12色相环

```
              红
      红橙    │    紫红
   橙        │         紫
黄橙         │          蓝紫
黄 ──────────+────────── 蓝
黄绿         │          蓝绿
   绿        │         青
      绿青   │    青蓝
              青
```

### 四种色彩关系

| 关系 | 角度 | 效果 | 情绪 |
|------|------|------|------|
| **互补色** | 180°（红/青、橙/蓝、黄/紫） | 最大对比，视觉冲击 | 冲突/张力/对立 |
| **近似色** | 相邻30°内 | 和谐柔和，无冲突感 | 宁静/优雅/舒适 |
| **分裂互补** | 主色+互补色两邻色 | 有对比但不刺眼 | 丰富/有层次/不打架 |
| **三原色** | 120°三等分 | 均衡鲜明，超现实 | 活泼/多彩/童话感 |

## 三、六大电影配色方案

### 3.1 单色方案（Monochromatic）
单一色相，仅变饱和度和明度。
- **效果**：统一/聚焦/情绪沉浸
- **适用**：回忆闪回 / 梦境 / 压抑场景 / 极简风格
- **提示词**：`银灰单色调，仅有明度层次区分空间深度，`
- **代表**：《黑客帝国》绿色单色、《辛德勒名单》灰调+红色

### 3.2 互补色方案（Complementary）⭐最常用
色环对面两色搭配，一主一辅。
- **效果**：视觉冲击力最大
- **适用**：动作片/冲突场景/广告大片
- **经典组合**：
  - **橙+青（Teal & Orange）** ← 好莱坞大片标配
  - **红+青** ← 科幻/电子感
  - **黄+紫** ← 奇幻/贵族
  - **蓝+黄** ← 轻松/阳光/旅行
- **提示词**：`橙青互补色调，暖色主体在冷色背景前突出，`

### 3.3 近似色方案（Analogous）
色环相邻2-3色。
- **效果**：柔和、自然、氛围沉浸
- **适用**：抒情文戏/自然风光/情感段落
- **经典组合**：红-橙-黄 / 蓝-青-绿 / 紫-蓝紫-蓝
- **提示词**：`暖金到琥珀的渐近色调，色彩过渡柔和无冲突`

### 3.4 分裂互补方案（Split-Complementary）
主色+互补色左右两邻色。
- **效果**：有对比但不冲，色彩丰富
- **适用**：叙事段落/群像戏/对话场景
- **提示词**：`金橙主调搭配孔雀蓝与深紫辅助色，`

### 3.5 三色方案（Triadic）
色环120°三等分。
- **效果**：均衡鲜明、活泼
- **适用**：喜剧/动画/超现实/欢快场景
- **提示词**：`红黄蓝均衡三色，画面活泼明快，`

### 3.6 冷暖对比方案（Warm-Cool）⭐动作片第二常用
画面分为暖色区和冷色区。
- **效果**：空间层次、焦点引导
- **适用**：打斗/火焰场景/魔法特效
- **经典**：火焰(暖金) vs 魔法(冷蓝) / 人物(暖肤) vs 环境(冷灰)
- **提示词**：`暖金火光区域与冷蓝暗面形成色温对比，`

## 五、调色操作顺序 —— 5步管线

> 来源：Noam Kroll（好莱坞调色师，调过数千小时影视+数字媒体）
> 核心原则：**技术修正在前，创意在后。反过来做=浪费时间。**

### 步骤1：中和与匹配（Neutralize & Match）

**做什么**：把所有镜头的白平衡/曝光/对比拉到同一个中性基线。

- 不做风格化，只做"归零"
- 选一个"英雄镜头"（hero shot）作为参考→其余镜头全部匹配到它
- 用示波器（scopes）辅助——饱和/偏色/对比在示波器上量化可见
- 目标：所有素材站在同一起跑线

### 步骤2：曝光修正（Exposure Fixes）

**做什么**：修复步骤1无法全局解决的小区域曝光问题。

- 例子：窗外过曝 / 暗色衣服死黑 / 人脸欠曝
- 工具：Power Window / 遮罩局部调整
- ⚠️ 不碰整体曝光（那在第1步做完了）

### 步骤3：色彩修正（Color Fixes）

**做什么**：修复个别颜色问题——特别是肤色。

- 红色T恤过饱和→选色降饱和度
- 肤色偏红/偏粉→微调中间调降温、补色偏移
- 肤色偏绿（荧光灯）→推紫色到中间调
- 工具：Qualifier / 选色限定器 + 局部遮罩

### 步骤4：上风格（The Look）

**做什么**：从干净的空白画布开始创意工作。

- 冷暖偏移 / 对比曲线 / 色调分离 / 饱和度曲线
- 前提：前3步没做好→风格层会**放大所有瑕疵**
- 例：如果肤色本来偏红，"青橙大片风"会让脸红得像烤虾

### 步骤5：最终微调 + LUT（Final Tweaks）

**做什么**：全局统一感。

- 轻微阴影上提 / 高光滚降 / 全局对比度微调
- 叠加创意LUT做整体"统一外套"
- 这个节点的调整可以跨场景复用——同一部戏所有场景套同一套微调

### 🔴 Seedance 实际应用

不用真调色软件，但这个顺序直接影响写提示词的逻辑：

```
Step 1-3 → 写之前先"归零"大脑：这段戏的"中性起点"是什么？
            （室内/室外？日光/夜光？先定环境白平衡）

Step 4   → 再叠风格：冷调还是暖调？互补还是单色？
            在色彩与影调行落地

Step 5   → 段尾追加：画面整体的统一感声明
            （如"全段保持青蓝冷调统一"）
```

## 六、调色决策的四维心理框架

> 来源：Noam Kroll "The Psychology of Color Grading"
> 每一维都是一个独立的情绪控制杆。同时操纵四杆=精确控制观众的心理反应。

### 6.1 维度一：暖 vs 冷（Color Temperature）

**这是对观众情绪最直接、最即时的冲击。**

| 方向 | 自然联想 | 情绪 | 电影场景 |
|------|---------|------|---------|
| **暖（橙金）** | 阳光海滩、热水、放松的午后 | 舒适/亲切/怀旧/邀请 | 浪漫喜剧、年代片、温馨日常 |
| **冷（蓝灰）** | 暴风雪、冰路、脊背发凉 | 疏离/冷酷/冷静/不安 | 惊悚、科幻、动作、夜景 |

**但可以反向使用**：
- 温暖调色制造虚假安全感→剧情反转时冲击更大
- 冷色包装温暖的剧情→制造疏离与克制的情绪张力

**年代感知**：
- 极端暖+低饱和（sepia）= 老照片/旧时代 → 脑海中自动标记为"过去"
- 极端冷+蓝紫 = 未来/科幻 → 大多数科幻片用冷蓝紫色系

**原则**：色彩温度最好**微妙使用**。不是往观众头上猛砸——是让观众潜意识感受到。

### 6.2 维度二：中性平衡 vs 风格化（Balance vs Stylization）

**中性=真相。风格化=进入另一个世界。**

| 程度 | 画面特征 | 观众反应 | 代表作品 |
|------|---------|---------|---------|
| **绝对中性** | 白=白、黑=黑，无偏色 | "这是客观事实" | 纪录片、现实主义 | 
| **轻度风格** | 轻微暖偏移或冷偏移 | "有态度，但可信" | 剧情片 |
| **明显风格** | 明显色彩方向（深蓝/暗红/绿） | "准备进入异世界" | 恐怖片、科幻、奇幻 |

**核心洞察**：画面越风格化→观众越愿意**暂停怀疑**（suspend disbelief）。
- 跟看动画片同理——你不会质疑一只会说话的狗，因为画面本身就告诉你"这不是现实"
- 同理，暗红色浓的镜头告诉你"欢迎进入恐怖世界"

### 6.3 维度三：高饱和 vs 去饱和（Saturation）

**饱和=扩张世界。去饱和=聚焦人物。**

| 程度 | 效果 | 观众注意点 | 典型案例 |
|------|------|-----------|---------|
| **高饱和** | 扩张画面、刺激视觉 | 被颜色/环境吸引 | Mad Max、Spring Breakers |
| **中去饱和** | 克制但不失色彩 | 平衡注意 | 多数剧情片 |
| **低去饱和（接近黑白）** | 聚焦、压缩视野 | 被构图/对比/人物吸引 | Saving Private Ryan |
| **完全黑白** | 纯聚焦 | 被迫关注对比/构图/表演 | Schindler's List |

**对比效果**：
- 高清大图→你的眼睛被蓝色天空和白云吸引
- 同一照片去色→你的眼睛直接跳到人物主体

**Noam Kroll 原话**：就像人失去一种感官后其他感官增强——去掉色彩信息，观众被迫用另一种方式激活大脑。

### 6.4 维度四：高对比 vs 低对比（Contrast）

**对比度=张力的直接计量器。**

| 程度 | 画面特征 | 情绪 | 场景 |
|------|---------|------|------|
| **高对比** | 深黑+锐白，层次斩钉截铁 | 张力/紧张/强度 | 动作高潮/冲突/悬疑 |
| **中对比** | 标准明暗层次 | 叙事/日常 | 多数场景 |
| **低对比** | 上提阴影（milky blacks）、滚降低光（rolled-off highlights） | 梦幻/柔和/非现实 | 梦境/回忆/浪漫 |

### 6.5 四维联动速查

| 想要的情绪 | 色温 | 风格程度 | 饱和 | 对比 |
|-----------|------|---------|------|------|
| 安全感/舒适 | 暖 | 低→中性 | 中高 | 中低 |
| 压迫感/恐惧 | 冷 | 中→高 | 低 | 高 |
| 英雄史诗 | 暖+冷互补 | 中 | 高 | 中高 |
| 孤独空虚 | 冷 | 低→中 | 低→极低 | 中 |
| 梦境/回忆 | 可暖可冷 | 中高 | 低 | 低 |
| 绝望/末世 | 冷 | 高 | 极低 | 高 |

## 七、经典电影调色案例分析

### 7.1 Mad Max: Fury Road —— 反向定位+逐场规划

**导演brief（一句话）**：`"应该是饱和的、画面像漫画，夜景是蓝色的"`

**调色师**：Eric Whipp

**决策链还原**：

1. **市场反定位**：30年来的末世片全部用漂白+去饱和→这次走饱和+浓烈+漫画感
2. **沙漠限制**：沙漠本身调色板极其有限（沙子+天空）→必须主动创造视觉变化
3. **粗剪出来后**：逐场看→规划不同look→打破单调
4. **"眼动扫描"哲学**（George Miller提出）：观众不应寻找焦点→用暗角/局部压暗引导视线，每帧一看就知道看哪
5. **锐度=砂砾感**：每镜独立锐化→防止饱和画面变塑料感→保持"粗粝"质感
6. **"救赎"场景**：全部天空替换成暴风雨天→白光打在人物身上+阴沉背景=不确定的紧张情绪（"这计划到底能不能成功？"）
7. **夜景95%纯蓝**：留5%原始色给血/绿植等叙事关键色
8. **眼珠ROTO**：每颗眼珠单独抠像→加对比+锐化→人类大脑80%注意力在角色眼睛上

**可复用的决策逻辑**：
- 知道同类型片做了什么→反着来
- 场景限制越少→越需要主动设计变化
- 每个场景的色彩应该推动叙事问题（"计划会成功吗？"→暴风雨=不确定）

### 7.2 O Brother, Where Art Thou? —— 导演一句话到精准执行

**导演想要的**：`"旧明信片的褪色感，特别要干掉所有绿色"`

**调色师**：Roger Deakins（摄影）+ Cinesite数字DI团队

**关键背景**：这是**史上首部100%数字DI完成的全长电影**。之前只有《Pleasantville》部分使用了数字DI。

**决策链还原**：

1. 实拍地密西西比"比爱尔兰还绿" → 拍摄素材里全是绿
2. 传统光化学试了6种工艺→全部失败→根本达不到"杀掉绿色但不变成纯sepia"的效果
3. 决定走数字DI→第一镜就花了几天→全片11周
4. **核心操作**：数字选定绿色通道→全局压掉绿色饱和度→保留暖黄/棕/金色系的微妙层次
5. **不是纯sepia**：是sepia的变体——旧明信片的褪色感≠单色sepia层
6. 结果：每片叶子看起来都被太阳烤枯→空气中弥漫尘土感

**可复用的决策逻辑**：
- "不是X"的定义比"是Y"更重要（"不是纯sepia"→才是他们要的）
- 杀一个颜色≠画面变单调→暖色系内部的微层次一样丰富

### 7.3 Saving Private Ryan —— 技法名服务于叙事意图

**工艺**：漂白旁路负片 + ENR显影100%IR → 独有的色彩空间

**导演想要的**：`"旧新闻片战争素材的质感——褪色但不浑浊"`

**决策链还原**：

1. 漂白旁路：跳过漂白步骤→银粒子保留在胶片上→黑更深/去饱和/对比更高
2. ENR工艺：进一步控制对比度和去饱和程度
3. **每场戏有独立look**——不是全片一个调
4. 结果：褪色冷灰 + 深红血液的强烈对比——血液在灰暗背景下格外触目

**可复用的决策逻辑**：
- "褪色但不浑浊"= 饱和度低 + 对比度高（饱和度降但层次不糊）
- 战争中红色是用来打痛观众的——在灰暗基调里红是唯一的触发器

### 7.4 Blade Runner 2049 —— 颜色作为信息编码

**决策逻辑**（Filmsupply分析）：

- **黄色**贯穿全片 = 启迪/信息/觉悟 → 每当K接近真相，画面被黄光淹没
- Wallace总部整场戏黄色→潜意识："这里储藏着关键信息，这个空间很重要"
- **橙色/琥珀** = 真实世界的尘埃感 → 洛杉矶/拉斯维加斯
- **冷蓝灰** = 系统/压迫/隐藏 → K的公寓、警察局

**可复用的决策逻辑**：
- 一种颜色可以绑定一个叙事概念→全片反复出现→建立潜意识编码
- 颜色不只是"好看"——是可重复调用的叙事语言

### 7.5 Harry Potter 系列 —— 跨8部的色彩弧线

**8部电影的色彩演进**：

| 阶段 | 电影 | 色彩特征 | 叙事含义 |
|------|------|---------|---------|
| 魔法初现 | 1-2部 | 暖金+高饱和+发光 | 魔法世界安全温暖，像童话 |
| 阴霾降临 | 第3部（阿兹卡班） | 第一个冷调转折点 | 蓝灰注入，暗黑开端 |
| 深度黑暗 | 4-6部 | 渐进去饱和+蓝灰绿主导 | 温暖从这个世界被抽走 |
| 终局之战 | 第7部（上下） | 高度去饱和+冷灰 | 童年结束，战争废墟 |

**可复用的决策逻辑**：
- 8部电影就是一次超大规模的"色彩弧线"
- 温暖→逐步抽离→冷灰——色彩本身在讲一个"失去纯真"的故事
- 在15秒短剧里同样适用：段1暖金→段5冷灰 = 情绪下行曲线

## 八、流派→情绪→调色决策链 🔴 第一步就用这个

### 8.1 决策流程

```
1. 确定流派 → 2. 锁定基底方向 → 3. 按情绪微调 → 4. 落笔方案
```

### 8.2 流派→基底调色（先锁这套）

| 流派 | 基底方向 | 色调 | 饱和度 | 明度 | 代表影片 |
|------|---------|------|--------|------|---------|
| 恐怖/惊悚 | 冷调+去饱和 | 深蓝灰/墨绿/青灰 | 低→极低 | 暗压 | 《招魂》《小丑回魂》 |
| 浪漫/喜剧 | 暖调+中高饱和 | 暖金/琥珀/桃色 | 中高 | 中明偏亮 | 《爱乐之城》 |
| 科幻/赛博 | 冷调+高对比 | 冷蓝/钢灰/霓虹点缀 | 选择性高饱和 | 暗+高亮点缀 | 《银翼杀手》 |
| 动作/大片 | 橙青互补 | 暖橙主体+冷青环境 | 中高 | 中调 | 《007:天幕杀机》 |
| 战争/严肃 | 漂白旁路+冷灰 | 去饱和钢灰/泥土棕 | 低 | 暗压 | 《钢锯岭》 |
| 史诗/奇幻 | 互补+拉满 | 金橙主体+冷蓝/紫环境 | 高 | 中调偏亮 | 《指环王》 |
| 文艺/情感 | 柔和近似或单色 | 暖琥珀/冷蓝紫单色 | 中→低 | 中调 | 《爱在黎明破晓前》 |
| 悬疑/黑色 | 高反差单色+冷调 | 蓝灰/银灰+红色点睛 | 中低 | 暗压 | 《七宗罪》《记忆碎片》 |
| 武侠/古装 | 依戏定 | 暖金（正气）/冷灰（阴谋）/褪色（回忆） | 中 | 中调 | 《英雄》五色叙事 |

### 8.3 情绪→微调（基底定完再调）

| 基底 | 需要表达… | 调什么 | 怎么做 |
|------|-----------|--------|--------|
| 任何基底 | 更沉重/压抑 | 饱和度↘ 明度↘ | 降饱和+压暗面+冷偏移 |
| 任何基底 | 更轻快/希望 | 饱和度↗ 明度↗ | 提亮中调+暖偏移+金色高光 |
| 任何基底 | 更紧张 | 反差↗ | 互补色强度拉大+切割光 |
| 任何基底 | 更柔和 | 反差↘ | 补色饱和度降低+柔光漫射 |
| 任何基底 | 更怀旧 | 饱和度↘ +暖偏移 | 褪色暖黄/微品偏移+胶片颗粒感 |
| 任何基底 | 更诡异 | 饱和度↘ +冷偏移 | 低饱和紫灰+单色补冷绿 |

### 8.4 完整决策示例

```
问：武戏打斗 → 流派=武侠/动作
     ↓
基底：橙青互补（动作大片标配）
     ↓
情绪微调：这次是绝境反杀 → 需要沉重感
     ↓ 饱和度↘ 明度↘
最终：低饱和冷灰蓝主调，暖金仅保留在铠甲反光边缘，暗面泛冷紫灰
```

## 九、场景→调色速查表 🔴 查完流派再来这里细化

### 9.1 打斗/战斗

| 阶段 | 方案 | 主调 | 写法 |
|------|------|------|------|
| 进场 | 互补 | 战甲暖金 vs 环境冷灰 | `暖金与冷灰互补，铠甲暖光在灰暗环境中突出` |
| 对抗 | 互补 | 橙火 vs 青蓝剑气 | `橙红火光与青蓝剑气形成强互补碰撞` |
| 冲击 | 冷暖对比 | 碰撞点暖爆 vs 边冷收 | `碰撞中心暖金能量爆发，边缘冷蓝蔓延` |
| 收束 | 低饱和 | 降饱和度→灰 | `饱和度随能量消散递减，画面趋于银灰` |

### 9.2 文戏/对话

| 类型 | 方案 | 主调 | 写法 |
|------|------|------|------|
| 温馨 | 近似 | 暖金→琥珀 | `暖金至琥珀温和渐近，色调柔和统一` |
| 紧张对峙 | 互补（弱化） | 暖肤 vs 冷环境 | `暖调肤色在冷灰环境中拉开空间，互补但不冲` |
| 回忆闪回 | 单色 | 褪色暖黄/银灰 | `褪色暖黄单色调，饱和度压低营造记忆感` |
| 悲伤/沉重 | 单色（冷） | 低饱和蓝灰 | `低饱和蓝灰单色调，画面明度整体压低` |

### 9.3 魔法/奇幻

| 类型 | 方案 | 写法 |
|------|------|------|
| 光明魔法 | 金色+白 高饱和 | `暖金色系高饱和，白光溢散降低环境饱和度` |
| 暗黑魔法 | 深紫+银青 单色 | `冷紫到银青的渐近冷色调，暗面泛紫光` |
| 元素魔法 | 按属性配色 | 火=橙红 / 冰=青白 / 雷=蓝紫 / 风=淡青 |
| 召唤/降临 | 分裂互补 | `中心金白高亮，周围紫蓝暗压，分裂互补层次` |

### 9.4 自然/外景

| 场景 | 色温建议 | 写法 |
|------|---------|------|
| 正午旷野 | 暖金+青空互补 | `暖金阳光与净青天空互补，高饱和硬光` |
| 日落黄昏 | 暖橙单色 | `暖橙色系渐近，从琥珀到深红层层递进` |
| 月下夜景 | 蓝灰单色 | `银蓝灰单色调，月色冷白为唯一高亮` |
| 竹林/森林 | 近似绿系 | `翠绿到墨绿的近似层次，透过叶缝的暖黄光斑` |
| 雨天/阴天 | 低饱和灰调 | `冷灰低饱和，画面褪色感，湿气弥漫` |

### 9.5 情绪氛围一键式

| 想要的感觉 | 方案 | 速写 |
|-----------|------|------|
| 压迫感 | 低明+低饱和+冷单色 | `低明度深蓝灰单色调，空间压缩感` |
| 史诗感 | 互补+高饱和 | `橙暖主体与青冷环境强互补，饱和度拉满` |
| 孤独感 | 单色+低饱和+中高明 | `蓝灰单色调，人物在大面积冷色空场中` |
| 希望感 | 暖色高亮+金 | `暖金高亮在灰暗中点亮，明度对比引导视线` |
| 紧张感 | 互补（高反差） | `红蓝强互补，高反差明暗切割` |

## 十、提示词中的色彩写法

### 10.1 画面基调行

一句话定全段色彩底色。写法格式：

```
[主调]+[色彩关系]+[明度说明]+[色彩氛围]

示例：
"冷灰蓝辅以金橙互补，高反差，压迫下的对峙感"
"暖金琥珀渐近单色，中低明度，深沉而不压抑"
"冰蓝与烈火橙强互补，高饱和，能量碰撞的瞬间"
"褪色银灰低饱和单色调，记忆中模糊的往事"
```

### 10.2 色彩与影调行

在现有格式内嵌入色彩方案。写法：

```
色彩：[主调]+[补色]+[饱和度]+[明度对比]

完整行示例：
"主调暖金色，补色冷灰，高饱和低明度反差，硬光锐利切割。大殿内弥漫战前的压迫感。"
"主调青蓝单色，饱和度逐渐从高向低递减，明度从中向暗过渡。能量释放后画面趋于沉寂。"
```

### 10.3 分镜中嵌入光色

不要在分镜里单独写色彩行——把颜色嵌进光效描述中：

```
✅ "暖金侧逆光打进画面，铠甲边缘泛起琥珀色反光"
✅ "他右手凝聚的青蓝光球将周围染上冷色调，"
✅ "火焰从橙红渐变为深红，周围空气因高温扭曲"

❌ "[色彩：暖金+冷蓝互补]"
❌ "此时画面切换为冷色调"
```

## 十一、Seedance 色彩输入铁律

### ✅ 能写的方式

| 方式 | 示例 |
|------|------|
| 色名+色温 | `暖金色` `冷灰蓝` `琥珀色` `孔雀蓝` `深绯红` |
| 色彩关系 | `橙青互补` `暖金渐近` `红蓝强对比` |
| 饱和度层次 | `高饱和浓郁` `低饱和褪色` `饱和度从高向低递减` |
| 明度变化 | `明度压低` `高亮突出` `从亮向暗过渡` |
| 色彩氛围 | `浓烈` `柔和` `压抑` `通透` `雾感` |

### ❌ 不能写的方式

| 禁止 | 原因 |
|------|------|
| `HSL(30,80,60)` `#FF6600` | 模型不认数值 |
| `色相偏移30°` | 模型不理解角度 |
| `RGB(255,128,0)` | 同上 |
| `饱和度+50%` `明度-20%` | 无法映射到视觉 |
| "然后画面变成蓝色" | 太生硬，写「冷蓝光漫入画面」 |

## 十三、常见色彩错误

| 错误 | 后果 | 纠正 |
|------|------|------|
| 画面基调写"冷暖结合"太笼统 | 模型随机选 | 明确：主调暖金+补色冷灰 |
| 每镜换一套调色方案 | 色彩跳跃 | 全段锁定一套方案，镜间微调 |
| 色彩与影调行和画面基调行矛盾 | 模型混乱 | 两行色彩方向一致，不打架 |
| "电影级调色"不写具体方案 | 无效指令 | 方案→色系→主调补色→饱和度 |
| 饱和度写"正常" | 无意义 | 高饱和/低饱和/中饱和 |
