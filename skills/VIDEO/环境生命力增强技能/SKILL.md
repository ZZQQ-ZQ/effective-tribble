# 环境生命力增强技能

> **创建**：2026-07-24 | 全网调研后整合（HunyuanVideo官方手册·seedance-2ai.org实战·OpenAI社区·Medium自然视频指南）
> **核心理念**：AI视频最大破绽是画面像静态渲染图。真实世界没有静止——环境元素一直在动。这个技能解决"画面里有东西在动"的问题。

---

## 一、核心理念

### 什么是环境生命力

**不是特效，不是粒子系统——是画面里本来就应该有的东西在动。**

| ❌ 死画面 | ✅ 活画面 |
|----------|---------|
| 只写主体动作，背景完全静止 | 主体+风+云+尘+水+飞鸟同时有运动 |
| 画面像一张渲染图在播放 | 画面像一扇窗户——外面世界本来是活的 |
| 环境是主体的背景板 | 环境是主体的共存者 |

### 导演定义（2026-07-24）

> "真实性不是PBR渲染层面——是环境生命力。画面里东西在动（风吹云飘尘雾浮动飞鸟），不是静态渲染图。"

### 全网验证

| 来源 | 核心理念 | 关键发现 |
|------|---------|---------|
| **HunyuanVideo 1.5 官方手册** | 环境元素必须写运动动词 | `blades of grass tremble` `clouds drift slowly past` `dust kicked up by paws` |
| **seedance-2ai.org V3迭代** | V3级别必须写"正在发生"的环境事件 | `wisps of cloud catch on rocky spires and trail into the wind` |
| **OpenAI 社区 Prompt Layers** | 环境写因果链非孤立标签 | `The wind creates ripples in the puddle and rustles the leaves` |
| **Medium 自然视频指南** | AI motion是照片故事的延伸 | `rippling water, shifting clouds, soft sunlight streaming` |

## 三、因果链写法（环境运动 = 事件，不是标签）

### 核心原则：不写"有风"——写"风做了什么事"

```
❌ 有风，有云                    → 标签堆砌
❌ 环境动态：风+云+尘             → 命名列表
✅ 阵风掠过，地面尘土被卷成旋涡，远处碎云快速飘移在地面投下移动阴影 
  → 这是一个正在进行的环境事件
```

### 因果链模板

```
[环境力] → [第一反应物] + [第一反应动词] → [第二反应物] + [第二反应动词] → [视觉效果]
```

**实例：**
- `风` → `草` + `起伏如波浪` → `树叶` + `翻白沙沙响` → `地面光影忽明忽暗`
- `热浪` → `空气` + `扭曲折射` → `远景` + `抖动变形` → `画面底部热浊波动`
- `雨` → `地面` + `激起水花` → `积水` + `涟漪扩散` → `倒影破碎重组`

## 五、环境运动与情绪匹配

| 情绪 | 环境运动特征 | 示例写法 |
|------|------------|---------|
| 悲伤/失落 | 静止/微风/薄雾 | `空气几乎静止，只有极细微的尘埃在光束中缓慢悬浮` |
| 紧张/压迫 | 狂风/碎云掠过/扬尘 | `狂风呼啸，地面沙尘被卷成旋涡，碎云快速掠过投下移动阴影` |
| 温馨/浪漫 | 斑驳光+树叶微动+微风 | `阳光透过树叶洒下斑驳光斑在地面缓慢移动，微风拂过花瓣轻轻颤动` |
| 荒凉/孤独 | 风沙/远雾/孤鸟 | `远处一只孤鸟掠过灰白天际，地面薄雾缓慢流动没过脚踝` |
| 愤怒/爆发 | 狂风+碎片飞散+雷暴云 | `狂风猛烈撕扯衣袍，碎石和火星在空气中横向飞散` |
| 希望/觉醒 | 云层散开+光线穿透+飞鸟 | `厚重云层缓慢散开，金色光线穿透云隙，远处飞鸟展翅飞向光的方向` |
| 恐惧/寒意 | 寒气白雾+冰晶+树叶静止 | `寒冷白雾从地面升腾，冰晶在静止的空气中悬浮，没有一丝风` |

## 六、使用原则（铁律）

### 6.1 每镜最多2个环境运动元素

塞多了模型处理不过来。远景用云/雾/群鸟/丁达尔，近景用浮尘/发丝飘动/衣摆/镜头前粒子。

### 6.2 写在分镜"画面内动"里，不单独开字段

环境运动是画面内容的一部分，不是独立的"环境动态"字段。

### 6.3 环境运动必须有因果链

不是"有风"→是"风做了什么→什么东西动了→产生了什么视觉变化"

### 6.4 必须写运动动词

| ❌ 死词 | ✅ 运动动词 |
|--------|-----------|
| 有云 | 云层缓慢流动/碎云掠过/积云翻滚 |
| 有尘 | 微尘在光柱中飘浮/地面尘土被卷起 |
| 有鸟 | 飞鸟从画右掠入/鸟群盘旋/翅膀缓慢扇动 |
| 有风 | 微风吹动发丝/阵风卷起落叶/狂风撕扯衣袍 |
| 有水 | 水面微微波动/涟漪扩散/浪花拍击 |

### 6.5 景别适配

| 景别 | 可写的环境元素 | 不能写的 |
|------|-------------|---------|
| 大远景 | 云层/群鸟/远雾/远山/海面 | 浮尘/发丝/微表情 |
| 全景 | 风+衣摆/地面扬尘/落叶/水面 | 细微粒子 |
| 中景 | 浮尘光束/衣摆摆动/发丝飘动 | 微尘近景 |
| 近景 | 镜头前飘过粒子/发丝/衣领微动 | 云层/飞鸟 |
| 特写 | 浮尘/水珠/皮肤上的风痕 | 大面积环境运动 |

## 🔴 铁律

1. **环境不是死的** — 每镜至少1个环境运动元素
2. **写动词不写名词** — 不写"有风"写"风吹动了什么"
3. **因果链必写** — 环境力→反应物→视觉变化
4. **每镜最多2个** — 塞多了模型崩
5. **情绪匹配强度** — 悲伤不给狂风，愤怒不给微风
6. **景别适配元素** — 全景不写浮尘近景，近景不写云层
