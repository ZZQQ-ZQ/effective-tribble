# 相机四维编码系统（Z/Y/X/F Camera Codec）

> 来源：MapleShaw/seedance2.0-prompt-skill `references/camera-codec.md`
> 集成日期：2026-07-13
> 定位：用坐标系思维替代关键词记忆，精准控制任何镜头的位置、运动和叙事身份

## 核心理念

**从「关键词记忆」升级为「坐标思维」**——先在脑中建立坐标系，再翻译成提示词自然语言。

```
一个完整镜头 = [Z距离] + [Y高度] + [X方位] + [F滤镜] + [运动] + [节奏] + [约束]
```

| 层级 | 控制什么 | 对应维度 |
|------|---------|---------|
| 位置层 | 相机放在哪里 | Z（远近）+ Y（高低）+ X（环绕） |
| 能力层 | 镜头物理特性 | F（焦段、景深、畸变、叙事身份） |
| 运动层 | 相机怎么动 | Z/Y/X 坐标随时间的动态变化 |

---

## Z轴 —— 距离与信息量

| 编码 | 景别 | 核心传递 | 典型用途 |
|------|------|---------|---------|
| Z1 | 大特写（ECU） | 极致感官 | 瞳孔、伤口、悬疑线索 |
| Z2 | 特写（BCU） | 局部特征 | 眼睛、嘴、手、物品细节 |
| Z3 | 近景（CU） | 表情 | 对话反应、情感高潮 |
| Z4 | 中近景（MCU） | **最通用** | 对话、Vlog、胸口以上 |
| Z5 | 中景（MS） | 肢体动作 | 拿东西、手势、互动 |
| Z6 | 四分三身（MLS） | 姿态 | 牛仔镜头、站姿亮相 |
| Z7 | 全景（FS） | 动作展示 | 穿搭、舞蹈、全身亮相 |
| Z8 | 远景（LS） | 环境关系 | 建立场景、交代地理 |
| Z9 | 大远景（ELS） | 世界观 | 开场史诗、宏大叙事 |

**速查口诀：** Z1-Z3看清表情 / Z4-Z6看清动作 / Z7-Z9看清世界

## X轴 —— 环绕方位与立体感

| 编码 | 名称 | 角度 | 效果 |
|------|------|------|------|
| X1 | 正面 | 0° | 直接互动、打破第四面墙 |
| X2 | 四分之三侧（**默认**） | ~45° | 最立体最自然 |
| X3 | 侧面 | 90° | 轮廓、对视、剪影 |
| X4 | 背面 | 180° | 留白、悬念、跟随进入 |

## F层 —— 镜头物理能力与叙事身份

### F1 焦段

| 焦段 | 空间感 | 情绪 | 适用 |
|------|--------|------|------|
| 18mm 超广角 | 极度夸张 | 宏大、眩晕 | 建筑、极限运动 |
| 24mm 广角 | 轻微夸张 | 开阔 | 室内全景 |
| 35mm 标准 | 接近人眼 | 自然叙事 | **万能**日常 |
| 50mm 标准 | 最自然 | 经典电影感 | **电影标准**对话 |
| 85mm 长焦 | 轻微压缩 | 柔美亲密 | **人像黄金**特写 |
| 135mm+ | 强烈压缩 | 窥视戏剧 | 远距偷拍 |

### F1 景深

- 浅景深 — 主体锐利背景虚化（人像、产品）
- 深景深 — 前后清晰（环境交代、群像）
- 拉焦 Rack Focus — 焦点从A转到B（悬念揭示）
- 散景 Bokeh — 背景光斑柔美（夜景、浪漫）

### F2 叙事身份

| 类型 | 关键作用 |
|------|---------|
| POV | 代入角色眼睛 |
| OTS过肩 | 两人空间关系 |
| 偷窥 | 被监视感 |
| 镜面 | 双重性、自我审视 |

### F3 构图几何

| 类型 | 心理效果 |
|------|---------|
| 荷兰角（倾斜） | 不安、失衡 |
| 完美对称 | 秩序、仪式 |
| 引导线 | 深度、方向 |
| 框中框 | 聚焦、束缚 |
| 大量留白 | 孤独、呼吸感 |

---

## 🔴 核心铁律

### 铁律1：每镜最多双轴运动
三轴同时变化 = 失控。拆成两个镜头或去掉一个轴。

### 铁律2：近景+大旋转=崩脸
Z1-Z3（近景特写） + X轴大幅旋转 → 面部变形。换Z4+或减旋转幅度。

`Z9+Y7 → Z5+Y4 → Z2+Y4`

### B：静→动（积蓄→爆发）
`固定 → 缓慢推近 → FPV穿越`

### C：冷→暖（建立→共情）

## ⚠️ 与导演现有技能的互补关系

| 现有技能 | 做什么 | Z/Y/X/F 如何互补 |
|---------|--------|-----------------|
| 镜头逻辑性技能 | 景别选择逻辑、镜头序列 | Z/Y/X/F提供精准坐标系替代感性描述 |
| camera-terminology | 术语词典 | Z/Y/X/F是术语之上的数学化思维框架 |
| 构图技能 | 画面内构图法则 | Z/Y/X/F的F3延伸构图到镜头物理层 |

**使用建议：日常写分镜继续用现有技能；需要精准机位控制或复杂调度时调用Z/Y/X/F坐标编码。**

## 防漂移约束模板

写提示词时视场景叠加：
```
稳定流畅、真实摄影机运动、运动缓入缓出
保持人物身份一致、面部比例稳定、五官不变形
背景结构稳定、直线保持笔直、真实透视
4K超高清、细节清晰、无模糊闪烁
不要面部变形、多余手指、肢体扭曲、闪烁、跳帧、几何扭曲
```

---

## DO / DON'T

✅ 先定坐标（Z+Y+X），再叠滤镜（F），再写运动和节奏
✅ 每镜单轴或双轴运动
✅ 用情绪词描述速度（"缓慢推近"），不用数字
✅ 焦段写数值：85mm，不写"长焦"
✅ 多镜头用坐标递进（Z9→Z5→Z2）

❌ 一镜三轴同动
❌ Z1-Z3做X轴大幅旋转
❌ 同一时间切片混多种运镜
❌ "推进"+"拉远"同时出现

# 附录：镜头污染与大气质感增强（2026-07-13 补·源MapleShaw）

| 血渍 | 血滴溅上镜头 | 近身战斗 |

## 使用原则

- **不超过2种镜头污染**：多了模型会过载，画面变脏
- **镜头污染 + 大气效果各选1个**：搭配使用，不堆叠
- **文戏不用镜头污染**：固定镜头干净画面，打斗/灾难/恶劣环境才加
- **大气效果=文戏帮凶**：薄雾+丁达尔让文戏画面更有层次
