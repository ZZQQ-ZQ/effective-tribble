## Part A：摄影机机身（8台）

### ARRI Alexa 65 / 35

- **画面特征**：暖调、柔和肤色、高动态范围（17+档）、有机胶片感、大画幅立体感
- **颜色科学**：偏暖黄，皮肤过渡丝滑，高光滚降自然
- **代表作品**：The Revenant、1917、Joker
- **🔑 匹配场景**：史诗/文戏/情绪/人物特写/历史题材
- **不匹配**：动作片、高速运动（重量大）
- **提示词写法**：`shot on ARRI Alexa 65`

### RED V-Raptor / Komodo

- **画面特征**：冷锐锋利、高反差、数字感强、暗部偏冷、8K分辨率
- **颜色科学**：中性偏冷，锐度极高，暗部信息丰富
- **代表作品**：Guardians of the Galaxy、Mindhunter
- **🔑 匹配场景**：打斗/夜戏/科幻/动作/现代都市
- **不匹配**：浪漫/怀旧/温暖叙事
- **提示词写法**：`shot on RED V-Raptor`

### Sony Venice 2 / Burano

- **颜色科学**：S-Gamut3，中性偏冷，极致细节保留
- **代表作品**：Top Gun: Maverick、The Crown

## Part B：镜头焦段（9档）

### 镜头基础

- **焦距**：决定视角宽度和放大倍率。mm值越大=越"望远"=背景压缩越强
- **光圈 f/** ：值越小=进光越多=景深越浅。T-stop才是电影镜头真实进光量
- **球面（Spherical）vs 变形宽银幕（Anamorphic）**：球面=圆形散景自然视角，变形=椭圆散景水平拉丝2.39:1宽画幅

## Part E：集成到提示词格式

### 块1追加行

```
不要字幕，不要背景音乐。[年代地域] + [光线风格] + [摄影质感]，4K，超写实，极致逼真，UE5，Octane渲染
摄影机：ARRI Alexa 65，85mm f/1.4，Kodak Vision3 500T
```

### 格式说明

可在块1尾部，与渲染引擎同行或另起一行：

```
摄影机：[机身]，[焦段 mm + 光圈 f/]，[胶片]
```

三者全写=完整锁死画面DNA。可简化为2项（如只需机身+焦段，胶片省略）。

### 分镜中嵌入镜头焦段

当某镜需要特殊焦段（与块1全局不同），可在「运镜手法」或「构图」中标注：

```
3-6秒
景别：特写
构图：正面，画右向画左微侧
运镜手法：85mm f/1.4 浅景深，缓推
画面内动：...光效：...
```

### 变形宽银幕标注

使用变形宽银幕时在「运镜手法」或块1标注：
- `anamorphic Panavision C-Series` → 完整品牌
- `anamorphic 2.39:1` → 仅画面比例

## Part F：核心原则

1. **三件套锁死画面DNA**：机身定冷暖（ARRI暖/RED冷/Sony中）、焦段定空间感、胶片定色彩颗粒
2. **500T 是万能底**：Seedance 2.0 认证的最强美学锚点，不确定胶片选它
3. **打斗用 RED，文戏用 ARRI**：最基础的二分法
4. **变形宽银幕写完整品牌**：模型识别 `Panavision C-Series` > `anamorphic` 裸词
5. **光圈配合景别**：特写≤f/2.8 中景f/2.8-5.6 全景≥f/5.6
6. **分镜可覆盖全局**：某镜特殊焦段直接在「运镜手法」标注，覆盖块1设定
