# Part B：写光效（31种打光 + 分镜光效增强）

## B4. 电影感质感公式

```
主体 + 场景 + 动作
+ 材质质感（皮肤毛孔/面料肌理/金属反光）
+ 光线质感（方向/色温/软硬/体积光/阴影层次）
+ 色彩质感（橙青/褪色胶片/冷暖对比/饱和度层次）
+ 空气质感（雾/尘/大气透视/景深衰减）
+ 镜头运动（固定/跟拍/环绕/推拉/手持）
+ 情绪氛围（宁静/压迫/温暖/冷寂/紧张）
```

# Part D：核心原则

1. **一个组合覆盖一个项目** — 传奇定下 `写实史诗，自然日光，暖金色调` 就不改，仅微调摄影质感
2. **室外白天 = 自然日光** — 天然单光源，不写"光线方向"
3. **标签选具象不选空词** — ❌"华丽质感" ✅"金色调"
4. **能少则少** — 15个字以内塞块1，多了模型注意力分散
5. **调性一致贯穿全片** — 块1标签定了，整个项目15秒片段都用同一套
6. **光效写入内容不另起行** — 光效描述自然嵌入分镜动作中，Part B是写法参考不是格式要求

# Part E：光影三层结构写作法（2026-07-13 补·源MapleShaw）

> **写光效不笼统，拆成三层——每层一句话，覆盖全维度。**

## 模板

```
光影：{光源层}，{光行为层}，{色调层}
```

## 第一层：光源（是什么光、从哪里来）

| 场景类型 | 光源词 |
|---------|--------|
| 灾难/动作 | 暴雨逆光、爆炸橙红火光、闪电侧光 |
| 奇幻/科幻 | 怪兽熔岩自发光、能量球蓝白光、飞船引擎尾焰光 |
| 都市/夜景 | 霓虹灯漫射、建筑玻璃幕墙反射、车灯流光 |
| 自然/写实 | 暴风雨漫射天光、黄金时刻侧逆光、月光冷辉、火焰跳动暖光 |
| 仙侠/古风 | 灵光逸散、符阵金芒、仙气氤氲、烛火摇曳 |

## 第二层：光的行为（光如何与材质/大气互动）

```
薄雾柔化高光 / 雾层强化阴影层次 / 烟尘散射光线 /
玻璃折射彩虹光斑 / 金属反射爆炸火光 / 雨水折射霓虹 /
丁达尔效应清晰 / 体积光穿透尘埃
```

## 第三层：色调（整体冷暖和色彩关系）

| 风格 | 色调公式 |
|------|---------|
| 灾难/压迫 | 冷蓝底调 + 熔岩红高光 |
| 赛博朋克 | 冷蓝底调 + 霓虹紫红高光 |
| 奇幻/仙侠 | 青灰底调 + 金色荧光高光 |
| 末日/恐怖 | 灰绿底调 + 暗红点缀 |
| 暖色/史诗 | 暗棕底调 + 橙金高光 |
| 现代/写实 | 中性底调 + 暖白高光 |
| CG游戏 | 锐利干净底调 + 金属亮银高光 |

## 使用示例

```
光影：穹顶破洞暖金日光斜穿（光源层），
薄雾柔化高光但强化阴影层次、丁达尔效应清晰、金属铠甲反射暖金（光行为层），
```
