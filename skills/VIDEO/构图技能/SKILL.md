## 🔴 核心原则

**构图 = 画面元素的视觉权力分配。三个目的：**
1. **引导视线** → 控制观众先看什么、后看什么、停在哪
2. **传递情绪** → 构图本身就是情绪语言，不是装饰
3. **服务叙事** → 每个构图选择都是对角色状态/剧情关系的视觉判断

**写每镜前先过进阶三问：**
- 主体在画面里有多少"视觉重量"？（大/亮/锐/暖=重，小/暗/虚/冷=轻）
- 观众的第一眼落在哪？第二眼？最后停在哪？
- 这个构图在替角色说什么？（权力/脆弱/孤独/压迫/自由/迷失）

# Part 1：高频构图技法（6种，覆盖90%场景）

> 瘦身自原23种技法。其余低频技法归档到附录。

| # | 构图法 | Seedance中文关键词 | 情绪 | 适用场景 |
|---|--------|-------------------|------|---------|
| 1 | **三分法** | `主体位于画面左三分之一处` / `视线焦点落在上三分之一线` | 平衡/自然 | 对话/单人/双人/风景 |
| 2 | **中心构图** | `居中构图` / `主体位于画面正中央` | 权力/仪式/庄严 | 英雄登场/权力宣告/BOSS压迫 |

# Part 2：视觉重量与画面张力

> **不再问"人在画左还是画右"，而是问"这个画面里谁最重、为什么"**

## 2.1 视觉重量的六个来源

画面里每个元素都在"抢"观众的注意力。重量大的元素第一眼就被看到。

| 重量来源 | 重的表现 | 轻的表现 |
|---------|---------|---------|
| **亮度** | 高亮区域（脸/天空/火光） | 暗部/阴影 |
| **锐度** | 焦点清晰的主体 | 虚化/失焦/运动模糊 |
| **色彩** | 暖色/高饱和（红/橙/金） | 冷色/低饱和（蓝/灰/褐） |
| **大小** | 占据大面积 | 画面中的小点 |
| **人脸** | 有人脸的区域→天生最重 | 无人的景物 |
| **运动** | 画面中唯一在动的东西 | 静止元素 |

**铁律**：人脸 + 清晰 + 亮 = 最重。视觉重量不是绝对值，是对比出来的。

## 2.2 平衡 vs 失衡：什么时候该打破平衡

```
平衡 = 左右/上下的视觉重量大致相等 → 稳定/和谐/平静
失衡 = 一侧明显重过另一侧 → 紧张/不安/能量/戏剧
```

| 场景需求 | 用平衡还是失衡 | 手法 |
|---------|-------------|------|
| 日常对话/平静情绪 | 平衡 | 双人各占一侧三分线，重量对称 |
| 权力不对等（BOSS vs 小兵） | 失衡 | BOSS占70%画面+居中，小兵挤在边缘 |
| 内心崩塌/崩溃 | 失衡 | 人物挤在角落，大面积负空间压倒 |
| 决战一触即发 | 失衡→平衡交替 | 对峙用失衡蓄力，爆发瞬间切平衡打击 |
| 史诗大场面 | 平衡 | 对称/几何秩序 |
| 角色迷失/无助 | 极端失衡 | 人物极小，环境吞没 |

## 2.3 Short-Sided Framing（短边构图）

**不给角色的视线方向留空间。** 人物看向画右，但被挤在画面右边缘——前方没有"看出去的空间"。

```
标准：人物看向右 → 放在画左三分处（前方有空间）
短边：人物看向右 → 挤在画右边缘（前方撞墙）

情绪：被困/不安/无路可走/心理压迫
Seedance关键词：`人物挤在画面边缘，视线方向不留空间`
```

**何时用：**
- 角色被逼到绝境
- 内心崩溃、无处可逃
- 对话中被对方压倒（弱势方用短边，强势方给足空间）
- 囚禁/被困的场景

## 2.4 色调重量

亮区和暗区本身就是构图元素，不只是光影。

| 色调分配 | 画面效果 |
|---------|---------|
| 亮区占2/3 + 暗区1/3 | 开放/希望/主动 |
| 暗区占2/3 + 亮区1/3 | 压抑/神秘/被动 |
| 亮区居中 + 暗区包围 | 聚焦/孤独/被围困 |
| 暗区居中 + 亮区包围 | 不安/危险潜伏 |

**Seedance写法**：不写"暗部大面积"，写`亮区集中在人物面部，四周自然渐暗` 或 `画面大面积明亮通透，仅边缘渐暗收束视线`

# Part 3：构图即叙事

> **每种构图选择，本质上都是对角色内心状态的视觉翻译。**

## 3.1 中心构图的四种叙事

| 叙事功能 | 什么时候用 | Seedance写法 |
|---------|-----------|-------------|
| **权力宣告** | 英雄登场/BOSS出场/王者归来 | `居中构图，仰拍，人物占据画面中央，气场压迫` |
| **仪式感** | 登基/加冕/决战前 | `严格居中对称，正面平视，仪式性的静止` |
| **被审视/被审判** | 角色处于众人目光中心 | `居中构图，四周人群虚化环绕，正面平视` |
| **荒诞/喜剧** | 刻意对称到不自然（Wes Anderson式） | `绝对对称居中，人物僵直正面，人工感强烈` |

> ⚠️ 中心构图≠永远威严。同一构图在不同上下文里叙事完全不同。

## 3.2 偏离中心 = 角色状态外化

| 偏离程度 | 叙事含义 |
|---------|---------|
| 稍微偏离中心（近三分线） | 自然/舒适/日常 |
| 明显偏离（三分之一处） | 拉开距离/观察感/过渡状态 |
| 推向边缘（四分甚至五分处） | 被排斥/边缘化/脆弱/孤立 |
| 挤在角落（象限角落） | 被困/崩溃/无路可退/内心坍塌 |

## 3.3 头空控制 = 角色掌控力指标

```
标准头空（眼睛在上三分之一线）= 角色掌控局面，观众舒适
极少头空（头顶被切/紧贴画框上沿）= 被困/窒息/无处可逃
大量头空（头顶上方大片空白）= 渺小/迷失/无力/被环境吞没
```

**实战用法：**
- 角色被压倒→切极近景+极少头空，画面逼仄
- 角色觉醒/燃起→切中心构图+标准头空，占据画面主导
- 角色绝望→大全景+大量头空+负空间，人物是环境中的一粒尘埃

## 3.4 画框套画框 = 不止于层次感

| 框的类型 | 叙事含义 |
|---------|---------|
| 门框/窗框 | 囚禁/被困/隔绝 |
| 前景模糊人影 | 被监视/被跟踪 |
| 镜面反射框 | 身份分裂/自我审视 |
| 自然枝叶框 | 窥视/秘密/偷看视角 |
| 建筑结构框 | 制度压迫/体制困住 |

**Seedance写法**：不写"前景虚化营造层次感"，写`前景门框框住人物，画面压抑，人物如在牢笼中`

## 3.5 几何形状的潜意识叙事

| 形状 | 潜意识感受 | 何时用 |
|------|-----------|--------|
| △ 三角（尖朝上） | 稳定/权力/金字塔 | 权力结构/三人对峙 |
| ▽ 三角（尖朝下） | 不稳定/脆弱/倒置 | 权力崩塌/危机 |
| □ 方形 | 秩序/僵化/困住 | 体制/囚禁/规则 |
| ○ 圆形 | 完整/循环/团结 | 围攻/团圆/轮回 |
| ◇ 菱形 | 动态/紧张/锋利 | 对峙顶点/冲突聚焦 |

# Part 4：视线流动设计

> **不只看"画面里有什么"，而是控制观众的眼睛怎么走。**

## 4.1 视线路径三段式

```
进入点（第一眼）→ 路径（眼睛移动轨道）→ 终点（最终停留）
```

| 手段 | 进入点 | 路径 | 终点 |
|------|--------|------|------|
| 引导线 | 线条起点的亮/锐区域 | 线条方向 | 线条汇聚处（主体） |
| 目光方向 | 人物本身 | 人物视线方向 | 目光落点 |
| 亮度梯度 | 最亮处 | 亮度递减方向 | 最有信息量的暗部细节 |
| 运动矢量 | 当前动作位置 | 动作方向 | 动作即将抵达处 |
| 色彩梯度 | 最暖/最饱和处 | 色温递减方向 | 最冷/最不饱和处 |

## 4.2 视觉层级 = 信息释放顺序

每个画面都有主焦点、次焦点、背景信息。观众先看到焦点A→再看到焦点B→最后消化环境。

```
主焦点：画面中最亮/最锐/最暖/有人脸的元素 → 观众0.1秒就锁定
次焦点：稍微暗/虚/冷的次要信息 → 0.5秒后看到
环境层：背景/氛围/空间信息 → 1秒后消化
```

**叙事节奏控制：**
- 想让观众立刻理解→主焦点=核心信息
- 想制造延迟揭示→主焦点=误导，真实信息藏在次焦点里暗下来让观众2秒后才发现
- 想让观众沉浸氛围→主焦点不抢戏，环境层占主导

## 4.3 引导线的三种层级

| 层级 | 类型 | Seedance写法 |
|------|------|-------------|
| **显性引导线** | 道路/栏杆/墙壁/柱子/走廊 | `道路延伸指向画面深处的角色` |
| **隐形引导线** | 人物目光/手势/身体朝向 | `角色目光投向画右，引导视线` |
| **光线引导线** | 光束/阴影边界/光路方向 | `斜射光束指向主体，视线沿光路凝聚` |

## 4.4 视觉节奏：一个镜头内的信息密度

```
信息密度高 = 画面元素多 + 对比强 + 多处锐利 → 观众眼睛忙，紧张/混乱
信息密度低 = 画面简洁 + 大面积负空间 + 单一焦点 → 观众平静/专注/呼吸
```

**实战匹配：**
- 打斗/追逐 → 中高信息密度，画面多元素引导线交织
- 文戏/对话 → 中低信息密度，主体突出背景简化
- 独白/静默 → 极低信息密度，大面积负空间+单焦点
- 大场面揭示 → 从低密度切高密度产生震撼

# Part 5：正负空间的情绪修辞

> **负空间不是"空的地方"，是画面里会说话的沉默。**

## 5.1 负空间的六种情绪

| 情绪 | 正负空间比例 | Seedance写法 | 经典参考 |
|------|------------|-------------|---------|
| **孤独/孤立** | 负空间>>正空间 | `人物极小，被广阔空旷的背景包围` | Punch Drunk Love |
| **自由/开阔** | 负空间>>正空间 + 运动前方有路 | `主体占据画幅上三分之一，下方大片天空留白` | 公路片 |
| **压迫/窒息** | 正空间>>负空间 | `主体填满画面，几乎不留边缘空间` | 心理惊悚 |
| **神秘/悬念** | 负空间>>正空间 + 暗调 | `大面积冷灰留白，人物仅占画面边缘一角` | Blade Runner 2049 |
| **平静/禅意** | 正负空间均衡 | `主体居中偏三分处，四周均匀留白` | 日式侘寂 |
| **不安/异化** | 负空间在非预期位置 | `人物挤在画面左下角，右上方四分之三留白` | Mr. Robot |

## 5.2 "不展示"比"展示"更有力

```
展示给观众看 = 告诉观众答案
不展示 = 让观众自己脑补 → 恐惧/好奇/代入感翻倍
```

**用法：**
- 凶手/怪物/威胁不从正面拍，从门缝/阴影/前景遮挡中隐约可见 → 恐惧靠想象
- 关键道具不拍全景，只拍手握的部分 → 悬念
- 大BOSS不出场，只拍地上投射的巨大影子 → 压迫感来自未知

## 5.3 负空间+景别联动的情绪矩阵

| 景别 | 大量负空间 | 少量负空间 |
|------|-----------|-----------|
| 大全景 | 史诗级孤独/崇高空旷 | 战场密集/末世废墟 |
| 全景 | 人物疏离/旁观感 | 日常/叙事推进 |
| 中景 | 关系疏远/冷感 | 亲密/对抗/紧张 |
| 近景 | 心理孤立/内在孤独 | 情绪爆发/压迫 |
| 特写 | 眼神放空/失神/迷失 | 冲击/窒息/崩溃 |

# Part 6：场景→构图速配（升级版）

| 场景类型 | 主构图 | 辅助构图 | 禁忌 |
|---------|--------|---------|------|
| 单人特写/近景 | 三分法（偏一侧） | 中心（权力时刻） | ❌ 大量头空（分散焦点） |
| 单人情绪爆发 | 填充画面（极少负空间） | 中心 | ❌ 给留白（削弱冲击） |
| 单人孤独/迷失 | 负空间（人物在边缘） | 引导线（延伸向空虚） | ❌ 填充画面 |
| 双人平等对话 | 三分法（各占一侧） | OTS | ❌ 中心（没对话互动感） |
| 双人权力不对等 | 对角线/失衡 | 并置对比（大小悬殊） | ❌ 对称（权力对称不成立） |
| 对峙（一触即发） | 对角线 + Short-sided | 中心（各自镜头切换） | ❌ 均衡三分（太平稳） |
| 英雄登场 | 中心 + 引导线汇聚 | 仰拍三角 | ❌ 负空间（削弱气场） |
| BOSS压迫 | 中心 + 填充满画面 | 仰拍 + 极少头空 | ❌ 三分法（太松散） |
| 打斗/动作 | 对角线 + 运动空间 | 手摇失衡 | ❌ 中心构图（打架不走稳定） |
| 追逐 | 运动空间 + 引导线 | 对角线加速 | ❌ 静止框架 |
| 离别/告别 | 负空间 | 三分法（偏一侧留离去方向） | ❌ 中心/填充 |
| 被困/囚禁 | 框架构图 + 极少头空 | Short-sided | ❌ 开阔留白 |

# Part 8：常见翻车案例

| 翻车 | 为什么翻 | 正确做法 |
|------|---------|---------|
| 打架用中心构图 | 中心=稳定，打架=不稳定 → 情绪冲突 | 用对角线/失衡/运动空间 |
| 孤独场景人物太大 | 人物太大=填满画面=压迫，不是孤独 | 大全景+负空间，人物是环境中的一粒尘埃 |
| BOSS用三分法 | 三分法=自然舒适，BOSS需要压倒性权力 | 中心构图+仰拍+填充画面+极少头空 |
| 两人平等对话用中心构图 | 中心=单人主导，对话=双人互动 | 三分法各占一侧，OTS交代空间关系 |
| 英雄登场给大量留白 | 留白=弱化气场 | 中心+引导线汇聚+仰拍三角 |
| 角色绝望给标准头空 | 标准头空=舒适/掌控 | 大量头空（渺小）或极少头空（窒息） |
| 负空间放错位置 | 负空间在人物背后=没意义，在视线前方=叙事 | 负空间放在运动方向/视线方向 |
| 先让观众看不重要的东西 | 视觉层级乱，信息释放顺序错 | 亮/锐/暖/人脸=第一眼，要确保主焦点是核心信息 |

# Part 10：人物调度

> **构图是"框里怎么摆"，调度是"人怎么站、怎么走、面朝哪"。调度在先，构图在后。**

## 10.1 人物朝向五级（Face Direction）

跟摄像机的角度关系决定观众和角色的心理距离。

| # | 朝向 | 角度 | 情绪/叙事功能 | Seedance写法 |
|---|------|------|-------------|-------------|
| 1 | **全正面** | 正对镜头 | 直面观众→情绪直接传递、坦诚、对抗、宣告 | `人物正面面对镜头，目光直视` |
| 2 | **45度正面** | 微侧30°~45° | **最常用的电影角度**。有表情又保留立体感，兼具表现力与叙事性 | `人物呈四分之三侧面对镜头，目光投向画外一侧` |
| 3 | **纯侧面** | 90°侧面 | 旁观者视角→角色专注于他物，观众是窥视者/观察者 | `侧面拍摄，人物轮廓清晰，目光投向画外水平方向` |
| 4 | **45度背面** | 背面偏侧45° | 偷窥视角→神秘感、暗中观察、角色未察觉被看 | `背面偏侧拍摄，人物肩线和半边侧脸隐约可见` |
| 5 | **全背面** | 完全背对 | 孤独/疏离/离去/启程→观众与角色同向面对未知 | `人物完全背对镜头，面向画面深处` |

**实战速配：**

| 你想让观众… | 用哪个朝向 |
|------------|----------|
| 直面角色情绪（愤怒/宣告/崩溃） | 全正面 |
| 自然进入剧情，关注角色在做什么 | **45度正面（默认选择）** |
| 冷眼旁观，看角色专注某事 | 纯侧面 |
| 偷看、跟踪感、悬疑 | 45度背面 |
| 感受角色的孤独、与角色一同面对未知 | 全背面 |

## 10.2 高度差调度（Height Staging）

人物之间的垂直关系 = 权力关系的视觉翻译。

| 高度关系 | 视觉表现 | 叙事含义 | 用法 |
|---------|---------|---------|------|
| **高度一致** | 两人视线水平 | 势均力敌、平等对话、情绪平稳 | 日常对话、平等争论 |
| **一方更高** | 站起俯视坐者 | 支配/压迫/情绪更强烈 | 站起来=情绪升级；俯拍被压制方+仰拍压制方 |

**Seedance写法：**
- 平等：`两人视线水平，平视构图`
- 压制：`秦站立俯视，晚晴单膝跪地抬头仰视，仰拍秦+俯拍晚晴`

**铁律：**
- 角色"站起来"=情绪转折点，镜头必须跟上这个高度变化
- 一人坐一人站时OTS必须写明俯拍/仰拍方向，不能省略

## 10.3 距离调度（Distance Staging）

人物之间的水平距离 = 情感关系的视觉翻译。

| 距离 | 范围 | 关系 | 示例 |
|------|------|------|------|
| **远距离** | >3米 | 陌生/疏离/冷战/关系紧张 | 两人分站大殿两侧，中间空旷 |
| **中距离** | 1~3米 | 社交距离、正常对话、普通熟人 | 两人面对面站，间隔两步 |
| **近距离** | <1米 | 亲密（情侣紧靠）/ 侵略性压迫（脸贴脸威胁） | 秦逼近晚晴面前，鼻尖几乎相触 |

**Seedance写法：**
- 疏离：`两人分站画面两端，中间大面积空旷留白`
- 亲密：`两人紧靠，肩膀相触，面部距离极近`
- 压迫：`秦逼近晚晴面前，占据画面主导，晚晴被压缩在画框边缘`

## 10.4 前后景配合叙事（Depth Staging）

在同一画面内用景深同时讲两个故事。

```
焦点在前景 → 前景是"讲给观众听的故事"
焦点在后景 → 后景是"观众自己发现的真相"
前后都清晰 → 两个叙事线同时推进
```

| 模式 | Seedance写法 | 示例 |
|------|-------------|------|
| 前景焦点+后景虚化 | `焦点落在前景人物面部，后景人物虚化成模糊剪影` | 晚晴（前景）警觉回头，后景BOSS虚影逼近 |
| 后景焦点+前景虚化 | `前景虚焦人物肩部/背影，焦点落在后景主体` | 过肩镜头：前景虚焦肩部，后景人物清晰 |
| 前后双焦 | `深景深，前景和后景均清晰可见` | 秦（前景）拔剑，晚晴（后景）在远处蓄力 |

## 10.5 运动轨迹调度（Movement Trajectory）

人物在画面中怎么走 = 视觉冲击力和叙事节奏的控制杆。

| 轨迹类型 | 运动方向 | 视觉特征 | 情绪 | 适用场景 |
|---------|---------|---------|------|---------|
| **径向** | 面向/背离镜头 | 视觉冲击力最强、压迫感/紧迫感 | 冲锋→压迫；离去→疏离 | 士兵冲向镜头、角色走出画面深处 |
| **横向** | 左→右或右→左 | 旅程感、过程感、探索感 | 平静叙事、展示环境 | 公路飞驰、角色穿过街市 |
| **对角线** | 一角→对角 | 空间纵深+横向叙事结合，动感最强 | 能量/冲突/速度 | 追逐、战场冲锋、坠落 |

**结合运镜：**
- 固定机位+人物径向移动 → 压迫感最强（人物冲向观众）
- 跟拍+人物横向移动 → 旅程沉浸感
- 摇镜+人物对角线移动 → 视差效果+电影感（正面奔跑→摇至背影）

**Seedance写法：**
- 径向：`秦从画面深处冲向镜头，身形急剧放大，压迫感拉满`
- 横向：`秦从画面左侧奔向右侧，摄像机侧面跟拍，背景环境横向掠过`
- 对角线：`秦沿对角线从画面右下冲向左上，摄像机低角度仰拍跟摇`

**运动轨迹组合（追逐/战斗复杂场景）：**
```
镜1：背面入画，径向→纵深跑入画面深处（建立空间）
镜2：侧面跟拍，横向→展示奔跑速度和环境
镜3：极低角度仰拍，对角线冲刺→结合动感与冲击力，收尾爆发
```

## 10.6 双人中景运镜规则

| 做法 | 效果 |
|------|------|
| 固定不动双人中景 | 画面平淡、代入感弱 → ❌ 避免 |
| 高位/低位环绕运镜双人中景 | 保持两人位置不变同时展示环境信息 → ✅ 推荐 |
| OTS过肩反打 | 切换对话视角，代入感极强 → ✅ 对话首选 |

**铁律：双人中景≠固定镜头。必须带运镜——环绕/缓推/微摇，不动就废了。**

## 10.7 调度→构图决策链

```
1. 定朝向（角色面朝哪→决定观众心理距离）
2. 定高度差（谁高谁低→决定权力关系）
3. 定距离（多远→决定情感亲疏）
4. 定运动轨迹（怎么走→决定视觉节奏）
5. 选构图（把上面四条落成画面布局）
```

> 交叉引用：人物朝向与镜头轴向 → 见镜头逻辑性技能 §四、180度轴线

# 🔴 铁律（v2.0 扩充）

1. **构图服务于叙事，不是反过来** — 先确定这段要什么情绪/角色状态，再选构图
2. **视觉重量由对比决定** — 亮+锐+暖+人脸=最重，不是绝对值是对比出来的
3. **运动方向必须有空间** — 人物朝右看→放画左；人物朝左跑→前方留空
4. **打架不走中心构图** — 打架=不稳定，中心=稳定，两者冲突
5. **负空间放在视线方向/运动方向** — 有意义的留白，不是随便空一块
6. **头空=角色掌控力** — 标准=舒适，极少=窒息，大量=渺小
7. **信息释放顺序就是叙事节奏** — 第一眼看到的不一定是真相
8. **一段15秒内不要换超过2种构图逻辑** — 构图跳来跳去观众头晕
9. **单人对话永远是近景或特写** — 别在对话镜头里搞复杂构图
10. **一人坐一人站时OTS必须写明角度方向** — 俯拍/仰拍/平视不能省略
