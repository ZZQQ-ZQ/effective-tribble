## 一、核心设计哲学

### 1.1 两派对照（知道自己在拍哪种）

| 维度 | 奇观型·第三人称 | 沉浸型·第一人称 |
|------|---------------|---------------|
| 代表作 | John Wick 系列 | 谍影重重 |
| 运镜 | Steadicam+Dolly 流畅跟拍 | 手持晃动+快速变焦 |
| 景别 | 广角全景/中景为主 | 近景/特写为主 |
| 剪辑 | 慢切，常长镜一镜到底 | 快切（1-3秒） |
| 光影 | 高饱和色彩对立，硬光 | 自然光，柔和 |
| 观众位置 | 坐在台下欣赏奇观 | 被卷入第一人称视角 |

**当前导演指定**：取奇观型的广角大景别和流畅运镜，融入第一人称的POV/FPV主观视角，两者切换——机动用POV代入，碰撞用广角展示。

### 1.2 最高铁律三条

1. **全程无静止画面** — 画面内永远有东西在动（人动/光动/尘动/衣摆动）
2. **全程无固定镜头** — POV/环绕/FPV/跟拍四种轮换
3. **快慢交替不超3秒** — 每段15秒至少2次速度切换

## 二、运镜体系

### 2.1 四种主运镜（只此四种，不可用固定镜头）

| 运镜 | 英文 | 镜头感受 | 适用场景 |
|------|------|---------|---------|
| **POV** | Point of View | 主观视角·我就是角色 | 秦冲向怪、怪看秦闪避逼近、石块砸向镜头 |
| **环绕** | Orbit | 压迫感·蓄力中绕圈 | 蓄力时绕秦转、对峙时绕双方转 |
| **FPV** | First-Person跟随 | 追逐感·跟在身后 | 镜头跟在秦身后/身侧高速移动，秦的身影在前景 |
| **跟拍** | Tracking | 轨迹感·侧面/背面 | 侧面或背跟，保持人物在画面中的运动轨迹 |

### 2.2 运镜搭配法则

- **不叠两种冲击型运镜**（如手摇+FPV→太晃看不清）
- **机动阶段**：POV↔FPV↔跟拍灵活切
- **蓄力阶段**：环绕+缓推结合
- **碰撞瞬间**：切POV或特写跟拍，慢镜
- **3秒/镜最多2种运镜**（硬限）

### 2.3 John Wick 4 级运镜执行标准

来自 Dan Laustsen 掌镜实战：
- A机 Dolly + B机 Steadicam AR 双机同滚
- Steadicam 可随时高/低模式切换
- 全片光孔 T/4，深景深保动作面全清晰（打斗不能重来太多次）
- 顶拍一镜到底用 Spidercam 架线横跨全场
- 广角 ALFA 变形宽银幕镜头（32mm端）覆盖全身动作空间
- "Don't move the camera too much" — 但当人物动时，镜头精确同步跟踪

## 三、节奏体系：快慢交替

### 3.1 速度切换原理（来自吴宇森《喋血双雄》）

**核心发现**：正常速度 → 慢动作 → 正常速度的反复切换，制造连续的加速/减速感。观众永远在适应新速度，感觉持续动态、永不静止。

```
慢镜·长焦特写 → 建立谁是主角/反派
正常速度·广角俯拍 → 混乱爆发
慢镜·广角仰拍 → 暴力放大
慢镜·长焦特写 → 反派反应（被困在同一镜头里）
正常速度·广角 → 主角冲入（突然加速）
慢镜 → 主角开枪（又减速）
正常速度 → 清理余敌（再加速）
```

### 3.2 两种快门角度的战斗用法

| 快门角度 | 运动模糊 | 视觉感受 | 战斗用途 |
|---------|---------|---------|---------|
| **45°** | 极少 | 锐利/卡顿/抽搐感 | 雷霆一击、断骨瞬间、决定性碰撞（拯救大兵瑞恩开战） |
| **90°** | 较少 | 锐利有力/能量感 | 机动蓄力、高速格挡、兵器交击 |
| **180°** | 标准 | 自然流畅 | 收势/喘息/常规运动（人类视觉） |
| **360°** | 严重拖尾 | 迷幻/眩晕 | 角色昏迷/醉酒POV |

**实战搭配**：机动阶段 90°快门（动作锐利有力）→ 碰撞瞬间 45°快门+慢镜头（每个关节位移都看清）

### 3.3 15秒节奏模板（每次出稿前心里必须有这条曲线）

```
[快起] 机动突进(2-3s) → [慢] 蓄力特写(1s) → [爆发] 碰撞慢镜(1s) → [快] 连斩追击(3-4s) → [慢] 角力较劲(2s) → [爆发] 震退拉开(1s) → [快] 再次接敌(2-3s)
```

- 全程不允许连续3秒以上同一节奏
- 全段至少2次快慢切换
- 快段：广角+正常速度+90°快门 / 慢段：长焦+慢镜+45°快门

## 四、卖打击技法：摄影的五个谎言

电影打斗不是拳拳到肉，是靠摄影+剪辑造的「谎言」。以下五种技法是全球动作电影的标准手段：

### 4.1 长焦压缩空间

**原理**：长焦镜头压缩纵深距离，两人面对面时，拳头距脸10cm观众也以为打中了。

**操作**：
- 过肩角度（OTS）+ 85-135mm长焦
- 摄影机放在被打者肩后
- 出拳方向朝向镜头（沿Z轴纵深运动）
- 配合轻微偏离正面的角度——观众看不到实际间隙

### 4.2 侧面+广角+朝镜头打

**原理**：钩拳从画外远处向镜头挥来，广角放大横向速度。

**操作**：
- 侧面双人中景 + 广角（24-35mm）
- 演员钩拳从远离镜头的位置向镜头方向挥
- 演员根本不用碰到对方——速度感和空间压缩已足够
- 来自 Taken 终极一战

### 4.3 武器插片特写

**原理**：碰撞瞬间切到武器的极近特写，下一镜切反应，观众脑补完成打击。

**操作**：
- 刀尖/剑刃/拳头贴到极近（广角微距）
- 划过画面、火花溅射、甲壳凹陷
- 切→对手的被击反应
- 中间省略实际接触
- 来自 Shotdeck 分析

### 4.4 动作轴线切镜

**原理**：在动作最高点切镜，不在准备阶段或收尾切。

**操作**：
- 出拳打出去的瞬间切（不是蓄力时）
- 刀尖划过画面→切→演员捂住伤口
- 剑劈下去的弧线最高速点切→下一镜被砍中

### 4.5 慢镜放大破坏

**原理**：决定性一击用慢镜头逐帧展示破坏。

**操作**：
- 45°快门 + 慢镜（48/60/120fps）
- 盔甲逐层崩开的纹理
- 碎片飞散轨迹清晰可见
- 火星凝固在空气中
- 持续0.5-1秒，接到正常速度的下一个动作

## 五、场景光效同步

### 5.1 John Wick 色彩对立体系（Dan Laustsen 方法论）

**核心信条**：「灯能放多远放多远」——远距硬光源让光质保持一致性，不会随人物移动跳跃。

**色彩对立原则**：
- 主光 青蓝/钢蓝 → 环境光 橙红/暖黄
- 拉开前后景层次
- 人物面部同时被冷暖和硬软光夹击 → 立体感拉满

**Pratical Lighting（画面内藏灯）**：
- LED灯条嵌在场景建筑的荧光灯罩中
- 广角怎么拍都不会穿帮
- 罗马地下墓穴：全蓝LED → 呼应Wick被拖回杀手世界的忧郁

### 5.2 招式联动光效（导演指定·关键）

光不是死的，是活的——随招式起落同步变化：

| 动作 | 光效同步变化 |
|------|------------|
| 刀身火焰爆燃 | 画面暖色光同时增强，周围树木被照亮的范围扩大 |
| 怪砸地 | 地面冲击波扩散时伴随暗光环境的短暂明灭 |
| 秦被震退 | 火焰刀拖在地面划出火星轨迹，火光在地面拖出长影 |
| 抱拳蓄力 | 红光从拳头炸开，环境Pratical灯同时闪 |
| 闪避后退 | 蓝绿暗角收缩，光线骤然压暗 |
| 武器碰撞 | 接触点火花作为瞬间补光改变人物面部光影方向 |

### 5.3 环境元素活性化

- 扬尘在光束中飘动 → 随冲击波扩散方向
- 落叶被剑气卷起 → 沿剑痕轨迹旋转
- 火星从碰撞点溅射 → 落在地面持续发光
- 碎石从地面震飞 → 不同大小不同抛物线
- 不是静止背景，是战斗的一部分

## 六、面部神态链

### 6.1 全程面部轴线

不是打完才看表情——每个阶段都有对应的面部。

| 阶段 | 表情描写 | 关键词 |
|------|---------|--------|
| 出招前 | 目光聚焦锁定、眉间微收、下颌微绷 | 聚焦、锁定 |
| 发力瞬间 | 牙关咬紧、鼻翼微张、脖颈筋腱凸显 | 爆发、绷紧 |
| 命中后 | 眼中一闪而过的满意/杀意，嘴角微抿收住（不是笑） | 收敛、暗涌 |
| 格挡/被击 | 眉间一紧、牙关咬合、闭气硬顶、脖颈绷住 | 硬顶、不服 |
| 喘息/恢复 | 嘴巴微张换气、眼皮微垂再抬起、喉咙吞咽 | 缓冲、再生 |
| 看见新威胁 | 眼睛微睁大→随即收窄、嘴角收敛变严肃 | 警觉、转场 |
| 蓄力中 | 下颌紧绷、眼中火光映照、嘴角收敛 | 蓄势、压迫 |

### 6.2 表情描写法则

- ✅ 表情压着写——模型会放大情绪，用微扬/淡淡/一抹/若有若无/隐约
- ✅ 同一场打斗中秦的表情要有弧线：兴奋→专注→受挫→越打越来劲→疲惫但硬撑
- ✅ 「嘴角一翘」一天只能用一次
- ✅ 表情服务于「窒息压迫感」——不让观众松一口气
- ❌ 每镜结尾都笑一下
- ❌ 永远「嘴角一翘，眼中映着火光」

### 6.3 《喋血双雄》级面部构图锁定

来自吴宇森手法：反派始终在**同一个长焦特写机位**中出现 → 反派被「困」在同一个镜头里；主角永远在**新机位、新画面**中出现 → 不动vs动本身就是权力关系的视觉外化。

## 八、完整对抗逻辑链

### 8.1 四段式循环（每轮战斗必须循环）

```
接触 → 角力 → 分开 → 机动再打 → 接触 → ...
```

1. **接触**：秦出招，怪迎击/格挡，兵器与爪碰撞
2. **角力**：双方兵器架在一起，互相推压较劲——肌肉紧绷、脚下碾碎石板、兵器颤抖
3. **分开**：一方被震退或双方同时弹开，拉开距离
4. **机动再打**：双方迅速调整站位，再次冲锋接敌——不站定等待

### 8.2 位置与距离

- 远近交替：贴身缠斗→震退拉开→远程攻击→再次近身
- 秦的路径：弧线+折线，连续变向不停顿
- 借蹬敌人身体反冲力换向（踩膝盖/蹬胸甲/踏后腰）
- 怪散落四周不集中正面，秦在包围中穿梭而非面向敌人站定

### 8.3 打击反馈完整链

每处碰撞必须包含三项：

| 反馈维度 | 内容 | 示例 |
|---------|------|------|
| **撞击材质反馈** | 什么碰什么→产生什么 | 金属碰爪=火星溅射+震颤 / 爪拍地面=碎石飞溅+尘土爆起 |
| **力量对冲表现** | 双方身体反应 | 肌肉紧绷+脚下碾碎石板+兵器颤抖+手臂回震 |
| **冲击波扩散** | 环境连锁反应 | 环形/锥形冲击波从接触点扩散，扬尘/落叶/碎石同步响应 |

## 九、动作词库

### 9.1 力度词+速度词（15G·永久规则）

**所有动作裸动词一律禁用，必须带力度词+速度词。**

| 力度词 | 速度词 |
|--------|--------|
| 快速/猛烈/暴射/凌厉/疾速 | 高速/极速/闪电般/瞬间/迅疾 |
| 迅猛/凶狠/奋力/狠砸/暴力 | 一闪/快如/骤转/刹那间/弹射般 |
| 急速/疾冲/猛蹬/狠劈/狂扫 | 暴砍/迅斩/骤然 |

**❌ 裸动词（禁用）**：挥剑、劈下、横扫、闪避、落地、格挡、蹬地、转身
**✅ 锚定写法**：快速挥剑、猛烈劈落、凌厉横扫、急速闪避、迅猛落地、奋力格挡、暴力横抡、凶狠砸地、猛蹬地面、骤然转身

### 9.2 剑气规则

- ✅ 每次挥剑都带剑气——剑划过的轨迹留下可见能量残留（淡色弧光/空气扭曲/微光拖尾），这是剑的视觉常驻效果
- ✅ 剑气延伸攻击距离：1米剑能砍到1.5-2米外的目标
- ✅ 剑气强度随力度变化：普通=轻淡弧线、加速重劈=宽幅气刃拖尾、全力爆发=剑气冲出剑身带冲击波
- ✅ 剑气长度不限物理剑身：重招剑气可拉数米长
- ✅ 能量巨剑：爆发时剑身火焰爆燃→剑体肉眼可见膨胀变大至数丈→巨大化剑劈下。物理剑和能量是一体的，不是分开两层。（2026-06-24 导演纠正）

### 9.3 发力链写法

从「挥剑」升级为完整关节链：
```
重心下沉→腰胯猛然拧转→肩部推送→肘关节伸展→腕部拧转发力→五指攥紧剑柄→剑身火焰爆燃膨胀→冲击波脱刃
```
每个关节的拧转/伸展/发力传导写进内容。

### 9.4 轻功身法（9式）

| 动作 | 核心描写 | 用途 |
|------|---------|------|
| 飞檐掠影 | 踏檐而过/身轻如燕/衣摆掠风 | 开场位移 |
| 借墙腾跃 | 脚尖点墙/二次借力/身体拔高 | 垂直空间 |
| 落地卸力 | 屈膝缓冲/手掌触地/力量化开 | 落地过渡 |
| 滑步换位 | 贴地滑步/瞬间错身/攻守反转 | 闪避反击 |
| 回身急闪 | 上身后仰/腰腹发力/回身避刃 | 闪避 |
| 半空转体 | 凌空转身/身体拉开弧线 | 腾空机动 |

## 十一、打斗描述规范

### 11.1 镜头内容描述顺序

每个镜头的【内容】按以下顺序写：
1. **光效描述**（光影方向/逆光/折射/火光映脸等）
2. **人物着装质感描述**（战袍衣料反光/皮毛纹理/布料随风等）
3. **背景描述**（山体轮廓/地面状态/远处环境等）
4. **动作描述**（发力/位移/出招/中招等）

### 11.2 单镜多节拍

- 打斗一镜多节拍：攻→防→转攻→攻→反→攻→收，每秒推进
- 连续事件装在一个镜头里——抬手→火幕出现→转头→说话
- 少镜头塞多节拍，一镜到底连杀多只

### 11.3 分级描写

- 不重要的小招用一句带过
- 关键时刻放大写
- 每招必须推进：位置变了/怪受伤了/被震退了/优势易手了
- 每招必有反应：秦出招→怪如何反应（后撤/挥爪迎击/硬扛/被震退）

### 11.4 出力源写法

写力量来源不写结果：
- ✅「腰部发力带动右肩，剑从右上斜劈而下」
- ❌「一剑劈下」

### 11.5 怪攻击特效锚定

大猩猩/土系怪物特效（土色系列）：
- 砸地 → 土黄色冲击波从落拳点环形扩散
- 挥爪 → 灰白色风压弧形气刃，爪尖带土黄色能量残影
- 丢石块 → 石块带尘土拖尾，爪尖能量渗入石块
- 咆哮 → 声浪化为土色半透明波纹
- 每镜具体锚定颜色/形态/程度

**#36 踢墙借力**：脚踩墙面借力折返，以更快速度飞冲回击，身体线条拉满。光效：落点爆闪。动态：墙面裂痕、腾空轨迹。镜头：镜头俯冲。适用：空间利用、机动突袭。

### 特殊爆发（#37-41）

**#37 连击追打**：短时间多段连续攻击，拳、腿、转身动作一气呵成。动态：多重残影、连续速度线、节奏卡点。镜头：镜头快速切换。适用：压制连段、收割残血。

**#38 爆气震退**：极限状态下瞬间爆气，力量自身体中心向外炸开，将敌人强行震退。光效：能量环、背景压暗、主角高亮、大范围冲击波。动态：碎石悬浮。适用：变身/觉醒、清场、扭转战局。

**#39 蓄力终爆**：先短暂停顿蓄力，再将全部力量一口气打出，攻击前后形成强烈反差。光效：周围能量汇聚、背景收束、落点大爆炸。动态：慢动作拉满。适用：BOSS终结技、压轴一击。

**#40 双向对冲**：双方同时冲刺后正面对撞，力量在画面中心爆开。光效：中心白热光、碎片四散。动态：对称构图、慢动作定格。镜头：镜头巨震。适用：巅峰对决、最终碰撞。

**#41 雨点乱打**：高速贴身输出，短拳、肘击、膝撞连续倾泻，对手几乎没有喘息空间。光效：节奏爆点。动态：密集残影、连环速度线、局部震屏。适用：狂暴输出、压制收割。

### 终结收招（#42-45）

**#42 终结一击**：最后一招集中全身力量爆发，角色气势拉满，攻击落点形成最强冲击。光效：大范围光效、背景压暗、主角高亮、爆炸式特效。动态：慢动作定格。适用：战斗收尾、必杀技。

**#43 踏墙反冲**：借助墙面发力反蹬，再次高速扑向目标，动作干净利落。光效：空气压缩感。动态：墙体碎屑、腾跃残影。镜头：镜头跟冲。适用：折返突袭、终结追击。

**#44 抢身直撞**：不留空档地抢身逼近，用胸肩和步伐一体推进，将对手瞬间撞开。光效：空气炸裂线。动态：前冲透视、地面摩擦痕。镜头：镜头跟退。适用：开场压迫、终结冲击。

**#45 收招余震**：最强一击结束后，角色缓缓收势，周围仍残留余震、飞尘和能量波纹。光效：能量波纹、碎屑飘落。动态：背景静止对比、主角压场姿态。镜头：慢收镜头。适用：战后余韵、气场收尾。

### 使用方式

写打斗分镜时：
1. 确定招式类型（拳/腿/膝肘/兵器/摔投/爆发/终结）
2. 在对应分类中选出具体招式编号
3. 将该组的 光效+动态+镜头 三项直接嵌入分镜内容描述
4. 多个招式连段时，逐组叠加效果

**示例**：
```
原分镜：秦牧右拳直击骷髅兵面部
查 #1 正面重拳 →
改为：秦牧蓄力右拳轰出，拳风在镜头前爆开，衣摆被气流掀起，
强对比边缘光勾勒手臂轮廓，冲击波自拳面扩散，镜头轻震
```

## 十四、AI速度感增强（2026-07-21 全网验证）

> **问题**：AI打斗视频常见动作绵软、拖沓、像在水中——不是动作设计问题，是关键词缺少「物理重量」和「速度锚定」指令。
> **来源**：Higgsfield/Seedance 2.0官方指南、Reddit r/generativeAI社区、Imagine.art实战指南

### 14.1 速度斜坡（Speed Ramp）——让打斗有张有弛

AI模型默认所有动作「匀速慢放」。必须用速度指令打破这个平衡。

**Higgsfield验证指令（直接嵌入分镜动作描述尾部）**：
```
→ RAMPS TO SLOW MOTION （碰撞瞬间进入慢动作）
→ SNAPS BACK to normal speed （打完切回正常速度）
```

**用法**：
- 机动突进阶段 = 正常速度
- 碰撞/命中/爆炸瞬间 = `RAMPS TO SLOW MOTION`
- 收招/拉开/追击 = `SNAPS BACK`

**每15秒至少2次快慢切换**（与 §3.3 节奏模板联动）。

### 14.2 物理约束词——防止角色「变成液体」

Reddit社区发现：Seedance在高动作密度场景中，角色出拳时身体会意外融化/扭曲。解决方式是追加物理约束：

```
consistent gravity, realistic material reaction
```

**嵌入位置**：画面基调行或块1尾部追加。

**效果**：
- 角色保持实体感，不会在高速动作中被「模糊化」成流体
- 碰撞点材质反应真实（金属碰金属有火花、非拳头穿胸而过）
- 重力感稳固——跃起后自然落回，不会悬浮

### 14.3 每镜一个清晰动词——别给模型塞选择题

Imagine.art Seedance指南强调：

> "Use one clear verb in the present tense. Focus on a single movement per shot."

**❌ 错误**：一个镜头塞多个动作（冲刺+跳起+挥剑+闪避+反击）→ 模型随机选一个，剩下的全糊
**✅ 正确**：每镜只写一个核心动作，动作咬动作靠镜头切换

**写作规范**：
- 每镜内容只含**一个主动词**（strike / slash / lunge / block / dodge / slam）
- 前镜结束状态 = 后镜起始姿态（§8.1 四段式循环落地）
- 不使用 `then` / `and then` 串联多动作（模型不遵循时序）

### 14.4 镜头语言制造速度感——不是让角色跑更快

AI不是让角色动作加速，而是通过运镜类型制造「视觉速度」：

| 运镜 | 速度感来源 | 提示词关键词 |
|------|-----------|-------------|
| 手持摇晃 | 不稳定=紧张 | handheld shake, camera jitter, micro-tremors |
| 快速横摇 | 背景模糊拉速度 | whip pan, fast lateral tracking |
| 震屏 | 碰撞重量感 | camera shudders, shakes on impact |
| 贴脸特写 | 透视夸张放大位移 | wide-angle close-up, perspective distortion |

**Higgsfield Orb格式示范（极速手持风格）**：
```
hyper-chaotic handheld motion, completely unstabilized, violent raw human movement, 
constant micro-jitters, aggressive head swings, frequent over-rotation and harsh correction,
moments of near motion blur loss, no smoothness at all, no stabilization
```

### 14.5 高频动作关键词——社区验证有效

| 效果 | 关键词 | 适用场景 |
|------|--------|---------|
| 速度感 | `high-velocity motion` | 机动/追击阶段 |
| 运动模糊 | `motion blur` | 快速挥砍/冲刺 |
| 连击密度 | `rapid-fire action sequences` | 多段攻击 |
| 打击重量 | `heavy impact, weight behind each strike` | 重击 |
| 惯性 | `momentum carries through` | 收招/过渡 |
| 爆发力 | `explosive force, sudden burst` | 蓄力后释放 |

### 14.6 后期剪辑补充速度感（提示词之外）

以下效果**不在提示词中写**，由剪辑阶段完成：

| 后期技法 | 用途 |
|---------|------|
| 击中瞬间0.1秒定格 | 让观众「看到」碰撞 |
| 变速曲线（speed ramping） | 慢→快→慢的自然过渡 |
| 动态模糊叠加 | 高速挥砍的残影感 |
| 黑白闪烁 | 配合重击的视觉冲击 |

> **原则**：提示词给「素材」——正常速度的动作 + 慢动作碰撞节点；后期给「爆发」——定格、变速、特效叠加。

### 14.7 写打斗提示词前的三步自问

1. **有速度斜坡吗？** → 碰撞节点 `RAMPS TO SLOW MOTION` / 机动 `SNAPS BACK`
2. **有物理约束吗？** → `consistent gravity, realistic material reaction`
3. **每镜只一个动词吗？** → 不串联多动作、不写 `then`
