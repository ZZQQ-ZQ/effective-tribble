# 人物表演管理技能

> 写分镜内容前强制调用。表情和语气是表演的一体两面——同一个原则管两样。
> 创建：2026-07-01 | 2026-07-06 合并人物表情自然化技能 | 2026-07-07 全网调研扩充（vocus/threads/阿里云/dexhunter）

---

## 核心原则

**AI模型对情绪词的响应是放大性的。你写"极度恐惧"，AI画"面部扭曲"。你写"眼神一狠"，AI画"瞪眼龇牙"。**

**写状态，不写表演。少即是多。**

**对话场景例外：写微表情，不写情绪标签。写肌肉状态，不写脸谱化形容词。**

---

## 🔴 应用规则

| 场景 | 规则 |
|------|------|
| 纯动作/战斗/无对话 | 写身体不写脸谱（铁律1-5适用） |
| 对话/互动/情绪交流 | 启用微表情拆解（§二 + §三），入画角色都要写 |
| 有参考图的角色 | 不写五官变化，只写姿势+身体微动作 |

## 一、表情管理（面部）

### 五条铁律

| 铁律 | ❌ 违规 | ✅ 正确 |
|---|---|---|
| **1. 写状态，不写切换** | "表情从茫然转为惊恐再跌为惊慌" | "嘴角微动，眼神发虚" |
| **2. 写身体，不写脸谱** | "眼睛瞪大，嘴角下拉，眉心紧锁" | "身体后仰半步，手不自觉伸出" |
| **3. 写动作链，不写情绪词** | "表情瞬间垮掉" / "眼神一狠" | "刀尖垂向地面" / "抬起的手微颤" |
| **4. 一镜一层** | 同一镜头写3种情绪 | 一镜只承载一个情绪状态 |
| **5. 面部留白** | 有参考图还写五官变化 | 有参考图→只写动作站位+身体微动作 |

### 🔴 铁律2的例外

对话场景中，入画角色的微表情必须写——不写"面无表情"，写自然放松状态或微表情。详见 §三。

## 二、情绪→微表情速查表（对话场景启用）

写微表情 ≠ 写表演。写的是面部肌肉的自然状态，不是"眼睛瞪大""嘴角下拉"。

### A. 喜悦类

| 情绪 | 微表情 |
|------|--------|
| 自然微笑 | 嘴角微微上扬，眼角微弯，眼神放松，脸部肌肉放松 |
| 温柔微笑 | 嘴角轻扬，眼神柔和，脸颊微陷酒窝 |
| 害羞笑 | 视线微向下，脸颊微红，嘴角含蓄 |
| 开心灿笑 | 眼睛弯成月牙，表情有活力，露出牙齿，脸颊微红 |
| 幸福笑 | 嘴角自然上扬，眼神温暖，眼角微带细纹 |
| 俏皮笑 | 眼神灵动，嘴角一侧微微上扬，带有玩味 |
| 得意 | 下巴微抬，嘴角一侧扬起，眼尾微挑 |
| 欢快 | 眉尾上扬，眼角弯弯，嘴角自然翘起 |

### B. 沉思类

| 情绪 | 微表情 |
|------|--------|
| 若有所思 | 视线望远，眼神沉静，眉间轻蹙，嘴唇微抿 |
| 专注 | 瞳孔微缩，嘴唇微抿，眉心轻收 |
| 好奇 | 眼神明亮，头微倾，嘴唇微张 |
| 困惑 | 眉间轻蹙，眼神飘移，唇角微垂 |
| 深思熟虑 | 目光低垂，眉间深锁，下颌线紧绷 |

### C. 冷/自信类

| 情绪 | 微表情 |
|------|--------|
| 自信 | 眼神坚定，嘴角自然上扬，下巴微抬 |
| 冷淡 | 嘴角若有若无微扬，眼神平静 |
| 沉稳 | 眼神温和专注，面部肌肉松弛 |
| 轻蔑 | 嘴角一侧撇起，下巴微扬 |
| 满意 | 嘴角微扬，眉尾轻挑，眼神带光 |
| 高冷 | 淡淡微笑，眼神平视，嘴角克制地上扬 |

### D. 紧张/压抑类

| 情绪 | 微表情 |
|------|--------|
| 紧张 | 喉结微动，嘴唇抿紧，眨眼频率略增 |
| 不安 | 眉心轻蹙，视线微移，嘴角微动 |
| 压抑 | 下颌微收，眉尾下压，嘴唇抿成一线 |
| 隐忍 | 咬肌微鼓，喉结滚动，眼睑微颤 |
| 焦虑 | 眉心皱起，视线飘忽，手指无意识轻敲 |

### E. 悲伤类

| 情绪 | 微表情 |
|------|--------|
| 悲伤 | 眼眶微红，嘴唇轻颤，眉尾下垂 |
| 哽咽 | 喉结滚动，眼眶湿润，嘴角微抽 |
| 落泪 | 单滴泪沿脸颊滑落，嘴角轻颤 |
| 心碎 | 眼眶蓄泪，嘴唇微张却无言，下颌轻抖 |
| 委屈 | 嘴角微微下垂，眼神楚楚可怜，眼眶微红 |

### F. 惊讶/恐惧类

| 情绪 | 微表情 |
|------|--------|
| 惊讶 | 瞳孔微缩，嘴唇微张，眉毛上扬 |
| 震惊 | 瞳孔放大，嘴微张未合拢 |
| 恐惧 | 瞳孔收缩，鼻翼微张，嘴唇微张却屏息 |
| 警觉 | 眉尾微挑，瞳孔微缩，鼻翼轻张 |
| 困惑不解 | 眉间紧锁，眼神迷茫，头部微侧 |

### G. 疲惫/不适类

| 情绪 | 微表情 |
|------|--------|
| 疲惫 | 眼皮微垂，眼神略带疲惫，黑眼圈隐隐可见 |
| 倦怠 | 面部肌肉松弛，眉间轻微舒展 |
| 生病 | 脸色苍白，嘴角下垂，眼神无光 |
| 困倦 | 眼皮沉重，头微微下垂，呼吸放缓 |

## 三、动作拆解法（核心技能·全网调研精华）

### 3.1 为什么不能写情绪词

AI对抽象情绪词的响应是放大+脸谱化：
- 写"哈哈大笑" → AI生成制式化笑容，缺乏个性
- 写"悲伤" → AI生成哭泣，脸谱化、缺乏层次

**正确方法：把情绪拆解成身体部位的连锁反应。**

### 3.2 动作拆解模板库

| 情绪/动作 | 拆解写法（身体部位连锁反应） |
|-----------|---------------------------|
| 哈哈大笑 | 身体前仰后合，嘴巴张大，笑到眼泪泌出，眼眶微红，耳朵泛红，手开始拍大腿，眼睛弯曲，肩膀随着笑声轻微抖动 |
| 慢慢露出笑容 | 听到对方说的话，先慢慢露出笑容，身体开始往后仰，头往上扬 |
| 哭泣 | 眼眶湿润→泪水涌出→嘴角抑制不住地抽动→肩膀起伏→手捂住脸 |
| 惊恐 | 身体本能后仰，瞳孔放大，喉结滚动，手微微抬起做防御姿势 |
| 愤怒 | 下颚收紧，咬肌鼓起，鼻翼微张，握住刀柄的指节发白 |
| 思考 | 视线望向远方的某处，眉间轻轻皱起，手指无意识轻敲桌面或下巴轻抬 |
| 害羞 | 视线微微向下移动，脸颊渐渐泛红，嘴唇轻轻抿起，手无意识绞着衣角 |
| 惊讶 | 眼睛微微睁大，眉毛上扬，嘴唇略微张开，头微微后仰 |
| 紧张 | 喉结滚动，嘴唇抿紧，眨眼频率加快，手指无意识握紧 |
| 冷漠 | 眼神平静，嘴唇自然闭合，面部肌肉完全放松，无任何表情变化 |

### 3.3 动作拆解三原则

1. **从外到内**：从最可见的（身体动作）写到最细微的（面部肌肉）
2. **有节奏有递进**：情绪有发展过程，不是瞬间完成
3. **留白给模型**：只描述物理状态，不写"结果表情"，让AI自己找到对应肌肉状态

### 3.4 示例对比

| ❌ 情绪词写法 | ✅ 动作拆解写法 |
|-------------|---------------|
| 秦牧看到三怪后非常惊恐 | 秦牧看清三怪后身体本能后仰半步，左手不自觉向前伸出做停止手势，右手刀垂下，刀尖微颤 |
| 晚晴开心地笑了 | 晚晴听到秦风的话，嘴角慢慢翘起，眉尾上扬，眼角弯弯——笑意从眼神漾开，低头看自己身上，手指轻捏衣料，眼波流转 |
| 秦牧愤怒地挥剑 | 秦牧下颚收紧，刀柄被握得指节发白，猛然转身，刀光划破空气 |
| 林莫若有所思 | 林莫视线望向远处，眉间轻蹙，下颌线紧绷，手指无意识轻叩膝盖 |

## 四、情境驱动法

### 不要堆面部肌肉清单 → 写情绪产生的"情境"

| ❌ 脸谱化堆砌 | ✅ 情境驱动 |
|-------------|-----------|
| 轻微皱眉+视线斜向左方+嘴唇微噘 | 想生气但又被可爱到消气 |
| 眼睛瞪大+嘴角下拉+眉心紧锁 | 不想认输但表情先输了 |
| 眼神凶狠+咬肌鼓起+鼻翼扩张 | 想骂人但词汇量跟不上 |
| 眉头紧皱+嘴唇抿紧+眼神闪烁 | 明知会被拒绝，还是忍不住开了口 |

**用法**：在画面内动描述中嵌入情境短语，让模型自行找到对应的面部肌肉状态。

## 五、对话场景三合一（动作+台词+表情）

### 🔴 铁律：入画角色都要有微表情

说话人和入画听者都不能"面无表情"。听者不写"面无表情"——写自然放松的面部状态。

| 角色 | 写法规则 |
|------|---------|
| 说话人 | 台词 + 面部微表情 + 身体微动作 |
| 入画听者 | 反应微表情 + 身体微动作 |
| 入画但不出声的 | 写"眼神平静，嘴唇自然闭合"（不死板状态） |

### 示例

```
（0-5秒）
过右肩拍晚晴——右手托起刀，暖光在刀刃折射到脸上游移。
嘴角慢慢翘起，眉尾上扬，眼角弯弯——笑意从眼神漾开。
低头看自己身上，手指轻捏衣料，眼波流转。
晚晴语气轻快：`比铠甲好看多了耶~~`

秦风右肩在画框左下虚焦——肩部微动（在身后听晚晴说话的自然反应）。
```

**关键**：
- 同一角色不同镜头换不同微表情部位——不经嘴角动眉尾，不每镜都"嘴角翘起"
- 情绪渐变："嘴角慢慢翘起" 不是 "嘴角一翘"

## 六、情绪→写法对照（非对话场景用身体）

| 角色情绪 | ❌ 夸张写法 | ✅ 克制写法 |
|---|---|---|
| 惊慌/害怕 | "眼睛瞪大，嘴角下拉，面如土色" | "身体本能后仰，手微微发颤" |
| 愤怒 | "咬牙切齿，眉头拧成一团" | "下颚收紧，握住刀柄的指节发白" |
| 紧张 | "额头冒汗，嘴唇发抖" | "喉结滚动了一下，刀尖轻轻晃动" |
| 坚定 | "眼神一狠，猛咬牙关" | "呼吸放缓，重心沉下" |
| 绝望 | "眼神空洞，面如死灰" | "肩膀塌下，刀尖滑落地面" |
| 从容 | "嘴角微扬，眼神淡然" | "站姿松而不懈，呼吸平稳" |

**一句话规则**：让观众从身体看出情绪，不要让AI画情绪给你看。

## 七、动作与台词同步（说话不串行）

### 核心原则
**人在说话的同时在做动作。没有"做完→停→开口"。**

| ❌ 串行写法 | ✅ 交叠写法 |
|---|---|
| "抬手→停→说话" | "边说边抬手，话落手到位" |
| "后仰半步→左手伸出→再开口" | "后仰同时左手已伸出，声音跟着身体动" |
| "火星窜动→熄灭→秦牧眼神变了" | "火星熄灭的同时秦牧喉结滚动" |
| "做完动作。他说：`台词`" | "边说边退。他说：`台词`——话到最后尾音已微颤" |

### 写法规则

1. **动作和台词写在一起，不分开**
2. **用"边说边""同时""话落时"连接**
3. **台词在句子中间穿行，不必放句首/句尾**
4. **话可以被打断或尾音变化，反映情绪状态**

## 八、语气管理（台词/声音）

### 五条铁律

| 铁律 | ❌ 违规 | ✅ 正确 |
|---|---|---|
| **1. 写物理状态，不写心理标签** | "声音发虚尾音上扬底气全无" | "声音很轻，尾音微扬" |
| **2. 一个词原则** | 叠3-4个声音描述 | 最多一个状态词 |
| **3. 肢体先于语气** | 先写声音再写动作 | 先写身体动作→再出声音 |
| **4. 语气描述≤6字** | "声音发虚尾音上扬如蚊子般" | "声音发虚" / "尾音上扬" |
| **5. 物理尺度>情绪尺度** | "充满绝望地说" | "声音很轻" / "喉头发紧" |

### 语气速查表

| 要表达的 | ❌ 太满 | ✅ 刚好 |
|---|---|---|
| 害怕/心虚 | "声音发虚尾音上扬底气全无" | "声音很轻" |
| 愤怒/爆发 | "怒吼咆哮声音炸裂" | "声音压低，尾音重" |
| 哽咽/难过 | "声音发颤尾音下沉带着哭腔" | "喉头发紧" |
| 冷漠/平淡 | "冰冷无情毫无波澜地说" | "声音很平" |
| 惊慌/急促 | "声音尖细颤抖语无伦次" | "尾音上扬" |
| 虚弱/受伤 | "气若游丝声音微弱几不可闻" | "声音很轻，断断续续" |
| 高兴 | "兴奋雀跃声音高八度" | "语气轻快" |
| 激动 | "声音颤抖激动万分" | "语气清脆上扬" |
| 撒娇 | "嗲声嗲气拖长音" | "语气拉长尾音" |

### Dexhunter官方参考格式
```
台词（角色A，哽咽）："你到底想骗我什么？"
台词（角色B，急切）："我没有骗你！"
```
一个状态词 + 台词内容。不多写。

### 阿里云百炼声音公式（补充）

人声 = 说话内容 + 情绪 + 语调 + 语速 + 音色 + 口音

在分镜中写声音时，抓住核心物理特征即可：
- 情绪 → 语气的一个词（哽咽/轻快/发紧）
- 语调 → "尾音上扬" / "声音压低" / "断断续续"
- 语速 → 不单独写，通过语境传达

## 九、负面提示词（避免假人感）

> 来源：vocus/分析师大叔 + 全网调研

**Seedance生成时，以下词汇会让AI自动优化掉"假人感"：**

```
fake smile, stiff expression, dull eyes, overly exaggerated expression,
doll-like, 3D model-like, game character-like, cartoonish,
unrealistic, perfect symmetry, wooden expression, staring blankly,
artificial smile, plastic texture, mannequin-like
```

**应用方式**：在提示词末尾追加avoid约束行：
```
避免面部特征漂移，避免形态变形，避免动作僵硬不自然
避免假笑，避免表情僵硬，避免眼神呆滞
```

## 十、表演管理检查（写分镜前逐镜对照）

```
① 这一镜有情绪变化吗？→ 只写一种，不写"从X到Y"
② 有对话吗？→ 有→写微表情（说话人+入画听者）
③ 无对话？→ 写身体状态不写情绪标签
④ 角色的脸有没有参考图？→ 有→不写五官变化
⑤ 台词后面跟了几个声音描述？→ 超过1个→删到1个
⑥ 语气描述超过6个字了吗？→ 超了→删
⑦ 这一镜的表演和上一镜是同一个情绪层吗？→ 是→必须变（相邻不重复）
⑧ 动作和台词是串行还是交叠？→ 串行❌ 交叠✅
⑨ 入画听者有没有写微表情？→ 没写→补上
⑩ 同一角色连续镜头的微表情换了部位吗？→ 没换→换（不经嘴动眼，不经眉动鼻）
⑪ 有没有用动作拆解法替代情绪词？→ 情绪词❌ 身体部位连锁反应✅
⑫ 提示词末尾有没有追加避免假人感的avoid约束？→ 没加→补上
```

**全部通过才能写下一镜。**

## 十一、表演层次与镜头节奏（全网调研新增）

### 表演三层次（身体→微表情→台词）

同一镜头的表演从外到内有三个层次，写分镜时至少覆盖两个：

| 层次 | 内容 | 适用场景 |
|------|------|---------|
| **身体层** | 站姿/手势/重心/呼吸/肢体动作 | 战斗/动作/无对话 |
| **微表情层** | 面部肌肉状态/眼神/嘴角/眉间 | 对话/情绪交流 |
| **台词层** | 对话内容+语气+语速 | 有对话的所有场景 |

**规则**：有对话时三层都要写。无对话时只写身体层+微表情层。

### 情绪表演的节奏曲线

情绪不是平直的，是有弧线的：

```
[平静] → [微妙变化] → [情绪累积] → [高点] → [回落]
```

写一个15秒段落的表演时，情绪必须有起伏，不能从头到尾同一个状态。
