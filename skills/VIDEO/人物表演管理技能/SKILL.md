## 🔴 §语音层：语速/语气不由 Prompt 控制（2026-07-21 订正）

### 核心结论
**AI视频模型（Seedance 2.0 / Kling / Runway）的提示词不能直接控制语速、语调、说话停顿。**

### 原因：音视频分离架构
- Seedance 2.0 支持上传音频文件（MP3 ≤15秒，最多3条）作为参考
- 模型做的是"口型同步"——根据已有音频匹配嘴唇动作
- 语速/语调/停顿 → 全部由**音频文件本身**决定，不在 Prompt 的控制范围内
- 没有"字/秒"这个参数——模型读不懂人类的语速公式

### 正确工作流
```
TTS/录音 → 生成音频文件（控语速/语调/停顿） → 上传到 Seedance → 口型同步
```

**语音层（控语速/语调）：**
- ElevenLabs / Minimax TTS / 火山引擎语音合成 → 调语速参数 → 生成音频
- 自己录音 → 剪辑控制节奏 → 导出音频
- 上传到 Seedance → 作为 @audio 参考 → 模型自动做口型匹配

**视频层能做的（仅视觉暗示语气）：**
- ✅ 身体动作暗示：`秦一拳砸在桌上` → 暗示生气语气
- ✅ 呼吸幅度暗示：`胸膛起伏深吸一口气` → 暗示压抑语气
- ✅ 嘴型幅度暗示：`嘴角紧抿从牙缝挤出几个字` → 暗示克制语气
- ✅ 说法动词暗示：`喝道`/`低声道`/`脱口而出` → 暗示语气方向
- ❌ 写语速参数：`语速极快 5字/秒` → 模型不理解
- ❌ 写节奏指令：`慢条斯理地说` → 只能控视觉动作节奏，不能控语音速度

### ⚠️ 之前错误纠正
> **误以为通过 Prompt 可以控制 AI 角色说话的语速 → 实际不能。**
>
> 旧规则"语速由情绪决定：爆发4-5字/秒，哽咽2-3字/秒"——这是人类的语速规律，不是 AI 视频 prompt 能控制的参数。
>
> 正确理解：语速 = TTS/音频环节控制，视频 Prompt = 视觉画面 + 口型同步。

## 一、三条核心指令公式

### 公式① 强度降维——防「情绪过载」崩脸

> AI视频模型在帧间连续性计算中，每个大情绪词都会被逐帧强化→第3秒起崩脸。
> 解决：用微量修饰词替代大情绪词，人为降低AI输出幅度。

| ❌ 大情绪词（过载→崩） | ✅ 微量修饰词（克制→稳） |
|----------------------|------------------------|
| 哈哈大笑 / laughing loudly | 嘴角微微上扬，眼角轻弯 |
| 非常愤怒 / furious | 嘴唇抿紧，下巴微微收紧 |
| 悲伤哭泣 / crying sadly | 眼眶微微泛红，嘴唇轻颤 |
| 开心地笑 / smiling happily | 嘴角淡淡笑意，目光柔和，面部肌肉放松 |
| 冷漠无情 / cold emotionless | 嘴唇自然闭合，面部肌肉完全放松 |

### 公式② 动作驱动——身体先，表情是副产品

> 只写情绪词→AI凭空捏造面部运动→像提线木偶。
> 先写身体做什么→AI根据人体运动学顺带生成自然表情。

| 目标 | ❌ 只写表情 | ✅ 动作在前·表情在后 |
|------|----------|-------------------|
| 害羞 | 她羞涩地笑 | 她立刻低头避开视线，收下巴，轻咬下唇 |
| 警觉 | 他表情警觉 | 他身体纹丝不动，视线锁定目标，嘴唇抿紧 |
| 愤怒 | 他咬牙切齿 | 他握剑的手紧了又松，下巴咬紧，脖子青筋微现 |
| 难过 | 他眼眶红了 | 他低下头，手无意识摩挲剑柄，喉结滚动了一下 |

### 公式③ 时间序列——视频是时间的艺术

> 静态描述"他很伤心"→5秒表情锁死。视频需要变化。

```
起始状态 → 触发动作 → 最终状态

→ 他保持严肃的表情，目光直视前方。
  他缓缓闭上眼，深深吸了一口气。
  一个淡淡的、释然的微笑慢慢浮上嘴角。
```

## 二、情绪四象→面部肌肉翻译

> 来源：tahou.com·Seedance 2.0微表情控制教程
> Seedance对抽象情绪词（开心/难过）没有直接响应能力，必须翻译成AI可识别的肌肉动作+五官状态。

| 情绪 | 肌肉动作翻译（直接嵌入提示词） |
|------|-----------------------------|
| **喜** | 嘴角自然上扬，眼轮匝肌轻微收缩，下眼睑略隆起，眉心舒展 |
| **怒** | 眉头内侧下压并聚拢，鼻翼轻微扩张，下颌线绷紧，唇部轻闭呈直线 |
| **哀** | 内眼角下垂，上眼睑微遮瞳孔，嘴角两端缓慢下垂，呼吸节奏放缓 |
| **乐** | 口角大幅上提，颧骨明显隆起，眼角出现放射状鱼尾纹，牙齿部分露出 |

**用法**：取对应行的肌肉翻译词嵌入画面内容。
- ❌ `她的表情很愤怒` → AI不响应
- ✅ `眉头内侧下压并聚拢，下颌线绷紧，唇部轻闭呈直线`

## 三、表情指令库（7类35种·可直接嵌入画面内动）

### A. 克制笑（防过载）

|---|------|
| 1 | 嘴角微微上扬，眼角轻弯，目光柔和，面部肌肉放松 |
| 2 | 嘴角轻扬，眼神柔和 |
| 3 | 嘴角一侧微微扬起，眼尾微挑 |
| 4 | 嘴角微扬，眉尾轻挑 |
| 5 | 嘴角一侧撇起，下巴微抬（冷笑） |

### B. 收紧/警觉/克制

| # | 指令 |
|---|------|
| 6 | 嘴唇抿成一条线，下巴微微收紧 |
| 7 | 视线锁定[目标]，嘴唇抿紧，身体纹丝不动 |
| 8 | 下巴微微收紧，嘴角弧度消失 |
| 9 | 嘴唇自然闭合，面部肌肉完全放松（冷漠） |
| 10 | 目光定在[目标]上，嘴唇抿紧，头不转只动眼睛 |

### C. 紧张/压抑

| # | 指令 |
|---|------|
| 11 | 喉结滚动了一下，手指无意识蜷了一下 |
| 12 | 视线飘了一下又回来，手在袖口下攥紧 |
| 13 | 嘴唇抿成一条直线，握着[武器]的手微微收紧 |
| 14 | 咬肌微微鼓起，喉结滚动，眼睛没眨 |
| 15 | 呼吸压到很浅，嘴抿着，眼角微微泛红 |

### D. 悲伤/脆弱

| # | 指令 |
|---|------|
| 16 | 眼眶微微泛红，嘴唇轻轻颤动 |
| 17 | 喉结上下滚了一下，嘴唇分开一条缝又合上 |
| 18 | 嘴唇微微张开却说不出话，下巴轻轻抖动 |
| 19 | 眼眶泛红，嘴唇微微抿着，下巴内收 |

### E. 惊讶（短暂·不持续强化）

| 23 | 眼睛睁大了一瞬，嘴唇张开，手本能抬了一下 |

| # | 指令 |

| 26 | 嘴唇抿成白线，咬肌鼓起，声音从齿缝挤出 |
| 27 | 猛地抬头，眼睛瞪圆，嘴张开，喉部发力 |

### G. 思考/专注/迟疑

| # | 指令 |
|---|------|
| 28 | 视线望向远处，嘴唇微抿，手指无意识地轻点 |
| 29 | 眼睛微微眯起，嘴唇抿紧 |
| 30 | 嘴唇分开一条缝又合上，视线飘移了一瞬 |
| 31 | 目光放空，嘴唇微张，眼珠缓缓转动 |

| # | 指令 |
|---|------|
| 32 | 立刻低头避开视线，收下巴，轻咬下唇 |

## 四、语气指令（8种·动作驱动动词）

> 用物理动词描述发声器官状态。嵌入说法动词。

| 要表达 | 指令 |
|--------|------|
| 爆怒 | 吼道/暴喝 |
| 压抑发怒 | 低喝/沉声道/从齿缝挤出 |
| 冷峻 | 冷声道/平静吐出 |
| 哽咽 | 哽咽道/颤声说/喉头发紧 |
| 急切 | 急声道/脱口而出 |
| 轻声 | 低声道/耳语/几乎听不到 |
| 迟疑 | 迟疑地开口/把话吞了一半 |
| 平静 | 声音很平/声音很轻 |

用法：`秦冷声道："你还活着。"`

## 五、标点符号·AI语气控制

| 标点 | AI语音效果 | 用法 |
|------|-----------|------|
| 。 | 降调+停顿 | 陈述收束 |
| ， | 短停+抬升（话没完） | 连说 |
| …… | 声断气连·渐弱 | 犹豫/哽咽 |
| —— | 停顿+悬停 | 转折/被打断 |
| ！ | 音量↑+语速↑+强度↑ | 爆发/命令 |
| ？ | 末字自动上扬 | 疑问 |
| ~ | 尾音上扬+声线变软 | 调侃/撒娇 |

> ⚠️ 全角中文标点是唯一安全协议。

## 六、负面控制词（10种·Seedance验证有效）

> 来源：tahou.com + 刺猬星球

**段尾强制追加：**
```
避免面部特征漂移，避免形态变形，避免动作僵硬不自然
避免假笑，避免表情僵硬，避免眼神呆滞
避免塑料感，避免蜡像皮肤，避免木偶脸
```

**微表情专用负向词（需要精准控制表情时追加）：**
```
无面瘫、无面肌抽搐、无单侧嘴角上扬、无瞳孔失焦
无眨眼频率突变、无笑容渐隐、无表情断层、无笑容突然消失
```

## 七、情绪→电影术语速查（运镜/光影/色调匹配）

> 来源：ttfake92-lab·mx-shell-prompt（GitHub开源·MIT协议）

| 情绪 | 运镜 | 影调 |
|------|------|------|
| 害怕/恐惧 | 浅景深、手持晃动、快速推进 | 冷色调 |
| 惊讶/揭示 | 缓慢前推、移焦（前景→主体）、停顿 | — |
| 动作/追逐 | 甩镜、速度变化、低角度跟踪 | 高对比 |

| 紧张/悬疑 | 固定全景、沉默、缓慢变焦 | 暗调 |
| 孤独/沉思 | 远景留白、固定机位 | 自然光、低饱和 |
| 欢乐/活力 | 手持跟拍、中景 | 高饱和、明亮光线 |
| 史诗/壮丽 | 摇臂升降、远景、IMAX画幅 | 金色光线 |

## 八、AI视频台词嵌入格式（三种·按场景选）

### 格式A · 一行嵌入式（通用首选）
`[镜头]。[身体动作]，[说法动词]："[台词]"`

### 格式B · 动作流嵌入式（战斗/边说边做）
`[连续动作] → [角色]低喝"[台词]" → [声效]`

### 格式C · 分段式（情绪戏/重对话）
```
Scene (0–5s): [完整画面+情绪]
Dialogue (角色, 情绪): "台词"
```

**铁律**：禁"说"用动作动词·身体先于声音·标点全角·说话时不写嘴唇动作。

## 九、出稿前自检（5条）

```
① 用大情绪词了吗？（哈哈大笑/愤怒→换成微量修饰词或身体动作）
② 写了身体动作还是只写了脸？（害羞→先写低头，脸跟着来）
③ 情绪有变化吗？（Start→Transition→End，不是一个状态锁死5秒）
④ 语气用动作动词了吗？（喝道/冷声道/脱口而出，不是"说"）
⑤ 段尾负面提示词加了吗？
```
