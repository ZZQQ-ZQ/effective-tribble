## 一、ACT游戏引擎运镜哲学

> **核心认知转变**：电影运镜 = 导演架机器 → ACT运镜 = 游戏引擎自动响应角色行为。
> 镜头不再是"谁在哪拍什么"，而是"引擎如何响应角色此刻的战斗状态"。

### 1.1 四大引擎运镜系统

| 系统 | 英文 | 触发条件 | 镜头行为 |
|------|------|---------|---------|
| **锁定跟焦** | Lock-On Focus | 锁定敌人瞬间 | 镜头自动拉远框入双方，敌人始终在画面中心偏上 |
| **绕轴轨道** | Orbital Tracking | 锁定中角色绕敌移动 | 镜头以锁定点为轴心圆周运动，距离恒定 |
| **碰撞回避** | Collision Avoidance | 镜头后方有障碍 | 自动向前推近/上移/侧移，避免穿墙穿模 |
| **速度FOV** | Dynamic FoV | 角色速度突变 | 高速移动→FOV加宽(空间感)；慢动作→FOV收窄(聚焦感) |

### 1.2 五大ACT名作战斗运镜参考

#### 🔴 鬼泣（Devil May Cry）— Stylish Rank 运镜

| 评价等级 | 镜头行为 | Seedance转化 |
|---------|---------|-------------|
| D~C | 中景固定，平淡跟拍 | 中景固定 |
| B~A | 镜头接近+环绕加速，角度变低 | 近景环绕+低角度 |
| S~SSS | 广角+慢镜特写+镜头剧烈晃动+短暂定格 | 广角推近+慢镜CU+whip pan切 |

**核心借鉴**：镜头兴奋度与战斗评价同步升级——越打越炫，镜头越疯。

#### 🔴 猎天使魔女（Bayonetta）— Witch Time 魔女时间

| 阶段 | 镜头行为 | Seedance转化 |
|------|---------|-------------|
| 正常战斗 | 标准第三人称跟拍 | 跟拍 |
| 完美闪避触发 | 时间减速+镜头环绕半圈+画面色调改变 | 慢镜+环绕+色调偏蓝紫 |
| Witch Time中攻击 | 每一拳带慢镜+镜头微震+特效粒子定格悬浮 | 慢镜+微震屏+粒子悬浮 |

**核心借鉴**：闪避成功→世界减速→镜头环绕→主角在慢世界中流畅攻击——三段式动态节奏。

#### 🔴 战神（God of War 2018）— 一镜到底越肩

| 阶段 | 镜头行为 | Seedance转化 |
|------|---------|-------------|
| 探索/行走 | 越肩近距跟拍，角色占画面左1/3 | OTS跟拍 |
| 遇敌 | 镜头自动后拉框入全部敌人，FOV变宽 | 拉远+广角 |
| 单挑 | 镜头推近+微环绕，保持敌人始终可见 | 近景缓推+微环绕 |
| 处决 | 镜头逼近+低角度仰拍+慢镜 | CU低角度+慢镜+震屏 |

**核心借鉴**：不切镜的连续性——所有动作在一镜中完成，靠镜头推拉环完成景别切换。

#### 🔴 塞尔达传说（BotW/TotK）— Flurry Rush 子弹时间

| 阶段 | 镜头行为 | Seedance转化 |
|------|---------|-------------|
| 锁定对峙 | 标准Z锁定，角色与敌人在画面左右 | 固定中景，双方对位 |
| 完美闪避/盾反 | 时间减速+镜头拉近+画面侧倾(Dutch angle 5°) | 慢镜+近景+轻微荷兰角 |
| Flurry Rush连击 | 每击镜头微震+快速多角度短切 | 震屏+快切(0.5s/镜) |

**核心借鉴**：慢镜不是全局降速——是主角在慢世界中正常速度攻击，制造"超反应"感。

#### 🔴 只狼（Sekiro）— 拼刀弹反运镜

| 阶段 | 镜头行为 | Seedance转化 |
|------|---------|-------------|
| 拼刀连弹 | 镜头跟随刀刃碰撞点微震，双方在画面中心 | 中景固定+震屏同步 |
| 完美弹反 | 慢镜+火花粒子炸开+镜头后坐 | 慢镜CU+粒子爆发+震屏 |
| 忍杀处决 | 低角度仰拍+背景虚化+喷血特效 | CU低角度+浅景深+特效 |

**核心借鉴**：镜头震动与刀刃碰撞同步——每一次金属撞击都有画面反馈。

### 1.3 引擎运镜"铁三角"

```
角色动作 ──→ 镜头响应 ──→ 视觉反馈
   │              │              │
  攻击状态      景别/角度      震屏/慢镜
  移动速度      FOV/跟拍      拖尾/模糊
  锁定目标      环绕/拉远      框取/特写
```

**写分镜时的思维转换**：
- ❌ 旧思维：「这个镜头我从侧面拍秦挥刀」
- ✅ ACT思维：「秦挥刀→引擎响应：中景→近景跟推，碰撞点震屏，FOV收窄聚焦」

## 二、战斗运镜节奏系统

### 2.1 动态循环五阶段

```
[侦察] → [接敌] → [激战] → [重击] → [收招]
  ↓         ↓         ↓         ↓         ↓
3rd跟拍   拉远广角  快切跟拍  慢镜CU   缓推拉远
                         多彩运动   冲击帧    呼吸感
```

| 阶段 | 镜头模式 | 节奏 | 时长占比 |
|------|---------|------|---------|
| 侦察/巡逻 | 第三人称越肩跟拍，稳定 | 呼吸级 | 10% |
| 接敌/锁定 | 拉远框入双方，微环绕 | 警觉级 | 15% |
| 激战/连招 | 快切+跟拍+whip pan+震屏 | 冲击级 | 50% |
| 重击/处决 | 慢镜CU+低角度+定格 | 宣泄级 | 15% |
| 收招/喘息 | 缓推拉远，恢复呼吸感 | 回落级 | 10% |

### 2.2 冲击帧四层体系

> **冲击帧 = 打击感在镜头维度的表现。没有冲击帧的打击 = 纸片。**

| 层级 | 名称 | 视觉表现 | 时长 | 适用 |
|------|------|---------|------|------|
| L1 | 微震 | 画面轻微抖动(±2px)，瞬发瞬收 | 0.1-0.2s | 普通拳脚、轻击 |
| L2 | 震屏 | 画面明显震动(±5px)，持续0.3s | 0.3s | 重拳、兵器碰撞 |
| L3 | 顿帧 | 画面冻结+慢镜展开+震屏叠加 | 0.5-1s | 要害命中、暴击 |
| L4 | 后坐力 | 镜头向后弹+震动+短暂黑边挤压 | 1-1.5s | 终结技、大招释放 |

**使用规则**：
- 连招阶段 L1密集（每秒1-2次微震）
- 重击节点 插L3（全段1-2次顿帧）
- 终结技能 用L4（每段仅1次，压轴）
- L2为日常打击默认级

### 2.3 速度斜坡（Speed Ramping）

```
正常速度 ──→ 慢动作(30%速度) ──→ 正常速度
  连招          碰撞瞬间         收招
```

- 碰撞前0.2s开始减速
- 碰撞瞬间最低速（20-30%）
- 碰撞后0.3s加速回正常
- 一段15秒中，速度斜坡不超过2次（一次最大冲击+一次收招定帧）

### 2.4 Whip Pan 甩镜转场

| 类型 | 方向 | 适用 | 关键参数 |
|------|------|------|---------|
| 水平甩 | 左→右 / 右→左 | 视线快速转移、换目标 | 甩速:0.2-0.3s完成 |
| 垂直甩 | 上→下 / 下→上 | 飞起/落地、高低差 | 甩速:0.3-0.4s |
| 斜向甩 | 对角线 | 出其不意的攻击方向 | 不可预测感 |
| 连环甩 | A甩→B甩→C（≤3次） | 一挑多快速清敌 | 每个0.15-0.2s |

**Seedance提醒**：甩镜模糊段不要超过总数10%时长，模糊太多=视觉疲劳。

### 2.5 镜头后坐力（Camera Recoil）

> 每次大招/终结，镜头不是「看着发招」，而是「被冲击波推了一把」。

- **方向**：冲击波向外 → 镜向后弹（沿冲击反方向）
- **幅度**：波及半径越大，后坐幅度越大
- **恢复**：后坐后0.5s缓慢回位（不是瞬弹回来）

## 三、电影打斗摄影六大技法

### 3.1 手持晃动 — 隐藏"假打"

| 场景 | 晃动幅度 | 频率 |
|------|---------|------|
| 对峙/蓄力 | 微小浮动，呼吸感 | 慢 |
| 连招交火 | 中等晃动，跟上动作 | 中 |
| 被击中/摔倒 | 大幅晃动+短暂失焦 | 快→突变 |
| POV/FPV | 剧烈晃动+超广角畸变 | 极快 |

**Seedance写法**：
- 轻晃：`subtle handheld micro-float, breathing camera`
- 中晃：`handheld shake matching combat rhythm`
- 重晃：`violent handheld camera shake, near motion blur`

### 3.2 多角度切换 — 一镜多机

同一动作段内快速切换2-4个角度：
```
正面对峙 → 侧面跟拍 → 背后POV → 低角度仰拍
  MS        MS-tracking    POV        CU低角
```

切换规则：相邻两镜角度差≥45°，否则看起来像跳帧。

### 3.3 Cutting on Action — 动作掩剪辑

剪辑点藏在动作中段：
- ✅ 在拳挥到一半时切 → 观众注意力在拳头上，察觉不到剪辑
- ❌ 拳打完收回来再切 → 动作停了，剪辑点暴露

### 3.4 慢动作冲击点 — 每段2次硬限

碰撞/重击瞬间用慢镜：
- ✅ 刀刃相撞火花炸开 → 慢镜0.5s → 正常速度
- ✅ 被击飞空中 → 慢镜1s → 落地正常速度
- ❌ 连招过程中每次碰撞都用慢镜 → 节奏坍塌

### 3.5 十字景框 — 快剪不乱

快速剪辑时主体始终在画面中心：
- 减少观众"找焦点"的认知负担
- 每镜主体偏移不超过画面宽度的15%
- 特例：有意识打破中心（如角色被击飞偏离中心）

### 3.6 脏镜（Dirty Frame）— 前景遮挡

物体/人物局部在镜头极近处虚焦遮挡：
- 过肩镜头 = 脏镜的经典应用
- 尘土/火花/碎片飞向镜头 → 短暂遮挡 → 露出下一画面
- 每段建议1-2处脏镜过渡

## 四、Seedance 2.0 动态运镜实战

### 4.1 Orb模式 — 第一人称持续镜头

> Seedance特有的连续POV格式：相机=角色眼睛，15秒不间断。

**格式特征**：
- `"Single continuous shot, first-person POV, the camera IS her eyes"`
- `"hyper-chaotic handheld motion, completely unstabilized"`
- `"wide-angle lens (strong distortion)"`
- `"her hands always visible in frame"`

**适用**：沉浸式主观战斗、闪避+反击、速度感追逐

### 4.2 Timeline Prompting — 分时段运镜

```markdown
0-3s:  第三人称越肩, 跟拍秦从画面左入画, 中景, 发现敌人
3-5s:  镜头自动后拉, 广角框入三怪, 环绕缓推
5-8s:  POV秦视角, 手持剧烈晃动, 冲向正中怪
8-10s: 侧面跟拍, 秦挥刀弧线, whip pan切至碰撞特写
10-12s: CU特写, 刀刃击中, 慢镜0.5s+震屏L3
12-15s: 拉远全景, 秦收刀, 怪化作光粒
```

### 4.3 Multi-shot Montage — 多镜蒙太奇

```markdown
Montage, multi-shot action, don't use one camera angle or single cut
```
→ Seedance会自动生成多角度多镜头的战斗片段

### 4.4 运镜提示词组合模板

```
Camera:  [运镜类型+幅度+持续时长]
Action:  [角色动作]
Impact:  [碰撞点镜头响应:震屏/定格/后坐]
Rhythm:  [速度斜坡参数]
```

示例：
```
Camera: handheld tracking, tight follow, medium shake
Action: 秦侧身从三怪中间滑步穿过，刀锋横向划出弧线
Impact: whip pan to collision CU, L3 hitstop 0.5s, screen shake
Rhythm: normal→slow-mo 30% on impact→normal recovery
```

## 五、运镜搭配矩阵

### 5.1 场景→运镜速查

| 场景类型 | 主运镜 | 辅运镜 | 禁止 |
|---------|--------|--------|------|
| 1v1单挑 | 越肩+环绕 | POV冲击帧 | 远景全程 |
| 1v多群战 | 广角跟拍+快切 | whip pan转场 | 单一角度固定 |
| 双人配合 | 侧面跟拍双人 | 交替POV | 景别不变 |
| 追逐战 | FPV+跟拍 | POV回头 | 正面固定 |
| 大招/终结 | 低角慢镜推近 | 环绕蓄力 | 远全景收尾 |
| 潜行/暗杀 | 越肩静跟 | POV出手 | 快切 |
| 对峙/谈判 | 中景半环绕 | 特写眼神 | 剧烈晃动 |
| Boss战 | 低角广角+震屏 | 环绕慢镜 | 中景平淡 |

### 5.2 运镜冲突表（避雷）

| 运镜A | 运镜B | 结果 | 可以同时？ |
|-------|-------|------|----------|
| POV晃动 | 震屏L3 | 画面完全看不清 | ❌ |
| whip pan | 慢镜 | 模糊段拉太长 | ❌ |
| 环绕 | 跟拍 | 冲突（一个绕一个跟） | ❌ |
| 固定 | 手持晃 | 风格矛盾 | ❌（打斗禁用固定） |
| 推近 | 震屏 | 推进力+震动力叠加 | ✅ 但需分时段 |
| 环绕+缓推 | 慢镜 | 环绕蓄力+慢镜释放 | ✅ 经典组合 |

## 六、写分镜前运镜决策流程

### 6.1 五问决策

```
Q1 这段是几人战斗？（1v1 / 1v多 / 多人混战）
Q2 这段的情绪曲线？（蓄力→爆发 / 碾压→终结 / 被压→反击）
Q3 谁是这段的"主角"？（秦主视角 / 晚晴主视角 / 双人均衡）
Q4 高潮点在哪一秒？（以此为中心设计冲击帧节奏）
Q5 这段在整体中的位置？（开场/中段/高潮/收尾 → 运镜强度递增/递减）
```

### 6.2 运镜强度曲线

```
开场段：L1微震为主，跟拍流畅，建立空间
  ↓
中段：L2震屏引入，快切+whip pan，节奏加速
  ↓
高潮：L3顿帧+L4后坐力，慢镜+低角度，冲击最大化
  ↓
收尾：回到L1，缓推拉远，呼吸回落
```

### 6.3 运镜词汇选择速查

| 强度 | 英文关键词 | Seedance提示词用 |
|------|-----------|-----------------|
| 轻 | subtle / gentle / smooth / floating | subtle handheld float |
| 中 | dynamic / tracking / responsive / active | dynamic tracking follow |
| 强 | aggressive / violent / chaotic / explosive | violent handheld shake |
| 极 | hyper-chaotic / unstabilized / raw | hyper-chaotic raw motion |

## 七、运镜自检清单

出战斗分镜前逐条勾：

### 镜头响应
- [ ] 每个攻击动作是否有对应的镜头响应（震屏/推近/环绕）？
- [ ] 运动镜头≠固定镜头？（打斗全程无固定镜头）
- [ ] 每种运镜是否在3秒内≤2种？

### 冲击帧
- [ ] 重击/碰撞节点是否标注了震屏层级(L1-L4)？
- [ ] 碰撞瞬间是否规划了慢镜/顿帧？
- [ ] 速度斜坡是否控制在每段≤2次？
- [ ] 终结技是否用了L4后坐力？

### 运镜多样
- [ ] 全段运镜类型是否≥3种？
- [ ] 角度切换是否≥45°差？
- [ ] 是否有至少1处POV/FPV主观镜头？
- [ ] 是否有至少1处whip pan转场？

### 节奏曲线
- [ ] 运镜强度是否按"起→升→爆→落"曲线？
- [ ] 每镜时长是否有变化（不全是同一长度）？
- [ ] 慢镜段落是否穿插在快节奏中（不是全慢或全快）？

### ACT感
- [ ] 镜头是否在"响应角色"而不是"静态观看"？
- [ ] 锁定/接敌时镜头是否自动拉远框入双方？
- [ ] 是否避免了"导演架机器"的旁观感？

## 八、关联技能

| 技能 | 关系 | 调用时机 |
