## 🔴 核心认知

**块1三变量不是标签，是执行指令。**

```
错误：不要字幕。德国表现主义，高反差硬光，黑白质感... ← 标签堆砌
正确：不要字幕。德国表现主义扭曲空间，几何形硬光切割，高反差黑白阴影图形化... ← 模型可执行
```

每个风格 -> 四个执行层面：
1. **风格标签** — 叫什么（模型需要知道是哪种风格）
2. **光线特征** — 光怎么打（方向+光质+色温的具体描述，不是标签）
3. **质感特征** — 画面什么感觉（色彩+材质+介质的具体描述）
4. **摄影机语言** — 怎么拍（构图/运镜/焦段/角度偏好）

## 七、执行层：怎么让模型拍出这个风格

### 原则
**风格不靠炸关键词，靠：摄影机语言 + 光线方案 + 构图偏好三个维度锁定。**

同一个15秒片段：
- 库布里克拍 = 正面单点透视+均匀光+广角畸变+极慢推轨

## 十六、风格→AI视频渗透全流程（🔴 核心方法论）

> 不是只在块1写一个风格名。风格必须渗透到提示词的每一层——从块1到最后一个分镜。

### 渗透五层结构

```
              ┌─────────────┐
    层1 块1   │ 风格标签+光线+质感   │ ← 全局声明
              ├─────────────┤
    层2 画面  │ 从风格提取的一句话    │ ← 15秒视觉底色
       基调   │ 画面感觉              │
              ├─────────────┤
    层3 色彩  │ 从风格提取的          │ ← 技术执行
       影调   │ 主色调+补色+反差       │
              ├─────────────┤
    层4 主体  │ 从风格提取的          │ ← 谁+在哪
              │ 空间关系+材质方向     │
              ├─────────────┤
    层5 分镜  │ 逐镜注入风格摄影机语言│ ← 每镜带风格
              │ 构图+运镜+景别       │
              └─────────────┘
```

**第四步：自检——每个分镜有没有徐克味？**

```

镜2 ✅ 侧逆硬光+光束穿透 = 徐克标志
镜3 ✅ 快速横摇+碎片飞散 = 徐克痕迹
镜4 ✅ 后拉展示巨型空间 = 徐克常用
```

### 风格渗透铁律

1. **块1不是口号，是指令** — `徐克武侠`后面跟的不是形容词，是侧逆硬光/横摇/广角畸变这些模型能辨认的动作级指令
2. **画面基调和色彩影调是翻译层** — 把风格的光线特征翻译成当前场景的具体描述（蓝雾→火山灰，金色逆光→岩浆自发光）
3. **摄影机语言不进块1，进分镜** — 低角度仰拍/横摇/广角畸变写在每镜的运镜和构图中
4. **逐镜过风格味** — 每写完一镜问：这一镜的构图/运镜/光线是不是这个导演/电影会用的？
5. **场景不同，同样的风格落到不同光线** — 王家卫拍夜景=霓虹雨夜，王家卫拍白天=胶片褪色暖调

### 维度→提示词位置映射

| 风格维度 | 提示词位置 | 怎么写 |
|---------|----------|------|
| **色调** | 色彩与影调行 | 主调+补色+反差（如"青蓝灰主调，暖橙岩浆补色，高反差"） |
| **光线** | 块1（光线特征） | 光的方向+光质+特效光（如"侧逆硬光+烟尘光束穿透"） |

| **打光构图** | 分镜 每镜内容 | 每镜的构图方式+景别+机位（如"低角度仰拍广角畸变"） |

### 🔴 色彩与影调 ←→ 真实感增强 联动锁

> 色彩影调行不是独立调色——写它的同时真实感技能五个维度必须响。

| 真实感维度 | 色彩影调行怎么写 |
|-----------|---------------|
| **PBR材质反光** | 阴影区必须写材质反光补足，不写"暗部" |
| **大气透视** | 远景色彩必须衰减饱和度，不写"远处依然饱和" |
| **不完美色彩分布** | 色彩不能均匀铺色，写"自然光下色彩有微妙冷暖偏差异" |
| **菲涅尔效应** | 湿表面/金属高光区写菲涅尔渐弱，不写"均匀反光" |
| **材质区分** | 不同材质同色写差异："布料暗哑≠金属镜面反光≠皮肤半透明散射" |

**铁律**：写色彩与影调行 → 真实感增强的PBR/大气透视/不完美/菲涅尔/材质区分五个模块同时响。只写调色不写材质反光 = 色彩行不合格。

### 使用规则

1. **六维全取，缺一不可** — 选了风格就得六维都查，不能只改块1
2. **先翻译再写入** — 表格里的值是通用描述，需翻译成当前场景的语言（如"蓝雾"→"火山灰"）
3. **六维同时联动场景画质五件套** — 风格六维+场景材质+光效+色彩+真实感+环境生命力，全维同步
4. **环境生命力按场景取值** — 不从风格表取（表中不预定义），从环境生命力增强技能§四场景速配表取。同一导演的不同场景（室内/室外/夜景）环境运动不同
