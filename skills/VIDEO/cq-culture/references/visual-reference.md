# 传奇视觉风格与美术参考

## 原始端游视觉（2001）

### 技术规格
- 2D像素贴图，45度俯视角
- 分辨率：800×600
- 色调：大地色系+高饱和度点缀

### 各区域色调

| 区域 | 主色调 | 氛围词 |
|------|--------|--------|
| 比奇省 | 绿色+金色（森林草原） | 温暖、家园、希望 |
| 银杏山谷 | 金黄色（银杏树） | 宁静、怀旧、新手 |

### 经典UI元素
- **红蓝双球**：HP（红色）和MP（蓝色）以圆球形式显示在画面左上角——传奇最标志性的UI符号
- **背包格子系统**：装备和道具以方形图标排列，不同品级有不同边框色
- **聊天框**：左下角文字区域，世界频道/行会频道/私聊
- **小地图**：右上角圆形/方形小地图
- **安全区光圈**：盟重土城中蓝色光圈标记的安全区域

## 场景描述速查（供提示词使用）

### 比奇城（新手主城）
```
石头砌成的城墙环绕城池，护城河水清澈见底。
城门口几棵银杏树摇曳着金色叶片，阳光透过树叶洒在石板路上。
城内武器店铁砧叮当作响，药店飘出草药香气。
远处青山如黛，白云悠悠。
色调：暖金色 + 翠绿 + 石头灰
```

### 盟重土城（PVP中心）
```
土黄色城墙在烈日下泛着白光，风沙吹过空旷的广场。
几株枯树歪斜在城角，残垣断壁间散落着破碎的陶罐。
安全区地面投射淡蓝色光圈。
远处沙丘起伏如海，黄沙漫天。
色调：土黄 + 沙色 + 枯木棕 + 淡蓝（安全区）
```

### 沙巴克（荣耀之城）
```
巍峨的灰色石头城墙耸立在戈壁之上。
城头旗帜迎风猎猎，箭垛后有弓箭手巡逻。
金顶皇宫在阳光下闪耀。
城外战场满目疮痍——箭矢插满地、攻城器械残骸、焦黑弹坑。
色调：灰石 + 暗金 + 焦黑 + 旗帜红
```

### 祖玛寺庙（迷宫副本）
```
昏暗的石头走廊两侧刻满古老壁画和碑文，火把在墙上投下跳动的光影。
空气中弥漫着檀香和血腥味的混合气息。
祖玛阁——无数相同的石室循环往复，令人绝望的迷宫。
第七层：巨大的地下石厅，火盆燃烧，祖玛教主的石座在正中央。
色调：灰白石头 + 暗金壁画 + 火把橙红
```

### 石墓七层（猪洞终极）
```
地下溶洞，石笋如利齿般从天顶垂下。
石棺散落各处，白骨遍地。
火把摇曳的微光中，白野猪在石笋间缓缓踱步。
墙壁渗出水痕，地面有些地方积水反射火光。
色调：暗绿 + 黑褐 + 火把橙黄
```

### 赤月魔穴（终极恐惧）
```
阴暗的巨大地下空腔，墙壁上是蠕动着的血肉组织。
血管一样的管道从墙壁延伸出来，铺满地面。
中央是赤月恶魔——巨大的肉瘤嵌入地面，不可移动，触手般的地刺随时从地下涌出。
硫磺味弥漫。
色调：暗红 + 黑色 + 血色
```

## 特效视觉符号

### 爆装备
- **金光**：金色光柱从地面炸开，装备悬浮在光芒中缓缓旋转
- **紫光**：极品装备/套装掉落时的紫色光芒
- **光芒收敛**：装备被捡起→光芒收敛→化作光点散去

### 升级
- 角色身上爆出光环→"轰"的一声雷鸣→光柱冲天

### 技能特效

| 技能 | 视觉特效 |
|------|---------|
| 烈火剑法 | 剑身燃起烈焰→重斩→地面炸裂→火焰余烬 |
| 冰咆哮 | 暴风雪呼啸→冰晶旋转飞溅→结冰地面 |
| 雷电术 | 乌云聚顶→蓝白闪电劈下→地面焦黑 |
| 火墙 | 地面燃起2×2火焰方块→火焰持续跳动 |
| 魔法盾 | 金色/淡蓝半透明光球包裹角色全身 |
| 召唤神兽 | 符纸燃烧→金色召唤法阵→神兽从光中跃出 |
| 施毒术 | 挥手洒出粉末→目标冒红烟(降防)/绿烟(持续掉血) |
| 治愈术 | 金色光环围绕目标→伤口愈合 |

### PK击杀
- 角色倒下→灰色→装备散落地面→胜利者捡起

## 视觉叙事公式

传奇类视频短剧的视觉语言：

```
角色（战法道外观） + 武器（屠龙/裁决） + 场景（土城/沙城/祖玛） + 光效（金光/技能光） + 情绪（爆装喜/攻沙燃/兄弟暖）
```

### 传奇短剧常见视觉序列

1. **开场**：盟重土城大全景 → 角色背影走向远方 → 标题
2. **升级**：杀怪→光柱升天→装备切换→角色变强
3. **PK**：沙漠对决→技能互轰→爆装备金光
4. **攻沙**：行会集结→城墙冲锋→皇宫血战→旗帜
5. **回归**：中年玩家打开电脑→登录画面→熟悉的比奇BGM→泪目
