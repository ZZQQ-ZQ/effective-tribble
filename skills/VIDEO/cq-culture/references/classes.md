# 战法道三职业详解

## 战士

### 定位
近战物理爆发、高血量、高防御。冲锋陷阵的"坦克+输出"。

### 外观特征
- 重型铠甲（板甲/锁子甲），金属光泽
- 宽刃巨剑或大刀，武器体积大
- 体格魁梧健壮
- 披风飘扬
- 面部粗犷坚毅

### 核心技能

#### 基本剑术（7级）
普攻。刀身直接砍中，朴实金属碰撞，无特效。

#### 攻杀剑术（19级）
概率暴击。刀身瞬间闪白光，命中产生小型冲击波——比普攻亮、比刺杀弱。

#### 刺杀剑术（25级）
核心输出。隔空一格格位出刀，刀尖发出剑气贯穿目标，无视防御。刀不碰身，剑气隔空命中。视觉：白色剑光从刀尖脱刃而出，贯穿目标躯干。

#### 半月弯刀（28级）
群攻。刀从右向左横扫，拉出180°弧光，正面扇形范围内多个目标同时被扫中。

#### 野蛮冲撞（30级）
身体前倾肩甲在前，化为残影直线冲出3-5米，撞到目标→目标被定住倒退。

#### 烈火剑法（35级）
双手举刀过顶→刀身被火焰吞没→全力下劈→命中瞬间火焰炸开。双烈火：第一刀火焰未散第二刀紧跟劈在同一位置，两次爆炸叠加，0.5秒内秒满血。

#### 开天斩（38级）
刀举过头顶→刀身急速膨胀成银色巨刃→全力下劈→刀气化为光柱劈入地面→地面炸裂冲击波环形扩散。伤害高于烈火，攻击距离更长，25秒冷却。

#### 逐日剑法（47级）
剑气凝聚成形，瞬间化成一道金色人形光影——一个手持利刃的金色人影向前冲刺四格距离，直线上所有目标被贯穿。蓄力→人影脱体冲出→直线群体贯穿。破坏力极强。

#### 龙影剑法（48级）
高级战士群体对战技能。起手：战士握武器向前挥砍，身后拉出一道主银色剑光；剑光到达剑尖后分裂出4道独立剑气，呈扇形散开向前冲击，覆盖身前大范围区域，范围远大于半月弯刀。
剑气外形：四道剑光轮廓自带微弱龙形虚影，边缘有流动光雾，形似四条小龙同步向前穿刺（因此得名"龙影"）。复古版本为冷银白色剑光，高倍攻速服可改为金色/紫焰色特效版本。
动态层次：剑气飞出时带有拖尾光痕，击中目标炸开细碎光粒，整体有"破空撕裂空气"的动态质感，释放瞬间屏幕轻微震动强化打击感。
动画序列：起手挥砍（0.3秒）→ 主剑光拉出（0.2秒）→ 四道龙影剑气扇形分裂冲出（0.5秒）→ 击中炸光粒（0.3秒）→ 收刀（0.2秒）。

### 经典战斗画面
- 烈火剑法：剑身燃起烈焰→重斩落下→地面炸裂→火焰余烬
- 野蛮冲撞：压低重心→冲锋→肩膀撞飞敌人
- 刺杀剑术：精准的直线刺击，白色剑光一闪
- 开天斩：银色巨刃膨胀→劈落→光柱入地→冲击波扩散
- 逐日剑法：蓄力0.5秒→金色人影脱体冲刺→直线贯穿
- 龙影剑法：向前挥砍→身后拉出主剑光→剑尖分裂4道龙形剑气扇形冲击→击中炸开光粒→屏幕微震

### 武器进化路线
`木剑 → 短剑 → 青铜斧 → 八荒 → 炼狱 → 井中月 → 裁决之杖 → 屠龙刀`

## 法师

### 定位
远程魔法范围伤害、脆皮高输出。刷怪最快、最容易被秒。

### 外观特征
- 长袍（布甲/法袍），飘逸轻盈
- 手持法杖，周围有魔法元素光效
- 身材偏瘦修长
- 指尖有魔法光芒
- 魔法盾展开时身体包裹在半透明光球中

### 核心技能

| 技能 | 等级 | 效果 |
|------|------|------|
| 火球术 | 7 | 基础火球攻击 |
| 雷电术 | 17 | 从空中召唤雷电劈击 |
| 火墙 | 24 | 地面铺设火焰墙壁，持续灼烧 |
| 魔法盾 | 31 | **生存核心**——半透明光球护体，减免伤害 |
| 冰咆哮 | 35 | **终极技能**——暴风雪，大范围冰系伤害 |
| 流星火雨 | 47 | 天降火雨，超越冰咆哮的恐怖范围伤害 |

### 经典战斗画面
- 雷电术：乌云聚顶→闪电劈下→蓝白色闪光→地面焦黑
- 火墙：法杖一挥→地面燃起2×2火焰墙壁→火焰跳动
- 魔法盾：金色/蓝色半透明光球包裹全身
- 冰咆哮：法杖旋转→暴风雪呼啸→冰晶飞溅→敌人减速

### 武器进化路线
`木剑 → 海魂 → 偃月 → 魔杖 → 血饮 → 骨玉权杖 → 嗜魂法杖`

## 道士

### 定位
辅助治疗+召唤+持续毒伤。35级带狗翻身——从弱鸡变大佬。

### 外观特征
- 道袍（布甲/丝质长袍），飘逸仙气
- 拂尘或剑，符纸飘飞
- 身后跟随神兽（大型犬形灵兽/四足行走）
- 气质沉稳、仙风道骨
- 中毒的敌人身上冒绿烟

### 核心技能

| 技能 | 等级 | 效果 |
|------|------|------|
| 治愈术 | 7 | 单体治疗 |
| 精神力战法 | 9 | 提升闪避 |
| 施毒术 | 14 | 红毒降防/绿毒持续扣血 |
| 灵魂火符 | 18 | 符纸飞出击中目标 |
| 召唤骷髅 | 19 | 召唤骷髅战士 |
| 隐身术 | 20 | 隐身潜行 |
| 集体隐身术 | 23 | 群体隐身 |
| 群体治愈术 | 33 | 范围治疗 |
| 召唤神兽 | 35 | **终极技能**——召唤神兽（狗），道士翻身标志 |

### 经典战斗画面
- 召唤神兽：符纸燃烧→召唤法阵→神兽从光中出现→四足行走→战斗时站立吐息
- 施毒术：挥手洒出毒粉→目标身体上冒出红/绿色烟雾
- 灵魂火符：手指夹符→符纸燃起→飞出击中目标→爆出火花
- 治愈术：法杖轻点→金色光环围绕目标→伤口愈合

### 武器进化路线
`木剑 → 半月 → 降魔 → 银蛇 → 无极棍 → 龙纹剑 → 逍遥扇`

## 合击版本（后期）

合击版本中，玩家可带英雄，形成9种组合：

| 组合 | 合击技能 | 特点 |
|------|---------|------|
| 战战 | 破魂斩 | 双战士近战爆发，中后期无解 |
| 战法 | 雷霆一击 | 战士控场+法师输出 |
| 战道 | 劈星斩 | 战士输出+道士回血 |
| 法法 | 火龙气焰 | 双法师极致范围伤害 |
| 法道 | 末日审判 | 法师输出+道士辅助 |
| 道道 | 噬魂沼泽 | 双道士最强后期团战收割 |

## 职业间的经典克制

```
战士 > 法师（近身贴脸→秒）
法师 > 道士（远程风筝→压制）
道士 > 战士（放毒跑+神兽磨→耗死）
```
