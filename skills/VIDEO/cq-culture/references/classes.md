# 战法道三职业详解

## 战士

### 定位
近战物理爆发、高血量、高防御。冲锋陷阵的"坦克+输出"。

### 外观特征
- 重型铠甲（板甲/锁子甲），金属光泽
- 宽刃巨剑或大刀，武器体积大
- 体格魁梧健壮
- 披风飘扬
- 面部粗犷坚毅

### 核心技能

| 技能 | 等级 | 效果 |
|------|------|------|
| 基本剑术 | 7 | 提升命中率 |
| 攻杀剑术 | 19 | 概率附加额外伤害 |
| 刺杀剑术 | 25 | 无视防御的直线攻击，破法师魔法盾 |
| 半月弯刀 | 28 | 前方扇形范围攻击 |
| 野蛮冲撞 | 30 | 冲撞推开敌人 |
| 烈火剑法 | 35 | **终极技能**——蓄力后斩出烈焰重击，"双烈火"秒人 |

### 经典战斗画面
- 烈火剑法：剑身燃起烈焰→重斩落下→地面炸裂→火焰余烬
- 野蛮冲撞：压低重心→冲锋→肩膀撞飞敌人
- 刺杀剑术：精准的直线刺击，白光一闪

### 武器进化路线
`木剑 → 短剑 → 青铜斧 → 八荒 → 炼狱 → 井中月 → 裁决之杖 → 屠龙刀`

---

## 法师

### 定位
远程魔法范围伤害、脆皮高输出。刷怪最快、最容易被秒。

### 外观特征
- 长袍（布甲/法袍），飘逸轻盈
- 手持法杖，周围有魔法元素光效
- 身材偏瘦修长
- 指尖有魔法光芒
- 魔法盾展开时身体包裹在半透明光球中

### 核心技能

| 技能 | 等级 | 效果 |
|------|------|------|
| 火球术 | 7 | 基础火球攻击 |
| 雷电术 | 17 | 从空中召唤雷电劈击 |
| 火墙 | 24 | 地面铺设火焰墙壁，持续灼烧 |
| 魔法盾 | 31 | **生存核心**——半透明光球护体，减免伤害 |
| 冰咆哮 | 35 | **终极技能**——暴风雪，大范围冰系伤害 |
| 流星火雨 | 47 | 天降火雨，超越冰咆哮的恐怖范围伤害 |

### 经典战斗画面
- 雷电术：乌云聚顶→闪电劈下→蓝白色闪光→地面焦黑
- 火墙：法杖一挥→地面燃起2×2火焰墙壁→火焰跳动
- 魔法盾：金色/蓝色半透明光球包裹全身
- 冰咆哮：法杖旋转→暴风雪呼啸→冰晶飞溅→敌人减速

### 武器进化路线
`木剑 → 海魂 → 偃月 → 魔杖 → 血饮 → 骨玉权杖 → 嗜魂法杖`

---

## 道士

### 定位
辅助治疗+召唤+持续毒伤。35级带狗翻身——从弱鸡变大佬。

### 外观特征
- 道袍（布甲/丝质长袍），飘逸仙气
- 拂尘或剑，符纸飘飞
- 身后跟随神兽（大型犬形灵兽/四足行走）
- 气质沉稳、仙风道骨
- 中毒的敌人身上冒绿烟

### 核心技能

| 技能 | 等级 | 效果 |
|------|------|------|
| 治愈术 | 7 | 单体治疗 |
| 精神力战法 | 9 | 提升闪避 |
| 施毒术 | 14 | 红毒降防/绿毒持续扣血 |
| 灵魂火符 | 18 | 符纸飞出击中目标 |
| 召唤骷髅 | 19 | 召唤骷髅战士 |
| 隐身术 | 20 | 隐身潜行 |
| 集体隐身术 | 23 | 群体隐身 |
| 群体治愈术 | 33 | 范围治疗 |
| 召唤神兽 | 35 | **终极技能**——召唤神兽（狗），道士翻身标志 |

### 经典战斗画面
- 召唤神兽：符纸燃烧→召唤法阵→神兽从光中出现→四足行走→战斗时站立吐息
- 施毒术：挥手洒出毒粉→目标身体上冒出红/绿色烟雾
- 灵魂火符：手指夹符→符纸燃起→飞出击中目标→爆出火花
- 治愈术：法杖轻点→金色光环围绕目标→伤口愈合

### 武器进化路线
`木剑 → 半月 → 降魔 → 银蛇 → 无极棍 → 龙纹剑 → 逍遥扇`

## 合击版本（后期）

合击版本中，玩家可带英雄，形成9种组合：

| 组合 | 合击技能 | 特点 |
|------|---------|------|
| 战战 | 破魂斩 | 双战士近战爆发，中后期无解 |
| 战法 | 雷霆一击 | 战士控场+法师输出 |
| 战道 | 劈星斩 | 战士输出+道士回血 |
| 法法 | 火龙气焰 | 双法师极致范围伤害 |
| 法道 | 末日审判 | 法师输出+道士辅助 |
| 道道 | 噬魂沼泽 | 双道士最强后期团战收割 |

## 职业间的经典克制

```
战士 > 法师（近身贴脸→秒）
法师 > 道士（远程风筝→压制）
道士 > 战士（放毒跑+神兽磨→耗死）
```
