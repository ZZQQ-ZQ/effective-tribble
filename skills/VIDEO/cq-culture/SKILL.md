## 核心创作公式

传奇内容创作的情绪锚点：

```
兄弟情 + 爆装备金光 + 沙巴克战场 + 慢练级苦行 + 高风险PK = 传奇
```

**不可缺失的传奇DNA**：
- 装备掉落必须有"金光"（金色光柱从地面炸开）
- 屠龙刀是视觉图腾——宽刃黑色大刀，传奇符号
- 沙巴克=终极荣耀——城墙、皇宫、旗帜
- 红蓝双球UI是视觉烙印
- "兄弟"二字贯穿一切——无兄弟不传奇

## 传奇风格提示词关键元素

写传奇类视频提示词时，场景/角色/装备描写需锚定以下核心元素：

**场景环境**：
- 比奇城 → 石头城墙、护城河、银杏树、绿水青山、东方古城
- 盟重土城 → 沙漠土黄色、残垣断壁、风沙、安全区光圈
- 沙巴克 → 高大城墙、皇宫、攻城战、硝烟
- 祖玛寺庙 → 昏暗迷宫、石柱、壁画、火光
- 石墓猪洞 → 阴暗潮湿、石棺、火把

**角色外观**：
- 战士 → 重甲、宽刃大刀/巨剑、魁梧体型、披风
- 法师 → 长袍、法杖、魔法光效环绕、身形偏瘦
- 道士 → 道袍、拂尘/剑、身后跟着神兽（大型犬形灵兽）

**装备视觉**：
- 屠龙刀 → 宽大黑色刀身、暗金纹路
- 裁决之杖 → 黑色长柄、杖头沉重
- 天魔神甲/圣战宝甲 → 厚重铠甲、暗金色调
- 六大重装各有独特外观

**经典音效/氛围**：
- 比奇城BGM → 悠扬、温暖、回家的感觉
- 盟重BGM → 苍凉、史诗感
- 沙巴克BGM → 激昂、热血、战鼓
- 爆装备 → "叮"一声清脆 + 金光特效
- 升级 → "轰"一声雷鸣 + 光柱

## 传奇短剧创作经典母题

1. **逆袭叙事**：木剑→屠龙，从新手村到沙城城主
2. **兄弟情**：行会并肩作战→为兄弟两肋插刀
3. **爆装惊喜**：打怪爆出极品装备的狂喜时刻
4. **沙城争夺**：万人攻沙的史诗战斗
5. **怀旧回归**：中年玩家重回玛法大陆的感动
6. **江湖恩怨**：PK爆装备→结仇→复仇→和解
