# AI角色动作描写提示词手册

> 2026-06-25 入库。解决AI角色动作僵硬如"PPT"的问题——动作越具体、越有细节，角色越活。

## 黄金公式

```
动作提示词 = 动作动词 + 身体部位 + 节奏变化 + 细节反应
```

**三大原则**：
1. **多写动词**：比只写"站着"有效一万倍
2. **有顺序**：动作要有先后（先动脚再动手），不是同时
3. **加细节**：衣摆/头发/视线的物理反馈极大增强真实感

## 六种基础动作

### 1. 转身
- 公式：右脚后撤半步 → 肩膀先转 → 腰部带动身体回身 → 动作自然连贯
- 关键：强调身体部位联动顺序（肩→腰），不是简单"转身"
- 提示词：`右脚后撤半步，肩膀先转带动腰部回身，衣摆随转身弧线飘起`

### 2. 回头
- 公式：先停下脚步 → 脖子缓慢转动 → 视线最后回望 → 停顿感
- 关键：停顿感+视线引导=故事性。不是猛地扭头
- 提示词：`停住脚步，脖子缓慢转动，视线最后回望镜头方向，停顿一瞬`

### 3. 抬手
- 公式：手臂从身体一侧抬起 → 手肘微弯 → 手掌自然打开 → 轻柔克制
- 关键："手肘微弯""轻柔克制"避免机械感
- 提示词：`手臂从身侧抬起，手肘微弯，手掌自然打开，动作轻柔克制不僵硬`

### 4. 挥手
- 公式：右手抬到脸侧 → 小幅左右摆动两到三次 → 节奏轻快亲切
- 关键：指定摆动幅度和次数，控制节奏
- 提示词：`右手抬到脸侧，小幅左右摆动两到三次，节奏轻快`

### 5. 奔跑
- 公式：身体微前倾 → 双臂前后摆动 → 脚步快速向前 → 衣摆随动作晃动
- 关键：衣摆晃动等物理细节增强真实感
- 提示词：`身体微前倾，双臂前后摆动，脚步快速交替向前，衣摆和头发随跑动剧烈晃动`

### 6. 坐下
- 公式：膝盖慢慢弯曲 → 身体重心下沉 → 双手轻扶地面或椅边 → 坐下后肩膀放松
- 关键：重心变化+坐下后状态=松弛感
- 提示词：`膝盖缓慢弯曲，身体重心下沉，单手扶椅边，坐定后肩膀放松吐一口气`

## 在5层格式中的用法

基础动作融入【分镜序列】→内容中，用公式写法替代简单动词：

**❌ 旧写法**（模型不理解）：
```
内容：秦转身，看向画右
```

**✅ 新写法**（关节联动+物理反馈）：
```
内容：秦右脚后撤半步，肩膀先拧转带动腰部回身，铠甲下摆随转身弧线扬起，视线最后落定画右暗处
```
