# 人物打斗开场模版

> **定位**：15秒单人vs多人打斗开场·5镜渐进·踩脚起尘→升空→放招→清场
> **版本**：v1.0（2026-06-29 从战士vs兽人实战提炼）
> **适用场景**：主角面对众敌的霸气开场、战场冲锋序章、无双型清场

---

## 触发条件

- 单人vs多人战斗开场
- 导演需要15秒炫技型清场
- 主角从静立到爆发的过程展示

---

## 模版特征

| 特征 | 说明 |
|------|------|
| 镜数 | 5镜/15秒 |
| 打斗类型 | 单人主角 vs 多人围攻 |
| 节奏曲线 | 蓄力→亮相→爆发→冲锋→清场 |
| 运镜风格 | 跟拍上升+手摇+环绕（运动镜头全覆盖） |
| 能量体系 | 武器脱手升空→火焰首现→巨剑化→冲击波AOE |
| 核心创新 | 环境-光效-镜头三者联动 / 踩脚起尘入画 / 武器脱手螺旋火焰 |

```
→ 冲锋（8.5-10s）→ 横扫+冲击波（10-15s）
```

静帧在 8.3-8.5s 档口——接武器爆燃后的 0.2 秒静态定格，视觉记忆点。

## 五、核心设计原则

1. **冷启动**：武器初始无能量，镜3脱手瞬才炸。蓄力→爆发的反差。
2. **镜头驱动**：不旁观。踩脚震镜头、冲击波抖镜头、上升跟拍——镜头一直在被场景力驱动。
3. **三联动**：光穿过扬尘=体积光柱 / 火焰投射动态光影到地面和铠甲 / 冲击波扭曲镜头边缘。
4. **不收招**：冲击波推出画框即止。不写站定/pose/收刀。
5. **双色分光**：日光（金）+ 特效光（暖橙/冷蓝）在静帧粒子上一半一半。
