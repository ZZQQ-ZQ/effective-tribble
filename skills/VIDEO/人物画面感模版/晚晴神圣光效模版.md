# 晚晴神圣光效模版（段6实战验证·效果不错）

> 来源：段6 晚晴初登场 | 验证时间：2026-07-01 | 导演确认：画面感+光效都有了
> 用途：人物神圣感/逆光发丝发光/柔美光效的通用模版，可用于其他角色同类氛围

---

## 核心光效组合

```
丁达尔光束从人物身后斜穿
→ 发丝边缘发光形成柔光光环
→ 头发飘起时每根发丝被勾出金色轮廓光（像镀了一层柔光）
→ 面部柔光，肤色通透发光
→ 眼神光清晰
→ 背景完全虚焦融入暖绿色光斑
→ 光束中可见细小微粒悬浮飘动
```

## 镜头语言

| 元素 | 写法 |
|------|------|
| 景别 | CU（肩膀以上） |
| 镜头 | 呼吸手摇——画面随人物呼吸微微浮动 |
| 角度 | 正面，人物身后有光源 |
| 节奏 | 慢镜头感——发丝缓缓飘起和飘落 |

## 人物表演

- 不写情绪词，只写身体动作
- 转头→发丝飘起（身体动作驱动视觉）
- 歪头→嘴角微动（微小表演）
- 台词尾音上扬，声音轻而随意

## 分镜内光效写法的关键

1. **逆光方向明确**：丁达尔从右上方穿透
2. **发丝发光是核心记忆点**：不是"光效好"概括，是"每根发丝被勾出金色轮廓光"
3. **面部柔光不堆词**：一句"肤色通透发光"够了
4. **背景景深虚化要有颜色**：不是"背景虚化"，是"融入暖绿色光斑"
5. **呼吸手摇配光束粒子**：光束中可见细小微粒悬浮飘动——增加空气感和神圣感
6. **手摇=呼吸**：不是乱晃，是随人物呼吸微微浮动

## 适用场景

- 人物初登场（制造惊艳感）
- 神圣/高光时刻
- 救场角色出场
- 任何需要"发光的人"的氛围
- 🔴 **日常近景特写通用**：逆光发丝轮廓光 + 面部柔光通透 + 背景虚焦暖色光斑 + 光束微粒悬浮——不限神圣场景，近特CU/MCU日常对话也能用这种柔光层次（2026-07-01 导演指定）
