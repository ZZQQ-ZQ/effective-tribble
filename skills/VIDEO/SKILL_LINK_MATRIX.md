## 🔴🔴 技能生态原则（2026-07-24 导演指定·永久）

**技能不是独立的。它们是链。触碰一个，整条链响。**

### 五大级联链

```
【风格链】影视风格 ─→ 视觉光效 ─→ 色彩调色 ─→ 场景质感 ─→ 真实感 ─→ 构图 ─→ 运镜 ─→ 镜头逻辑性 ─→ 环境生命力
 触发词：王家卫/徐克/库布里克/某电影风格... → 整条链10个技能同时响

【打斗链】打斗设计 ─→ 动态运镜 ─→ 传奇打斗逻辑 ─→ 构图 ─→ 镜头逻辑性 ─→ 人物表演 ─→ 真实感 ─→ 场景一致性 ─→ 环境生命力
 触发词：打斗/战斗/攻击/对峙/碰撞/放技能... → 整条链9个技能同时响

【人物链】人物表演管理 ─→ 真实感 ─→ 场景一致性 ─→ 人物服装 ─→ 镜头逻辑性 ─→ 构图 ─→ 环境生命力
 触发词：对话/表情/情绪/台词/独白... → 整条链7个技能同时响

【场景链】场景质感 ─→ 视觉光效 ─→ 色彩调色 ─→ 真实感 ─→ 构图 ─→ 镜头逻辑性 ─→ 环境生命力
 触发词：场景/环境/地点/建筑/自然/室内... → 整条链7个技能同时响

【环境生命力链】🆕 环境生命力增强 ─→ 真实感 ─→ 场景一致性 ─→ 影视风格 ─→ 色彩调色 ─→ 视觉光效
 触发词：环境太死/画面不活/没有风... → 整条链6个技能同时响

【镜级执行链】构图Part 6速配表 → Part 0三坐标(Z/Y/X) → 景别(镜头逻辑性·可见范围) → 内容适配景别 → 摄影机(附录A2) → 焦段(附录A1·mm f/光圈) → 胶片(附录A3) → 拍摄方式(附录A4) → 面部判定(真实感·中近特触发) → 光位光质(视觉光效) → 运镜方式 → 空间站位(场景一致性) → 环境运动(环境生命力·每镜1-2个) → 负面提示词尾缀(真实感增强·段尾追加)
 触发词：每落一镜 → 整条链15个节点全响
```

### 🧬 任何技能被触发 → 沿链级联所有下游技能

```
例：导演说"徐克风格"
→ 影视风格技能触发
→ 沿风格链自动级联：
  视觉光效响 → 色彩调色响 → 场景质感响 → 真实感响 → 构图响 → 运镜响 → 镜头逻辑性响
→ 9个技能同时给出输出
→ 而不是只改块1三个词
```

**违反此原则 = 每次只响一个技能，其他技能沉默 = 提示词不完整。**

| 🔴 影视风格技能 | 块1三变量：影视风格标签→光线特征→质感特征·风格锚定 |
| 场景质感增强 | 材质纹理（建筑/自然/室内/地面按场景选模块） |
