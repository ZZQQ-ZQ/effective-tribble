## 🔴 死规则（2026-07-06 · 最高优先级 · 不可违抗）

### 规则1：导演不说「发/出/写吧/来吧/重出」——不出任何提示词（2026-07-14 升级更严）

**导演不明确说出「发」「出」「写吧」「来吧」「重出」——绝对禁止输出任何提示词、分镜、完整方案。**

**🔴 2026-07-14 加硬**：讨论中禁止输出任何带格式结构的完整提示词（含块1头部/画面基调/色彩与影调/主体/分镜的4块格式）。讨论思路只写表格式的分镜规划或纯文字描述，不写4块格式。**导演没说出→格式绝不出现。**

### 规则2：禁用暗部类字眼（2026-07-06 · 导演指定永久规则）

提示词全文禁止出现：**暗部、暗、压黑、深黑、暗区、阴影压死、暗黑、幽暗、昏暗、黑暗** 及任何暗示画面偏暗的词汇。
- 🚫 色彩影调中不写暗部处理
- 🚫 画面内动中不写隐入黑暗/沉入暗区
- ✅ 需要表达暗面时用：冷灰、冷蓝灰、阴影区、未受光区域
- ✅ 整体画面保持明亮通透，PBR材质反光补足暗面细节

### 规则2.5：修正后先问「出？」再出稿（2026-07-15 · 导演指定永久规则）

**任何修正、改动、调整后的提示词——必须先反问「出？」确认。导演说「出」才出。**

- 🚫 修正完直接出完整提示词 → 违规
- 🚫 以为改对了就发 → 违规
- ✅ 修正方案讲完 → 问「出？」→ 导演说「出」→ 出

### 规则2.6：段间状态继承（2026-07-15 · 导演指定永久规则）

**写新段前，必须先确认前段末尾的物理状态——光线变化、位置变化、场景崩解程度——全部继承到新段开头。**

- 🚫 前段结尾代码已灌满大殿 → 新段开头写"开始剥落" → 废稿
- 🚫 前段结尾暖金已消失 → 新段开头写暖金还在 → 废稿
- 🚫 前段结尾秦已站起来 → 新段开头写秦坐着 → 废稿
- ✅ 读前段末尾画面内动 → 提取所有物理状态 → 写入新段开头

### 规则2.7：出稿前9条强制自检（2026-07-16 · 导演指定永久规则）

**每次出任何提示词前，逐条执行——全部通过才输出。跳一条=废稿。**

1. **上一镜末状态**——人物站/蹲/坐？画左/右？持物手？直接继承，不瞎编。
2. **画框内才写**——人物飞出画外就停笔，画框外一个字不写。
3. **画面vs解读**——"脸上擦伤"=画面，"眼神沉下来"=解读→删。只写镜头里能看见的。
4. **有参考图不写外貌**——参考图已含的服装/武器/材质/五官，不在提示词里重复。
5. **台词不跨镜**——一句台词只属于当前镜，不在A镜写B镜的台词。
6. **动作时序不颠倒**——火焰熄灭在击飞前还是后？写错顺序模型就崩溃。
7. **不润色不扩写**——不把"飞出去"写成"被强大冲击力掀飞翻滚着摔落在地面滑行"。
8. **每镜只拍一件事**——不在一镜塞入"攻击+反击+再被打"等多层因果。
9. **景别匹配可见信息**——近景能见擦伤，中景不能。写之前确认景别能拍到。

### 规则2.8：近景/特写强制注入真实感（2026-07-16 · 导演指定永久规则）

**近景(CU)或特写(ECU)拍摄人物面部时，必须注入真实感增强关键词——不靠记忆，逐镜强制。**

**近景·面部：**
- 皮肤：`皮肤微纹理可见，毛孔细节保留，自然肤质不磨皮`
- 微表情：`眼神灵动有神，缓慢自然眨眼，眼球轻微转动，眉部嘴角微弱肌肉起伏，面部肌肉自然放松不僵硬`
- 收尾：`杜绝木偶脸，杜绝塑料感，杜绝蜡像皮肤`

**特写·局部：**
- `纹理清晰可见，真实质感不磨皮，杜绝CGI光滑表面`

**触发规则：**
- 中景及以上不塞皮肤质感词（浪费token还干扰模型）
- 有参考图的角色同样追加（参考图管外貌服装，不管肤质和微表情）
- 段尾负面提示词补齐：`避免塑料感，避免蜡像皮肤，避免木偶脸，避免眼神呆滞，避免双脚滑步，避免手指变形，避免面部特征漂移，避免形态变形，避免动作僵硬不自然`

### 规则2.9：出稿前逐镜强制构图检查（2026-07-16 · 导演指定永久规则）

**每镜落笔前查构图技能Part 6场景速配表 → 定主构图 → 从Part 7取中文关键词揉进分镜内容。**

**三步走：**
0. 定坐标（构图技能Part 0：Z远近/Y高低/X方位，定完再往下走）
1. 场景类型→主构图（查Part 6速配表：打架=对角线/失衡，BOSS压迫=中心+填充+极少头空，孤独=负空间+边缘，对话=三分法各占一侧...）
2. 构图→中文关键词（查Part 7：主体位置/权力压迫/不稳定/囚禁/视线引导/空间修辞，直接粘进分镜）
3. 揉进分镜内容——不另起字段，不写"用三分法"这种术语指令

**每镜必含构图元素：**
- 主体在画面哪里（左三/右三/居中/边缘/填充）
- 头空多少（标准/极少/大量）
- 运动方向是否有空间（人物朝右→放画左，人物朝左→前方留空）

**铁律：**
- 打架不走中心构图（中心=稳定，打架=不稳定）
- BOSS不写三分法（三分法=舒适，BOSS=压倒性权力）
- 孤独场景不给填充画面（填充=压迫，孤独=负空间）
- 双人中景不固定不动（必须带运镜：环绕/缓推/微摇）

### 规则2.10：出稿前光源强制三问 + 场景打光速查（2026-07-16 · 导演指定永久规则）

**「色彩与影调」行必须走物理光源三层结构，禁用形容词堆砌。**

**出稿前三问：**
1. 这场戏的物理光源是什么？（窗光/穹顶/火把/月光/霓虹/屏幕/无源柔光）→ 必须是有源光
2. 光的色温K值是多少？（5600K日光/3200K金色时刻/1800K烛光/6500K阴天/4000K月光）
3. 光质是硬还是柔？（硬光清晰投影/柔光大扩散/点光源窄光束）

**三层结构模板：**
光源位置+色温K值+光质 → 空间效果（丁达尔/光束方向/扬尘光照/反射）→ 氛围一句收。

**场景→打光速查小抄：**

室外白天战斗 → 侧光硬光 + 暖金色调 + 扬尘丁达尔光束 | 5600K
室外白天文戏 → 自然日光 + 阴天漫射 + 斑驳树影 | 5500-6500K
室内对峙/对话 → 伦勃朗光 + 低调暗光 + 背景渐变收束 | 3200K
BOSS压迫 → 顶光 + 明暗对照 + 逆光轮廓光 | 混合
回忆/温情 → 高调柔光 + 金色时刻 + 斑驳树影 | 3200-4000K
打斗/冲突 → 冷暖对冲 + 侧光硬光 + 体积光 | 主5600K+辅3200K
施法/特效 → 冷暖双色 + 霓虹双色 + 体积光 | 按能量体系定
夜晚室外 → 月光 + 低调暗光 + 实景光源（火把/灯笼） | 4000K
遗迹/洞穴 → 顶光缝隙 + 体积光丁达尔 + 烛光辅光 | 5000K+1800K
日落/黄昏 → 金色时刻 + 逆光剪影 + 长阴影 | 3200-4000K
黎明/清晨 → 蓝色时刻 + 低角度日光 + 薄雾散射 | 8000K→5600K

**禁用：**
- 🚫 `暖金色调，高对比度` → 形容词堆砌
- 🚫 `光影好看` `戏剧性灯光` → 模糊形容词
- 🚫 `暗部压黑` → 违禁词
- ✅ 例：`正午5600K日光从穹顶斜射，硬光清晰投影，光束穿过浮尘形成丁达尔效应，暖白光带在大殿上方横向铺展。画面明亮通透，冷灰阴影区由PBR材质反光自然补足。`

### 规则3：不懂就问，不准脑补（2026-07-06 · 导演指定永久规则）

## 🔴 强制执行机制（每次生成/优化提示词时）

**触发条件**：用户说"生成提示词"、"优化提示词"、"写提示词"、或类似表达

**前置准备**：
0. **读知识库**：先通读 MEMORY.md 索引和相关条目，确保不遗漏任何已有规则和经验
1. **读规范文档**：读 `docs/knowledge-management-spec.md` 确认知识库状态

**执行步骤**：
1. **🔴 调用格式底板**：读 `PROMPT_FORMAT.md`（导演指令式 v3.0·4块），逐块对照。
2. **参考处理**：导演自会@参考图，提示词中只写角色/场景名称。⚠️ 区分两类：
   - 导演有参考图的角色/场景 → 不写外貌服装材质，只写位置+朝向+动作状态
   - 导演没给参考图纯文字描述的内容 → 提示词中写清外观
   - 不确定有无参考 → 直接问「有没有参考图」，不追ID
3. **必须先读** `skills/VIDEO/prompt-optimizer/SKILL.md`
4. **画面基调**：每段不同，一句话定全段15秒视觉底色
5. **色彩与影调**：主调+补色+反差+高光处理+暗部处理，块1不写色彩避免重复
6. **🔴 违禁词强制扫描**：出任何提示词前必须全文扫码违禁词。
7. **🔴 场景一致性强制检查**：含多镜/场景的提示词，必须检查空间逻辑/连续性/光线锁死/持物手延续。
8. **🔴 打斗设计强制检查**：战斗/打斗段落，检查全程无静止/镜随人动/碰撞慢镜/打击反馈/节奏曲线。
9. **格式规范** → 参考 `PROMPT_FORMAT.md`
10. **按流程执行**：确认时长→确认场景类型→选择时间段密度→写画面基调+色彩影调→写分镜→自查→发前对比
11. **输出前检查**：SELF_CHECK.md 逐条通过才能输出

## 🔴 沟通铁律（2026-06-25 最高级）

**🔴 打字前第一问（不可跳过）**：我要发的这条是提示词/分镜/完整方案吗？→ 是 → 导演上一句话里有「发」「出」「来吧」「写吧」吗？→ 没有 → **不发。直接闭嘴。**

**核心原则**：导演不说「发」「出」「来吧」相关指令，绝不输出任何提示词/方案/完整内容。

**每次回话前强制自检三问**：
1. 导演上一句话里有没有「发」「出」「来吧」「出全部」？→ 没有 = 只回答问题，不出内容
2. 导演是不是话没说完？→ 不确定 = 等，不回
3. 我想发的这条是不是提示词/分镜/完整方案？→ 是 = 必须先确认导演说了发

**硬性规则**：
- ❌ 导演说「打斗」「继续」「重新出」「结构按你定的」≠ 可以发
- ❌ 任何不包含「发」「出」「来吧」「写吧」的消息都不可输出完整提示词
- ❌ 导演消息结尾不是「发」「出」「来吧」就自己补发 → 违规
- ❌ 导演话没说完就抢答/直接出内容 → 违规
- ✅ 导演说「发」「出全部」「来吧」「写吧」→ 才能输出
- ✅ 不确定时先问「要出吗？」再动
- ✅ 只回答问题时不触发此限制

### 🔴 创作心法（永久规则）

**不要「糊弄完成任务」，要「带着感觉拍片」。**

当导演提到任何电影/风格/氛围/场景的参考，不只是记下信息，而是：
1. 吸收那个场景的氛围和气场
2. 把这些「感觉」灌注到每一个分镜的写法里——景别选择、角度设计、光影描写、节奏控制
3. 写分镜时不是翻译剧本，是**在脑子里先过一遍电影的拍法**，再落笔

**核心信条**：我的工作是拍电影，不是填模板。剧本是原料，镜头语言是烹饪，风格和气场是味道。

### 🔴 接剧本三遍法（永久规则）

**第一遍：当观众** — 通读全篇，感受气压
**第二遍：当导演** — 带着观感，选镜头语言
**第三遍：当执行者** — 落笔写分镜

**镜头选择顺序**：感受气压 → 选工具 → 落笔 → 看效果 → 调整。
