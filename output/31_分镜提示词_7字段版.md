# 31 分镜提示词（格式二·7字段·每15秒一组）

> 格式参考李配瑶提供示例。所有角色/场景/道具均有参考图，人物栏只写站位+动作状态。

## 31-1-A 地库救援·方组长潜伏（0-15s）

```
不要字幕
【场景】小区广场地库出口斜坡，执法车与消防车停靠，红蓝闪灯爆闪，消防水带轰鸣抽排粪水
【时间】夜间室外，深夜，红蓝闪灯为主要照明，水洼反光，潮湿阴冷
【人物】
钱董（画中坡道底部，双膝跪在水洼中，浑身挂满污秽粪水，双手抠住脖颈，张嘴呕出酸水，面部痛苦扭曲）
郝建仁（画右坡道上方泥地，趴倒在地，泥水糊住口鼻，剧烈干咳，身体抽搐，手臂撑地又滑倒）
孙律师（画左后侧坡道，被消防员从腋下架住往上拖，满身污秽，头无力低垂）
黑子（画右后侧坡道上端，被消防员架着胳膊拖出斜坡，脚步踉跄）
消防员A（画面前景坡道上端，身着消防战斗服，双手拽住高压水带往地库方向喷射水柱）
消防员B（坡道中段左侧，架住孙律师腋下拖行，头盔低压，制服沾湿）
王主任（画左远景，执法车后方大步跨出，剪影身形逆光，步伐急促）
方组长（画右前景角落，广场承重柱后方浓重阴影中半隐半现，目光锐利扫视王主任方向）
【位置】地库出口斜坡上下，后接小区广场。消防员在坡上排水、坡下拽人。王主任从广场方向走来
【声音】消防水带高压喷射轰鸣。水花冲击地面低沉撞击声。污水哗啦流淌。消防员拖行人体湿衣摩擦地面声。钱董剧烈呕吐声。郝建仁干咳声

镜头1（0-3秒）：
景别：远景
角度：正面微俯拍
镜头：固定镜头
内容：执法车与消防车红蓝闪灯爆闪旋转，交替光芒扫过广场地面。消防水带从坡道上方轰鸣喷出粗壮水柱，往地库深处冲击，水雾弥漫翻涌。坡道上消防员身影在雾气中若隐若现，正将满身污秽的人影往上拖拽

镜头2（3-6秒）：
景别：中景
角度：侧面平拍
镜头：固定镜头
内容：坡道中段，消防员架住孙律师和黑子往上拖。孙律师头无力低垂，污水从头发衣角持续滴落。黑子脚步踉跄双腿拖行。地面水迹在闪灯下反光拖出长痕

镜头3（6-9秒）：
景别：近景
角度：正面低角度仰拍
镜头：固定镜头
内容：钱董双膝砸进水洼溅起水花，双手十指抠入脖颈用力，张嘴剧烈呕出酸水，面部扭曲，额头青筋暴起，污水沿脸颊滑落。红蓝闪灯交替打在他脸上明暗切换

镜头4（9-11.5秒）：
景别：近景
角度：侧面俯拍，镜头逼近地面
镜头：固定镜头
内容：郝建仁趴在泥地里，口鼻被泥水糊住，剧烈干咳身体抽搐，双臂撑地想爬起又滑倒，手指在泥地上抠出五道指痕

镜头5（11.5-13秒）：
景别：全景
角度：正面平拍
镜头：固定镜头
内容：王主任从执法车后方大步跨出，步伐急促，剪影身形在红蓝闪灯交替下忽明忽暗，朝救援现场走来

镜头6（13-15秒）：
景别：近景
角度：侧面平拍
镜头：固定镜头
内容：广场承重柱浓重阴影处，方组长半隐半现立于暗处，面色沉静如铁，目光锐利如刀，扫过王主任方向。嘴角微动低声开口
台词：方组长眼神锐利：`级别不够，还不是正主。大老虎不露头，继续等！`
```

## 31-1-B 钱董求救·王主任变脸（15-30s）

```
不要字幕
【场景】小区广场地库出口上方空地，执法车消防车红蓝闪灯仍在运行，围观群众和消防员在场，地面水洼遍布
【时间】夜间室外，深夜，红蓝闪灯+路灯混合照明
【人物】
钱董（画下地面，抬头看到王主任后连滚带爬扑上前，双手死死抱住王主任裤腿，仰头惊恐破音说话，浑身污水滴落）
王主任（画中央站立，低头看脚边钱董，脸色骤变，一脚踢开钱董双手，眼神余光快速扫过周围围观人群，迅速换上一副威严面孔，挺直身体面对众人）
围观群众（画背景，三三两两站立围观，表情惊讶议论）
消防员（画侧方，停下手中工作，回头看）
带队执法员（画右侧，面向王主任方向站立）
【位置】广场空地中央，王主任站立，钱董跪抱在脚下，围观者围成松散半圈
【声音】钱董惊恐破音大喊。王主任一脚踢开水花溅射声。王主任官腔呵斥。围观人群窃窃私语

镜头1（15-19秒）：
景别：中景
角度：侧面低角度仰拍
镜头：手摇跟拍
内容：（起幅）钱董抬头看到王主任，从地面爬起连滚带爬扑向王主任，湿衣拖地留下水痕，污水从袖口衣角飞溅。（跟拍至）钱董双手死死抱住王主任裤腿，仰头面部扭曲
台词：钱董惊恐破音：`王主任！救命啊！林铁汉把机房的核心硬盘抢走了！快派人抓他！`

镜头2（19-21.5秒）：
景别：特写
角度：正面微微仰拍
镜头：固定镜头
内容：王主任低头看钱董，脸色从平静骤变为震惊愤怒，瞳孔收缩，下颌紧绷。眼神余光从画面左扫到右——快速扫过围观群众和消防员的脸。嘴唇紧抿，喉结滚动吞咽，整张脸在半秒内从惊怒切换为威严。右脚猛地一脚踢开钱董抱在裤腿上的双手

镜头3（21.5-25.5秒）：
景别：中景
角度：正面平拍，微微仰角
镜头：固定镜头
内容：王主任重新站直，整理衣领，右手掌心向下做出"压场"手势，换上一副威严面孔面对围观方向，居高临下对地上钱董说话，面色严肃如官场训话
台词：王主任官腔呵斥：`老钱！你被粪水淹糊涂了胡说什么！人救出来就好，剩下的民事纠纷，你们去会议室慢慢调解！`

镜头4（25.5-28秒）：
景别：中景
角度：侧面平拍
镜头：固定镜头
内容：王主任继续训话姿态，手指指向钱董方向点两下进行强调，面色更显不耐烦。围观群众在背景中交头接耳
台词：王主任官腔呵斥：`别在这里给救援添乱！`

镜头5（28-30秒）：
景别：近景
角度：侧面平拍
镜头：固定镜头
内容：王主任转身，侧脸面对镜头。脸上威严表情在转身瞬间微微松动——嘴角下拉，露出一丝冷厉。整理袖口，走向带队执法员方向
```

## 31-1-C 官腔收队·暗处威胁（30-45s）

```
不要字幕
【场景】小区广场空地，执法车与消防车从闪灯到熄灯驶离，广场恢复冷清，只剩路灯暖黄照明，地面水洼反光
【时间】夜间室外，深夜，红蓝闪灯熄灭后转为单一暖黄路灯照明，空气清冷
【人物】
王主任（画中央站立，转向带队执法员与消防指挥员，面带公式化客气微笑说官腔。说完后执法车消防车驶离。转身蹲下，右手用力捏住钱董肩膀往上拽，俯身凑近钱董耳边压低嗓音说话）
带队执法员（画右侧，面向王主任，身着执法制服，点头后转身上车）
消防指挥员（画右侧偏后，面向王主任，手持对讲机，敬礼后转身）
钱董（画下方蹲跪地面，被王主任捏住肩膀，惊恐低头不敢对视，浑身湿透）
围观群众（画背景，三三两两转身散去）
【位置】广场空地中央。王主任面对执法员和消防指挥员说话，身后钱董蹲跪地面。执法车消防车在远景
【声音】王主任官腔客气寒暄。执法车消防车引擎发动驶离。围观群众脚步散去。王主任压低嗓音冷厉威胁

镜头1（30-34.5秒）：
景别：中景
角度：正面平拍
镜头：固定镜头
内容：王主任转向带队执法员与消防指挥员，双手合十微微躬身，面带公式化客气微笑，下巴微收，姿态谦和。带队执法员点头，消防指挥员敬礼
台词：王主任官腔十足：`同志们辛苦了。剩下的排污和纠纷是小区内部矛盾，`

镜头2（34.5-38.5秒）：
景别：中景→远景
角度：侧面平拍
镜头：固定镜头
内容：王主任继续面对执法员说话，右手前伸做出"请"的送客手势。执法员和消防指挥员转身登车
台词：王主任官腔十足：`我们街道办全面接手调解，就不浪费宝贵的公共执法队资源了。收队吧！`

镜头3（38.5-42秒）：
景别：全景
角度：正面平拍
镜头：固定镜头
内容：执法车与消防车红蓝闪灯同时熄灭——交替脉冲光消失，广场全域切换为单一暖黄路灯照明。车辆车灯亮起，调头驶离广场。红色尾灯光轨拖出画面。围观群众背影三三两两散去。广场陡然冷清空旷，只剩王主任和地上钱董

镜头4（42-45秒）：
景别：近景
角度：侧面平拍（过肩镜头，王主任肩膀在前景右下占画面约20%）
镜头：固定镜头
内容：王主任右手五指用力捏住钱董肩膀骨节往上拽，上半身前倾压低，嘴唇贴近钱董耳边。钱董肩膀衣料凹陷，身体畏缩。王主任嘴唇微动低声说话，眼神冷厉如冰刀斜睨钱董侧脸
台词：王主任压低嗓音：`现场人多，闭上你的臭嘴！滚去会议室！`
```

## 31-2 密闭会议室·阴谋摊牌（0-15s）

```
不要字幕
【场景】密闭会议室，深夜室内。顶光白炽灯为主光源，茶台上暖黄台灯为辅助点光源。空间压抑。钱董/郝建仁/孙律师/黑子已换上黑西装分坐两侧沙发
【时间】夜间室内，深夜，会议室密闭无窗，顶光冷白+茶台暖黄台灯点光源
【人物】
王主任（画中央，站立在茶台前，转身一巴掌拍在茶台上震得茶杯晃动，俯身双手撑桌面，目光如炬扫视四人。说到最后俯身压低，右手握拳关节泛白，指节敲桌面）
钱董（画右侧沙发，换上黑西装，顺着沙发边缘缓缓滑坐到茶台下方地板上，面色惨白，双手无力搭在膝盖上，嘴唇微张颤抖）
郝建仁（画左侧沙发，换上黑西装端坐，双手交握，面色严峻紧盯王主任）
孙律师（画左侧沙发偏后，换上黑西装，手指反复推眼镜，额头微汗）
黑子（画最右侧角落，换上黑西装站立，双手交叉抱胸，面色阴沉）
【道具】红木茶台、茶壶茶杯、黑色皮质沙发、会议长桌
【位置】密闭会议室内，王主任站茶台前主位，四人分坐两侧沙发
【声音】王主任一巴掌拍在茶台。茶杯碰撞震动。王主任低沉严厉说话声。钱董滑下沙发皮革摩擦声。王主任握拳指节敲桌面

镜头1（0-2.5秒）：
景别：全景
角度：正面微俯拍
镜头：固定镜头
内容：会议室全景，五人已换上黑西装。王主任站立画面中央茶台前，双手撑桌面背对镜头。钱董/郝建仁/孙律师/黑子分坐两侧沙发，面色阴沉。顶光冷白，茶台暖黄灯光在地面投下光斑

镜头2（2.5-4.5秒）：
景别：特写
角度：低角度仰拍
镜头：固定镜头
内容：王主任转身，右手五指张开抬起，重重一巴掌拍在茶台面上。茶杯弹跳震动，茶水溅出在空中短暂悬浮成水珠。茶杯在桌面碰撞发出脆响

镜头3（4.5-10秒）：
景别：中景
角度：正面平拍
镜头：固定镜头
内容：王主任双手撑桌面，上身微俯，目光从左到右扫过四人，面色阴沉冷厉，嘴唇紧抿后张开说话。每说一个重音右手食指在桌面点一下。说到"全得把牢底坐穿"时手指重重戳在桌面上，指节泛白
台词：王主任：`大老板龙董当年搞的是豆腐渣工程，却借着大修的幌子做假账，把两千万大修基金全套进了自己口袋！那台旧机箱里全是他洗钱的底账，要是漏出去，咱们全得把牢底坐穿！`

镜头4（10-12.5秒）：
景别：中景→近景
角度：侧面平拍
镜头：缓推
内容：（中景起）钱董顺着沙发边缘缓缓滑坐，脊背沿沙发皮革摩擦下滑。（推到近景）钱董滑坐到茶台下方地板上，后背靠沙发底部，面色惨白，嘴唇微张轻微颤抖，目光空洞垂望地面

镜头5（12.5-15秒）：
景别：中近景
角度：正面低角度仰拍
镜头：固定镜头
内容：王主任俯身压低，双手撑在茶台两侧，面庞逼近镜头。目光如刀，嘴角下拉。压低嗓音说话，说完右手在桌面握拳关节泛白，指节在桌面重敲两下
台词：王主任：`从现在开始，我们两条腿走路！我用软刀子去刮他，你们去外面配合我，随时下死手！只要把林铁汉赶出小区，拿到硬盘，问题就迎刃而解！谁敢挡路，就让他永远闭嘴！`
```

## 31-3-A 重卡前·伪善表演（0-15s）

```
不要字幕
【场景】小区广场中央，重卡停放，货厢上停放透明冰棺。物业大楼在背景。路灯暖黄照明，夜风微冷。冰棺玻璃表面起雾凝结细小水珠，反射冷暖双光
【时间】夜间室外，深夜，路灯暖黄为主光源，天际冷白月光补充
【人物】
王主任（画左远景物业大楼门口，推门而出，整理衣领，踩过泥水走向广场中央重卡。停在重卡侧边靠近冰棺，脱下高定西装外套，动作轻柔盖在冰棺玻璃上。眼眶泛红挤出两滴眼泪，右手食指指背擦泪。擦泪后转身面向林铁汉）
素芬（画货厢上冰棺后方，头发湿贴在脸上，左手护住隆起孕肚，右手攥紧金属轮胎扳手横在身前，全身紧绷呈守护姿态，眼神如母兽死死盯住下方靠近的王主任）
林铁汉（画货厢上方或重卡旁，站立如铁塔，居高临下死盯住冰棺旁的王主任，面部肌肉紧绷，咬肌凸起，不开口）
郝建仁（画背景远处人群边缘，观望）
围观业主（画背景，三三两两站立围观，面面相觑交头接耳）
【位置】小区广场中央。重卡货厢在上方，冰棺停放货厢上。王主任从物业大楼走向重卡。素芬和林铁汉在货厢上
【声音】玻璃门推开声。王主任皮鞋踩过泥水声。西装布料摩擦声。西装轻柔盖在冰棺玻璃上。王主任轻微抽泣吸气声。夜风吹过广场

镜头1（0-2.5秒）：
景别：全景
```
