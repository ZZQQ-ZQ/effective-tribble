## 门0：沟通铁律

  - 有 → 进门1。

## 门1：写作规范

### 结构
- [ ] 4块完整：头部→画面基调→色彩与影调→主体→分镜
- [ ] 块1格式：`不要字幕，不要背景音乐。[年代地域] + [光线风格] + [摄影质感]，4K`（三标签，不写色彩）
- [ ] 画面基调：一句话定全段15秒视觉底色，每段不同
- [ ] 色彩与影调：主调+补色+反差+高光处理+暗部处理
- [ ] 主体：分行列角色/场景/道具
- [ ] 分镜格式：`X-Y秒 景别，运镜。动作。光效。台词。` — 一行一镜
- [ ] 无旧格式残留（❌`景别：/角度：/镜头：/光效：` 等字段名）

### 光效
- [ ] 光效嵌入动作描述，不另起行
- [ ] 写成画面事件，不写概括词（✅"月光穿透扬尘形成光束" ❌"丁达尔光效"）

### 时间
- [ ] 文戏每镜3-5秒，打斗每镜1-2.5秒
- [ ] 台词秒数 = 字数÷语速 + 表演留白

### 台词
- [ ] 用反引号包：\`台词\`，带发言者
- [ ] 与原剧本一字不差

### 打斗
- [ ] 镜头是运动镜头（跟拍/手持/POV/环绕/甩镜/震屏）
- [ ] 动作连续不站桩，每镜有力度+速度
- [ ] 有来有回，对方也攻击

---

## 门2：跨镜检查

- [ ] 角色左右位置关系一致（不越轴）
- [ ] 动作承接上一镜结束状态（不跳步）
- [ ] 有@图引用的角色 → 不写外貌
- [ ] 只列实际出现角色

---

## 门3：总检

- [ ] 全文无违禁词
- [ ] 段头=动作进行中（非传送/降临/铺垫）
- [ ] 段尾=最后一击即结束（非站定/pose/定帧）
