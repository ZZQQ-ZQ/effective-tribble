## 门1：写作规范

### 结构
- [ ] 4块完整：头部→画面基调→色彩与影调→主体→分镜
- [ ] 块1格式：`不要字幕，不要背景音乐。[影视风格]，[该风格的光线特征]，[该风格的质感特征]，4K，超写实，UE5路径追踪渲染，Octane渲染，次世代PBR材质写实渲染。`
- [ ] 画面基调：一句话定全段15秒视觉底色，每段不同
- [ ] 色彩与影调：主调+补色+反差+高光处理+阴影区处理
- [ ] 主体：分行列角色/场景/道具
- [ ] 分镜格式：每镜拆为 `景别 / 构图 / 运镜手法 / 画面内动` 四字段
- [ ] 无旧格式残留（❌`光效：/角度：/镜头：` 等字段名）

### 光效
- [ ] 光效嵌入画面内动，不另起行
- [ ] 写成画面事件，不写概括词（✅"月光穿透扬尘形成光束" ❌"丁达尔光效"）

### 时间
- [ ] 文戏每镜3-5秒，打斗每镜1-2.5秒
- [ ] 台词秒数 = 字数÷语速 + 表演留白

### 台词
- [ ] 用反引号包：\`台词\`，带发言者
- [ ] 与原剧本一字不差

### 打斗
- [ ] 镜头是运动镜头（跟拍/手持/POV/环绕/甩镜/震屏）
- [ ] 动作连续不站桩，每镜有力度+速度
- [ ] 有来有回，对方也攻击

### 🔴 场景氛围（场景级必过 · 2026-07-24 v2）

- [ ] 空气 → 尘+灰+烟+热浪浑浊+水雾+火山灰。纯视觉浑浊介质，远景被微扭曲或朦胧。按场景类型替换空气介质（雪/雨/沙尘/雾气/扬尘/火山灰等）。🔥 写介质不写运动——运动归环境生命力。
- [ ] 环境生命力 🆕 → 画面里有东西在动。风→云飘→飞鸟→落叶→火星浮动→碎屑漂移→草动→水面波纹。每镜至少1个、最多2个环境运动元素。写因果链：环境力→反应物→视觉变化。🔴 联动环境生命力增强技能。
- [ ] 真实性 → PBR材质区分+大气透视+动作反馈+不完美细节+段尾负面提示词尾缀（真实感增强技能联动）

### 🔴 构图决策（写每镜前必过，不过=废镜）

```
[ ] 这镜的场景类型？→ 查下表定主构图
[ ] 主构图是否匹配场景情绪？→ 打架不走中心 / 孤独不填充画面 / BOSS不走三分法
[ ] 头空是否正确？→ 标准=掌控 / 极少=窒息 / 大量=渺小
[ ] 正负空间比例对吗？→ 压迫=正>>负 / 孤独=负>>正 / 冲突=正负对抗
[ ] 视觉重量分配对吗？→ 第一眼看到的是核心信息吗？
[ ] 如果多人 → Short-sided给了弱势方？/ 运动空间给了主动方？
```

**场景→主构图速查卡：**
| 单人情绪爆发 | 填充画面 | 追逐 | 运动空间+引导线 |
| 单人孤独/迷失 | 负空间+边缘 | 被困/囚禁 | 框架+极少头空 |
| 英雄登场 | 中心+引导线汇聚 | BOSS压迫 | 中心+填充+仰拍 |
| 双人平等对话 | 三分法各一侧 | 双人权力不等 | 对角线/失衡 |
| 对峙一触即发 | 对角线+Short-sided | 打斗/动作 | 对角线+运动空间 |
| 离别/告别 | 负空间 | 回忆/梦境 | 前景虚化+框架 |
| 史诗/宏大 | 对称+三层景深 | 希望/觉醒 | 运动空间+三分法 |

## 门3：总检

- [ ] 全文无违禁词
- [ ] 段头=动作进行中（非传送/降临/铺垫）
- [ ] 段尾=最后一击即结束（非站定/pose/定帧）

### 门4：人物表情语气弧线（2026-07-20 新增）

- [ ] 同一角色连续多镜→表情有过渡弧线（不是每镜贴独立标签）
- [ ] 每镜面部状态可以从前一镜自然推导（同一个部位/不同的幅度）
- [ ] 身体反应有时间差（皮肤先动→呼吸再变→眼球最后）
- [ ] 语气有延续或收束（不是镜镜换语气）
- [ ] 末镜表情没有回到跟首镜一模一样（弧线有变化）
