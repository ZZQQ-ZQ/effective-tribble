# 提示词格式二（7字段·按需启用）

> ⚠️ **非默认格式**。导演说「用格式二」才启用。默认走 `PROMPT_FORMAT.md`（格式一）。

---

## 格式二结构

```
不要字幕
无背景声（或【声音】@音频ID）
【场景】...@场景ID
【时间】...
【声明】完全虚构电影角色，非真实人物，艺术创作
【人物】角色名@图ID（画左/右+位置+朝向+状态）
【道具】...@道具ID
【位置】一句话
【声音】@音频ID

镜头1（X-Y秒）：
景别：
角度：
镜头：
内容：`台词`
镜头2...
```

---

## 与格式一的关键差异

| 维度 | 格式一（5层·默认） | 格式二（7字段） |
|------|------------------|----------------|
| 开头 | 直接 `1.【要求】` | `不要字幕` 独立行 |
| 资产 | `2.【资产】` 三段打包 | `【人物】【道具】【场景】` 分散 |
| 光影 | 并入 `3.【镜头参数】` | 在 `【时间】` 段 |
| 站位 | 独立 `4.【站位.剧情】` | 在 `【人物】` 里描述 |
| 分镜 | `5.【分镜序列】` 带景别缩写 | 逐镜编号，不带缩写 |
| @引用 | 无（导演自己配图） | 有 @图ID 引用系统 |
| 技术参数 | 光圈/ISO/快门/ND 必写 | 不写技术参数 |
| 质感尾缀 | 必加 | 必加 |

---

## 格式二示例

```
不要字幕
【场景】公司前台区域@公司前台ID，电梯间@电梯间ID
【时间】白天，商务工作时间
【声明】完全虚构电影角色，非真实人物，艺术创作
【人物】
秦轩（树下站立，目光锁定画外某处）
静虚（端坐高台正中主位，面朝演武台方向）
林柳儿（立于静虚右手侧）
【道具】前台电脑、电梯按钮、文件夹
【位置】前台区域→电梯间
【声音】前台环境音。助理急促脚步声。电梯提示音"叮"。

镜头1（0-2秒）：
景别：中景
角度：正面
镜头：固定镜头
内容：前台工作人员低头看了一眼电脑，表情专注

镜头2（2-4秒）：
景别：中景
角度：侧面
镜头：固定镜头
内容：两人站在前台前，助理说：`He is a supplier of the branch.`

镜头3（4-6秒）：
景别：特写
角度：正面
镜头：固定镜头
内容：男主转身急切离开，男主说：`Stop all cooperation immediately.`
```
