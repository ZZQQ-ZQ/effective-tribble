# 提示词格式三（打斗/动作·ACT视角）

> 🔴 打斗段落强制使用此格式。导演说「打斗」「战斗」「打」→ 自动切格式三。

---

## 结构

```
1.【要求】
实拍感，全画幅手持+稳定器混合，真人写实电影质感，超高精度细节，电影级柔光。UE5，Octane渲染，画质精致。
不要字幕，不要背景音乐。

2.【资产】

人物
角色名 （画左/中/右 + 武器）质感尾缀

道具
武器名 （光效：XXX）

场景
场景名 多视角
（天气：XXX。
光效：XXX。）

3.【镜头参数】
全程ACT视角，镜头同步跟随攻击方向，打击瞬间自带顿帧和镜头震动
光影：XXX

4.【站位.剧情】
剧情概述

5.【分镜序列】

（0-X秒）：画面描述 | XX说：`台词`

（X-Y秒）：画面描述 | ——

……

【声音】
人声.环境声.动作声.打斗声.

避免面部特征漂移，避免形态变形，避免动作僵硬不自然。
```

---

## 格式三关键规则

| # | 规则 |
|---|------|
| 1 | 分镜只写 `(秒数)：画面描述 | 台词`，不写景别/角度/镜头/光效 |
| 2 | ACT视角在 3.【镜头参数】全局声明：`全程ACT视角，镜头同步跟随攻击方向，打击瞬间自带顿帧和镜头震动` |
| 3 | 人物段只写【谁+在哪+武器】，表演全放分镜 |
| 4 | 道具只写光效，不写材质 |
| 5 | 【声音】：`人声.环境声.动作声.打斗声.` 有哪种写哪种 |
| 6 | 结尾必加 `避免面部特征漂移，避免形态变形，避免动作僵硬不自然。` |
| 7 | 大场面：火焰爆燃冲天、十米能量刃、冲击波横扫百米、草皮整片掀飞 |
| 8 | 不站桩、不收势、有来有回 |
| 9 | 每击命中带冲击波扩散 |
| 10 | 顺砍+上挑为主，横砍少用 |
| 11 | 全身皆武器（拳脚膝肘头） |
| 12 | 环境+光效+运镜一体 |
| 13 | 战斗前先查项目技能设定 |
